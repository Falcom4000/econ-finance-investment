# 35.1 投资绩效评价

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.24
- 相关旧笔记：本笔记 Ch.24

投资绩效评价的核心问题不是“赚了多少钱”，而是“在承担什么风险、相对于什么基准、扣除什么成本之后赚了多少钱”。如果不调整风险，牛市中高 beta 组合看起来会像优秀经理；如果不选择基准，任何收益都无法判断好坏。

## 收益率不是绩效

绝对收益率只是起点。一个基金一年赚 12%，如果市场指数赚 20%，它表现差；如果市场跌 10%，它表现好。若基金通过高杠杆赚 12%，风险调整后也未必优秀。

因此绩效评价至少要考虑：

- 基准收益；
- 系统性风险；
- 总风险；
- 主动风险；
- 费用和交易成本；
- 税收；
- 投资期限。

## Sharpe 比率

Sharpe 比率衡量每单位总风险获得的超额收益：

```text
Sharpe = (组合收益 - 无风险利率) / 组合标准差
```

它适合评价整个组合，尤其是投资者全部财富都配置在该组合时。Sharpe 比率高，表示每单位总波动带来的补偿较高。

局限是：它用标准差衡量风险，默认上下波动同等重要；对非正态、含期权或尾部风险策略可能误导。

## Treynor 比率

Treynor 比率衡量每单位 beta 的超额收益：

```text
Treynor = (组合收益 - 无风险利率) / beta
```

它适合评价已经充分分散的组合，因为充分分散投资者主要关心系统性风险。若组合未充分分散，仅看 beta 会忽略公司特有风险。

## Jensen alpha

Jensen alpha 基于 CAPM：

```text
alpha = R_P - [r_f + beta_P (R_M - r_f)]
```

它衡量组合收益是否超过按 beta 应得的收益。正 alpha 表示风险调整后跑赢，负 alpha 表示跑输。

若使用多因子模型，alpha 应在控制市场、规模、价值、动量等因子后计算。否则，所谓 alpha 可能只是因子暴露。

## 信息比率

信息比率衡量主动收益相对于主动风险：

```text
Information Ratio = 主动收益 / 跟踪误差
```

主动收益是组合收益减基准收益；跟踪误差是主动收益的标准差。信息比率适合评价相对基准管理者。

主动管理不是看一次跑赢，而是看每单位主动风险是否稳定产生超额收益。

绩效评价还要服务于投资者目标。养老金关心资产相对负债的表现，个人关心税后实际购买力，基金投资者关心相对基准和费用后收益。一个组合 Sharpe 比率高，但在负债上升或需要现金时表现差，未必是好组合。评价指标必须和投资政策中的目标、期限和约束一致。

## 小结

投资绩效评价必须风险调整并选择合适基准。Sharpe 比率评价每单位总风险收益，Treynor 比率评价每单位系统性风险收益，Jensen alpha 衡量相对 CAPM 或多因子模型的异常收益，信息比率评价主动收益相对于跟踪误差的质量。高收益不等于好绩效，真正重要的是风险、成本和基准调整后的结果。
