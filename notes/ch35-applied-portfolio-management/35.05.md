# 35.5 主动投资管理：Alpha 与 Treynor-Black

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.27
- 相关旧笔记：本笔记 Ch.24

主动投资管理的核心不是“有观点”，而是把观点按风险和置信度转化为组合权重。Treynor-Black 模型提供了一个清晰框架：以被动市场组合为核心，将少数有 alpha 的证券组成主动组合，再按信息比率决定主动组合占比。

## 主动组合与被动组合

Treynor-Black 的思想与指数模型一致：

```text
总风险组合 = 被动市场组合 + 主动组合
```

被动市场组合提供分散化和市场风险溢价；主动组合来自分析师对少数证券的 alpha 判断。

这种结构承认一个现实：大多数资产难以战胜市场，但投资者可能在少数证券上有信息优势。

## alpha 与残差风险

主动组合内部权重与 alpha 成正比，与残差方差成反比：

```text
w_i ∝ alpha_i / sigma^2(e_i)
```

高 alpha 证券更值得配置，但如果残差风险很大，仓位应降低。同样的 alpha，风险越低，信息质量越高。

这体现了信息比率思想：主动管理看的是每单位非市场风险能产生多少 alpha。

## 主动权重

主动组合占总风险组合的比例取决于主动组合的信息比率。信息比率越高，偏离指数的理由越强；信息比率越低，越应接近被动投资。

这避免了主动管理中常见错误：把主观信心直接变成大仓位。模型要求把观点强度、残差风险和市场组合一起考虑。

## 主动管理的价值

主动管理价值来自两个维度：

- 深度：单个观点的信息比率；
- 广度：独立有效观点的数量。

如果经理能在许多低相关证券上形成小而可靠的 alpha，组合层面可能有高信息比率。若所有观点都来自同一风格或同一宏观主题，实际广度很低。

## 小结

Treynor-Black 模型把主动管理组织成被动市场组合和主动 alpha 组合的结合。主动组合权重取决于 alpha 相对于残差风险的强度，整体主动偏离取决于信息比率。它的价值在于给主动观点设置风险纪律：不是有观点就重仓，而是让观点、风险和分散化共同决定仓位。
