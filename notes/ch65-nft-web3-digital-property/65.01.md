# 65.1 NFT 的产权、稀缺性和可验证所有权

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.10；补充参考本笔记 Ch.2、Ch.5、Ch.52、Ch.57、Ch.64。

NFT 最容易被误解为“很贵的图片”。如果只看到图片，读者会自然产生疑问：既然数字文件可以复制，为什么还要为一个 NFT 付钱？这一节要先把问题放回经济学的产权框架。NFT 不是让图片不能被复制，而是让某个链上凭证可以被唯一识别、被公开验证、被转让，并且和某种数字内容、会员权益或使用权连接起来。

## 数字世界为什么需要产权

产权不是简单的“我喜欢这个东西”，而是一组被社会承认的权利：谁可以使用，谁可以排除别人使用，谁可以转让，谁可以从中获得收益。第 2 章讲公共资源和信息问题时已经看到，如果权利边界不清楚，市场交易很难稳定发生。

数字内容的特殊之处在于复制成本接近零。一张图片、一段音乐、一个视频或一个游戏道具，一旦以文件形式存在，就可以被无限复制。复制便利提高了传播效率，却削弱了稀缺性。传统互联网用平台账号、数据库和版权法来管理数字产权：游戏公司记录谁拥有道具，音乐平台记录谁有播放权限，社交平台记录谁发布内容。

NFT 提供另一种做法：把某个独一无二的 token 放在区块链上，由钱包地址持有。别人可以复制图片，但不能复制链上那一个 token 的所有权记录。换句话说，NFT 解决的不是“文件无法复制”，而是“某个可验证的编号和持有人关系无法随意篡改”。

## 非同质化是什么意思

比特币、ETH、稳定币和大多数协议 token 是同质化资产。你给我 1 个 USDC，我给你另 1 个 USDC，通常没有差别。NFT 是非同质化资产，每一个 token 都有唯一编号和属性，不能和另一个 token 简单等价替换。

这种非同质性很像艺术品、收藏卡、房产地块和游戏角色。两套房子都叫房子，但地段、楼层、朝向和产权状态不同；两张球星卡都属于同一系列，但稀有度、品相和历史故事不同；两个游戏角色可能有不同技能、装备和成长记录。NFT 把这种“每个资产都不一样”的特征搬到链上。

原书用 ERC-721 和 ERC-1155 说明 NFT 标准。ERC-721 适合每个 token 都独一无二的资产，例如 CryptoPunks、数字艺术和虚拟土地。ERC-1155 更灵活，可以在一个合约里同时管理同质化和非同质化资产，适合游戏中大量道具、材料、角色和货币共同存在的场景。

## 所有权记录和内容本身不是一回事

理解 NFT 必须分清三层东西。第一层是链上 token，它记录 token 编号、持有人和转让历史。第二层是元数据，通常包括名称、描述、图片链接和属性。第三层是链下内容或权益，例如图像文件、会员资格、游戏功能、版权授权或实物商品关联。

很多 NFT 的图片和元数据并不完全存在链上，而是通过 URI 指向外部服务器或 IPFS 等存储系统。这样可以降低链上存储成本，但也带来新的风险：链接是否长期有效？图片是否能被替换？项目方是否继续维护？如果元数据丢失，链上 token 仍然存在，但它指向的内容可能变得不完整。

因此，NFT 的可验证所有权不等于完整法律产权。你拥有一个 NFT，通常意味着你拥有链上 token；它是否同时赋予商业使用权、版权、会员权、游戏道具使用权或线下权益，要看项目条款。BAYC 之所以被重视，不只是因为它有图片，还因为持有人被赋予一定社群身份和商业使用空间。

## 稀缺性是技术和社会共同形成的

NFT 项目常强调限量。例如 CryptoPunks 和 BAYC 都是 10,000 个。限量供应形成稀缺性，但稀缺性本身不等于价值。一个项目可以只发行 100 个 token，如果没有需求、故事、社群、审美或使用场景，稀缺只是无人关心的稀缺。

经济学上，价格来自供给和需求共同作用。NFT 的供给端可以通过合约规则限制数量，需求端则来自收藏偏好、身份信号、社群参与、预期转售、品牌效应和实际权益。技术能让发行量和持有人记录更透明，但不能保证人们愿意为它付钱。

这和传统艺术品市场很相似。一幅画的物理唯一性很重要，但价值还取决于艺术家声誉、历史地位、收藏群体、拍卖市场和文化叙事。NFT 把这种机制搬到数字空间，只是把证书和交易记录放到了链上。

## 为什么可验证所有权重要

可验证所有权降低了交易中的信息摩擦。买家可以查看某个 NFT 是否真实来自官方合约，当前属于谁，过去是否转让，是否属于某个稀有属性组合。相比平台内部数据库，公链记录更容易被不同市场、钱包和应用读取。

这也是 NFT 与 Web3 应用连接的基础。一个游戏可以识别玩家钱包中的道具，一个社区可以识别会员 NFT，一个品牌可以识别持有人资格，一个 DeFi 协议可以识别可抵押的 NFT。只要不同应用都承认同一链上凭证，NFT 就可以跨场景使用。

不过，可验证所有权也不能解决所有问题。它不能自动判断作品是否侵权，不能保证项目方继续运营，不能防止用户私钥被盗，也不能保证二级市场始终有买家。产权记录清楚只是市场形成的必要条件之一，不是充分条件。

## 小结

NFT 的核心不是让数字文件不能复制，而是让某个链上凭证具有唯一编号、可验证所有权和可转让记录。NFT 通过非同质化标准表示数字艺术、收藏品、虚拟土地、游戏道具和会员权益，但 token、元数据和实际权利必须分开理解。稀缺性由合约供应和社会需求共同形成，技术只能保证记录和供应规则更透明，不能保证价值本身。分析 NFT 时，第一步要问：我拥有的是链上 token、内容使用权、会员权，还是某种可转售的社会身份？
