# 65.4 品牌 Web3、用户关系和平台经济

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.14；补充参考本笔记 Ch.2、Ch.5、Ch.24、Ch.33、Ch.57。

Web3 不只出现在金融和游戏中，也进入品牌、奢侈品、票务、会员和粉丝经济。原书用 Nike/RTFKT、Clinique、IWC、Ticketmaster、Adidas、Lacoste、NBA Top Shot 等案例说明，品牌使用 NFT 和区块链，不一定是为了发行投资品，而是为了改变用户关系：让消费者拥有可验证的身份、会员资格、数字收藏品、活动通行证或实物商品的数字凭证。

## 品牌为什么关心数字所有权

传统品牌依靠产品质量、广告、渠道、会员系统和社交传播建立用户关系。消费者购买商品后，品牌通常通过平台账号、邮件、积分系统或社交媒体继续触达用户。问题是，这些关系大多存在于品牌自己的数据库或第三方平台中。

Web3 提供的想法是：用户可以用钱包持有某种可验证凭证，品牌可以根据凭证提供权益。这个凭证可以是 NFT、会员 token、数字票据或实物商品数字孪生。它既能证明用户身份或购买记录，也能在某些情况下被转让或组合。

对品牌来说，这可能形成新的用户关系。品牌不只是把消息推给消费者，而是围绕 token 持有人建立社区、活动、投票和二级市场。消费者也不只是被动接收营销，而可能成为品牌叙事的一部分。

## Token-gated access 是数字会员制

Token-gated access 指只有持有特定 token 的用户才能进入某些内容、活动、商品或社区。IWC 的 NFT 会员、Ticketmaster 的 NFT 票务、Adidas 和 Lacoste 的 NFT 社区，都可以看作数字会员制的不同形式。

这种模式和传统会员卡相似，但有几个差异。第一，会员资格可以由钱包自动验证，不必依赖单个平台登录。第二，会员凭证可能可转让，形成二级市场。第三，凭证可以被多个应用读取，扩展到线下活动、线上社区、限量发售和合作品牌。

但可转让会员也有权衡。传统会员强调长期消费关系，可转让 NFT 会员可能吸引短期投机者。若会员价格快速上涨，真正想参与活动的用户反而被排除。品牌需要决定会员资格到底是消费关系、身份信号，还是可交易资产。

## 数字孪生和真伪验证

奢侈品和收藏品高度重视真伪、来源和二级市场。区块链可以为实体商品创建数字孪生：一件鞋、一块手表、一张门票或一件艺术品，对应一个可验证 token。这个 token 可以记录来源、购买、转售、维修、活动参与和认证状态。

这对高价值商品尤其重要。假货和来源不明会损害品牌声誉，也会降低二级市场信任。如果买家能验证商品来自官方发行、是否曾转手、是否参与官方认证，交易摩擦会下降。

不过，数字孪生仍要解决链上和链下对应问题。一个 token 证明某件实体商品有官方记录，但不能自动防止实物被调包、损坏或伪造。品牌仍需要防伪标签、认证流程、售后服务和法律执行。第 64 章 RWA 的逻辑在这里同样适用：链上凭证必须连接可靠的链下流程。

## Web3 社区与平台经济

品牌 Web3 经常强调 community。社区不是普通粉丝列表，而是一组可以被识别、互动和激励的持有人。品牌可以给持有人提前购买权、限量商品、线下活动、共同创作机会或治理投票。持有人也可能通过二级市场和社交网络传播品牌。

这和平台经济有关。传统平台把用户、商家和内容聚集在同一系统里，通过网络效应提高价值。Web3 品牌社区试图把一部分平台关系交给 token。持有人越多、互动越强、权益越丰富，token 越有吸引力；token 越有吸引力，品牌社区越容易扩张。

但网络效应也会反向运行。如果权益减少、价格下跌、社区变冷，持有人可能离开，品牌声誉受损。品牌不能只把 NFT 当作一次性销售工具，否则短期收入可能换来长期信任损失。

## 用户数据和身份控制

原书还强调 Web3 身份和数据管理。传统互联网中，用户数据通常由平台收集和控制。Web3 设想用户通过钱包、可验证凭证和去中心化身份控制自己的身份信息，并选择向品牌披露哪些信息。

如果实现得好，品牌可以在保护隐私的同时确认会员资格、年龄、购买记录或活动参与。用户不必把所有个人信息交给每个平台，品牌也能获得更准确和更合规的交互数据。

现实中，这仍然很难。钱包地址可能暴露交易历史，身份系统需要用户体验、监管合规和数据安全配合。品牌使用 Web3 身份时，不能只强调“去中心化”，还要处理隐私、同意、撤销、误用和数据泄露责任。

## 品牌资产、无形资产和投资视角

第 33 章讲股票估值时提到，品牌、网络效应和用户关系都是企业护城河的一部分。Web3 对品牌的意义，也可以从无形资产角度理解。一个成功的 NFT 社区，可能增强客户忠诚、提高复购、扩展数字商品收入，并为品牌创造新叙事。

但如果 Web3 项目设计粗糙，价格暴跌或权益未兑现，也会损害品牌。对投资者来说，关键不是“公司发了 NFT”，而是它是否提高了用户生命周期价值、降低了获客成本、增加了可持续收入，或增强了品牌差异化。

因此，品牌 Web3 要回到商业模式：它是一次营销活动，还是长期会员基础设施？它创造的是消费价值，还是把消费者变成短期投机者？它增强品牌信任，还是消耗品牌信任？

## 小结

品牌 Web3 把 NFT、钱包、数字身份和 token-gated access 用于用户关系管理。它可以创造数字会员、限量内容、活动通行证、数字孪生、真伪验证和社区参与。与平台经济类似，成功的 Web3 社区依赖网络效应和持续权益，但网络效应也可能反向损害品牌。对品牌和投资者来说，重点不是 NFT 本身，而是它是否改善用户关系、增强品牌资产，并形成可持续商业价值。
