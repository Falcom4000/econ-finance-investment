# 65.6 Web3 应用为什么经常从经济机制走向投机循环

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.10、Ch.13-Ch.14；补充参考本笔记 Ch.2、Ch.21、Ch.33、Ch.57、Ch.61、Ch.63。

NFT、GameFi 和品牌 Web3 都有真实机制：可验证所有权、数字稀缺、游戏资产产权、社区会员、真伪验证和用户数据控制。但这些机制很容易被价格叙事覆盖。一个应用本来想解决产权或用户关系问题，最后却变成“买入 token，等待上涨，再卖给下一个人”。理解这种转变，是分析 Web3 应用最重要的一步。

这也是为什么本章不能停留在“有哪些应用场景”。场景存在，不代表商业模式成立；token 可以交易，不代表用户真的需要它；社区热闹，不代表权益能兑现。Web3 的分析要同时看产品使用、产权安排和金融化后果。

## 从使用价值到转售预期

任何资产都可以同时有使用价值和转售价值。房子可以居住，也可以投资；股票代表企业所有权，也可以短期交易；艺术品可以欣赏，也可以炒作。Web3 应用的问题不在于存在转售市场，而在于转售预期常常过早压过使用价值。

NFT 项目如果主要靠“地板价上涨”吸引用户，持有人就会把注意力集中在价格而不是内容、社区和权益。GameFi 如果主要靠代币收益吸引玩家，玩家就会在收益下降时离开。品牌 NFT 如果主要靠稀缺发售制造抢购，而没有长期会员价值，用户关系就会变成一次性投机。

一旦转售预期成为主要需求来源，项目就需要不断吸引新买家。新增买家放缓时，价格下跌，社区情绪恶化，更多人卖出，应用价值被金融价格反向拖累。

## 激励补贴会掩盖真实需求

DeFi 和 GameFi 常用 token 奖励启动网络。奖励可以解决冷启动问题：早期没有用户和流动性时，用 token 补贴吸引参与者。第 61 章讲流动性挖矿时已经看到，这种方式在短期内很有效。

但补贴会掩盖真实需求。用户来是因为产品好用，还是因为奖励高？玩家留下是因为游戏好玩，还是因为收益率还可以？品牌社区活跃是因为有认同，还是因为期待空投？如果无法区分，项目方可能误把补贴带来的交易量当作产品市场匹配。

补贴停止后，真实需求才会显现。若没有足够使用价值，用户会离开，token 卖压增加，价格下跌。许多 Web3 应用的周期都呈现这种路径：奖励启动、价格上涨、社交传播、更多人进入、奖励稀释、收益下降、价格下跌、用户流失。

## 网络效应和反身性

Web3 应用常依赖网络效应。一个 NFT 社区成员越多，社交价值越高；一个游戏玩家越多，市场越活跃；一个品牌会员生态越大，权益越丰富。网络效应可以创造真实价值。

但加密市场中的网络效应常和反身性结合。价格上涨会吸引注意力，注意力吸引用户，用户增加又推高价格，价格上涨被解释为项目成功。这种正循环在牛市中很强。

反过来，价格下跌会削弱身份认同和参与意愿。持有人不愿展示亏损资产，玩家不愿为贬值代币投入时间，品牌不愿继续强调失败项目。价格变化不只是投资结果，也会改变社区心理和应用活跃度。

因此，Web3 项目的基本面和价格更容易相互影响。投资者不能只看价格上涨带来的活跃数据，要判断活跃来自真实使用还是价格反身性。

## 信息不对称和散户参与

Web3 应用往往面向全球散户，进入门槛低，交易 24 小时开放。低门槛有积极意义，但也意味着复杂资产更容易被缺乏经验的用户购买。NFT 权益、GameFi tokenomics、品牌会员规则、智能合约风险和二级市场流动性，都不是简单信息。

信息不对称会让项目方、早期投资者、做市商和大户处于优势。普通用户看到的是路线图、社群热度和价格图；内部人更了解发行计划、解锁节奏、合作真实性、资金状况和后续运营能力。第 2 章讲逆向选择和道德风险时的逻辑，在 Web3 市场中同样存在。

披露不足时，投机循环更容易形成。因为用户无法估计真实价值，只能观察别人是否买入。价格本身变成信号，而价格又容易被营销和薄市场操纵。

## 稀缺叙事容易被过度复制

NFT 和品牌 Web3 常强调稀缺。限量 10,000 个、白名单、空投、独家活动和稀有属性，都能制造紧迫感。稀缺叙事有效，是因为人们害怕错过，也因为稀缺资产在传统市场中确实可能有价值。

问题是，单个项目可以限量，整个市场的项目数量却不受限制。今天有一个限量系列，明天可以有十个新系列。单个 NFT 的供给固定，不代表替代品供给固定。如果用户只是追逐“下一个热门系列”，稀缺性会被新项目不断稀释。

这和品牌经济中的护城河有关。真正稀缺的是长期文化地位、稳定社区、持续创作能力和信任，而不是合约里写着供应有限。没有护城河的限量发行，很容易被另一个限量发行替代。

## 如何判断机制是否强于投机

判断一个 Web3 应用是否有长期价值，可以从几个问题开始。第一，用户即使没有 token 上涨预期，是否仍愿意使用？第二，项目是否有真实收入或消费场景，而不是只靠发行新 token？第三，权益是否清楚、可兑现、可持续？第四，价格下跌时，社区和产品是否仍运转？第五，激励是否奖励真实贡献，而不是奖励短期刷量？

还要看资金流向。健康应用中，资金应来自用户为内容、服务、娱乐、会员或效率改善付费；脆弱应用中，资金主要来自新买家接盘旧买家的资产。前者像企业经营收入，后者像不断滚动的投机池。

这并不是否定 Web3。恰恰相反，只有把投机循环和真实机制分开，才能看清哪些应用可能留下来。NFT 的产权记录、GameFi 的资产所有权、品牌的数字会员和去中心化身份都有实际意义，但它们必须经得起价格退潮后的检验。

## 从金融约束回到经济机制

前面几章反复出现同一个原则：技术改变交易形式，但不能取消经济约束。区块链能降低验证和转让成本，智能合约能自动执行规则，token 能协调激励，但价值仍要来自稀缺资源、真实需求、风险承担和可信权利。

NFT、GameFi 和品牌 Web3 的共同风险，是把“可以交易”误认为“必然有价值”，把“可以发行 token”误认为“已经形成经济体”，把“社区热闹”误认为“存在可持续需求”。金融学的任务，是把这些说法拆开，看现金流、权利、激励、流动性和风险。

因此，学习 Web3 应用不能只问它用了什么链、什么标准、什么钱包，而要问它解决了什么经济问题。若答案清楚，技术可能带来效率；若答案只是价格上涨，项目很可能进入投机循环。

## 小结

Web3 应用容易从真实机制走向投机循环，是因为 token 和 NFT 天然可交易，价格会反过来影响用户、社区和项目叙事。激励补贴能启动网络，也会掩盖真实需求；网络效应能创造价值，也会与价格反身性结合；信息不对称和稀缺叙事会放大散户投机。判断一个项目，要看没有价格上涨预期时用户是否仍愿意使用，资金是否来自真实消费和服务，权益是否可兑现，价格下跌时产品是否仍有生命力。技术叙事最终必须回到经济机制和金融约束。
