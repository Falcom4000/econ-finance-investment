# 65.5 NFT 与 DeFi 的结合：抵押、租赁和碎片化

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.10、Ch.13；补充参考本笔记 Ch.24、Ch.58-Ch.61、Ch.63-Ch.64。

NFT 最初常被看作收藏品或游戏资产，但只要它有市场价格和所有权记录，就会被金融系统继续改造。NFT 可以被抵押借款，可以出租使用权，可以被拆成碎片化份额，也可以进入指数、基金和衍生品。这个过程把 NFT 从“持有和展示的数字物品”推向“可融资、可杠杆、可交易的金融资产”。

## NFT 抵押借贷

NFT 抵押借贷的基本逻辑很简单。持有人把 NFT 作为抵押品，借出 ETH、稳定币或其他资产；如果按期还款，NFT 返还；如果违约，贷款人可以获得或拍卖 NFT。它类似典当、艺术品质押或股票质押，只是抵押品是链上非同质化资产。

难点在估值。ERC-20 抵押品通常有连续交易价格和较深流动性，NFT 则不同。每个 NFT 独一无二，成交不频繁，地板价可能快速变化，稀有属性估值分歧很大。一个 BAYC 的价格不能简单等同于另一个 BAYC，普通地板 NFT 和稀有 NFT 的流动性也不同。

因此，NFT 借贷通常需要更高折扣率、更低贷款价值比和更谨慎的清算机制。贷款人要防止抵押品价格下跌或无法及时卖出；借款人则要防止短期价格波动导致优质 NFT 被清算。

## 地板价预言机和清算风险

DeFi 借贷依赖价格输入。NFT 借贷若用地板价作为估值，就会遇到几个问题。第一，地板价是最低挂单或最低成交，不一定代表某个具体 NFT 的真实价值。第二，薄市场中少量成交就能改变价格。第三，在市场恐慌时，挂单撤销和买盘消失会让价格跳跃式下跌。

如果借贷协议按地板价自动清算，价格被短期操纵或快速下跌时，可能引发集中清算。若清算拍卖没有足够买家，贷款人仍可能亏损。这和第 59 章 DeFi 抵押借贷中的清算机制相同，只是 NFT 的流动性更弱、估值误差更大。

有些协议采用点对点贷款，由贷款人自己评估某个 NFT；有些采用资金池贷款，用集合资金按规则放贷。点对点更灵活，但匹配效率低；资金池更方便，但需要更强的估值和风险控制。

## NFT 租赁和使用权分离

NFT 不一定只能出售，也可以出租。游戏资产、虚拟土地、会员权益、活动通行证和品牌权益都可能有使用价值。如果持有人暂时不用，可以把使用权租给别人，收取租金，同时保留所有权。

这类似现实世界中的房屋出租、设备租赁或证券借贷。所有者把使用权让渡给使用者，使用者支付费用。GameFi 中的资产租赁尤其常见：没有足够资金购买高级角色或装备的玩家，可以租用资产参与游戏；资产所有者则获得收益。

租赁设计要明确权利边界。承租人能否转租？能否获得游戏奖励？能否损坏或消耗资产？租期结束后如何自动收回？若 NFT 附带会员权益，出租是否违反项目规则？链上合约可以执行部分规则，但真实权益仍可能依赖项目方和平台。

## 碎片化让昂贵 NFT 变成可分份额

高价 NFT 可能只有少数买家能参与。碎片化把一个 NFT 锁入合约，再发行许多同质化 token 表示份额。投资者可以买入一小部分经济权益，而不必购买整个 NFT。

碎片化降低参与门槛，也可能改善价格发现。更多投资者可以买卖份额，市场似乎更流动。这和第 64 章资产代币化中的分割性类似：把大额、低流动性资产拆成小份，可能扩大投资者基础。

但碎片化也带来治理和法律问题。谁决定是否出售底层 NFT？若部分持有人想退出、部分想继续持有，如何投票？碎片 token 是否构成证券？底层 NFT 如果没有现金流，碎片持有人买到的主要是转售预期。此时碎片化可能把收藏品进一步变成投机证券。

## NFT 指数、基金和组合管理

随着 NFT 金融化，市场会自然出现指数、基金和组合管理工具。投资者不想挑选单个 NFT，可以买入某个系列篮子、蓝筹 NFT 指数或游戏资产组合。这类似共同基金和 ETF 的逻辑：用组合降低个体资产选择难度。

但 NFT 组合管理比股票基金更难。股票有相对统一的交易市场、披露规则和价格数据，NFT 则有项目差异、稀有属性、流动性断层和估值不确定性。基金若按模型估值，可能与真实可成交价格偏离；若按地板价估值，又可能低估稀有资产或高估薄弱系列。

费用和利益冲突也要关注。基金经理是否与项目方有关？是否用基金资金推高某些 NFT？是否能公平处理赎回？这些问题和传统资产管理一样重要，只是资产更不透明。

## 金融化不是天然坏事，但会改变风险

抵押、租赁、碎片化和基金化都能提高资本效率。持有人不必出售 NFT 就能获得流动性，用户不必购买昂贵资产就能获得使用权，小投资者可以参与高价资产，市场可以形成更多价格信息。

问题在于，金融化会把一个原本偏消费和收藏的市场变成杠杆市场。一旦价格下跌，抵押借贷触发清算，碎片 token 折价，基金赎回压力上升，持有人可能被迫卖出。金融工具越多，价格越不只反映文化和使用价值，也反映融资条件和风险偏好。

因此，NFT 与 DeFi 结合时，要同时用投资学和金融稳定的视角分析：抵押品是否可靠，价格来源是否稳健，杠杆是否过高，清算是否有真实买家，资产是否有最终使用需求。

## 小结

NFT 与 DeFi 结合后，可以形成抵押借贷、租赁、碎片化份额、指数和基金。它们能提高流动性和资本效率，也会引入估值误差、清算风险、治理冲突和证券属性争议。NFT 抵押比同质化 token 更难，因为每个资产独特且交易稀疏；NFT 租赁需要清楚分离所有权和使用权；碎片化降低门槛，也可能把收藏品变成高度投机的金融工具。金融化不是问题本身，真正的问题是杠杆、流动性和底层需求是否匹配。
