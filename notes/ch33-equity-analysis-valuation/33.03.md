# 33.3 股利贴现模型

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.18
- 相关旧笔记：本笔记 Ch.9, Ch.22

股利贴现模型（DDM）从最直接的股东现金流出发：股票价值等于未来股利的现值。虽然很多公司不稳定分红甚至不分红，但 DDM 是理解权益估值的基础，因为股票最终价值必须来自可分配给股东的现金流。

## 基本公式

股票价值可以写成：

```text
P_0 = D_1/(1+k) + D_2/(1+k)^2 + ... + D_t/(1+k)^t + ...
```

其中：

- `D_t` 是第 `t` 期股利；
- `k` 是股权资本成本；
- `P_0` 是今天股票价值。

这个公式就是现值原则在权益上的应用。债券现金流较固定，股票现金流更不确定，因此股权折现率通常高于债券收益率。

## Gordon 增长模型

若股利以稳定增长率 `g` 永续增长，且 `k > g`，则：

```text
P_0 = D_1 / (k - g)
```

这就是 Gordon 增长模型。

它揭示三个核心变量：

- 下一期股利 `D_1` 越高，价值越高；
- 股权资本成本 `k` 越高，价值越低；
- 长期增长率 `g` 越高，价值越高。

模型对 `k - g` 极其敏感。若 `k=9%`、`g=4%`，分母是 5%；若增长率提高到 5%，价值会上升 25%。因此，长期增长率假设必须非常谨慎。

## 增长率与留存收益

可持续增长通常来自留存收益再投资。若公司净资产收益率为 `ROE`，股利支付率为 `payout`，留存率为 `b`，则：

```text
g = b × ROE
```

这说明增长不是免费产生的。公司少分红、多留存，可以提高增长率，但前提是留存资金能赚到足够高的 ROE。若公司把钱留在低回报项目中，增长可能摧毁价值。

这与公司金融中的 NPV 原则一致：只有投资回报高于资本成本的增长才创造价值。

## 多阶段 DDM

许多公司不会永远稳定增长。成长公司可能先经历高增长期，之后增速下降，最终进入稳定增长。多阶段 DDM 将估值拆成：

1. 显性预测期股利现值；
2. 终值现值。

终值通常用稳定增长模型：

```text
Terminal Value = D_{T+1} / (k - g)
```

多阶段模型更接近现实，但也更依赖预测。尤其是终值往往占估值大部分，长期增长率和资本成本的小变化会导致估值大幅变化。

## DDM 的适用性

DDM 适合：

- 分红稳定的成熟公司；
- 公用事业、银行、消费品等现金流较稳定行业；
- 股利政策与盈利能力关系稳定的公司。

DDM 不太适合：

- 不分红或分红很少的成长公司；
- 股利政策不稳定的公司；
- 周期性盈利波动很大的公司；
- 回购比股利更重要的公司。

不过，即使公司不分红，DDM 的思想仍然成立：股票价值最终来自未来可分配现金流。

## 小结

股利贴现模型认为股票价值等于未来股利现值。稳定增长下，`P_0 = D_1/(k-g)`，价值由股利、资本成本和长期增长率决定。增长来自留存收益和 ROE，但只有投资回报超过资本成本的增长才创造价值。DDM 对长期增长率和折现率非常敏感，最适合分红稳定的成熟企业。
