# 33.4 自由现金流估值

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.18
- 相关旧笔记：本笔记 Ch.22

自由现金流估值比股利贴现模型更常用于公司估值。原因是许多公司不按可分配能力分红，股利受管理层政策影响很大。自由现金流模型关注公司真正能提供给资本提供者或股东的现金流。

## FCFF 与 FCFE

自由现金流有两种常见口径。

公司自由现金流（FCFF）是所有资本提供者可获得的现金流，包括债权人和股东：

```text
FCFF = EBIT × (1 - 税率) + 折旧摊销 - 资本开支 - 营运资本增加
```

股权自由现金流（FCFE）是归属于普通股股东的现金流：

```text
FCFE = 净利润 + 折旧摊销 - 资本开支 - 营运资本增加 + 净新增债务
```

FCFF 折现用加权平均资本成本 WACC，得到企业价值；再减去净债务得到股权价值。FCFE 折现用股权资本成本，直接得到股权价值。

## WACC 与企业价值

FCFF 模型的基本形式是：

```text
企业价值 = 未来 FCFF 按 WACC 折现的现值
```

WACC 反映债务和股权资本的机会成本：

```text
WACC = E/(D+E) × 股权成本 + D/(D+E) × 债务成本 × (1-税率)
```

使用 WACC 时要注意口径一致。FCFF 是支付给债权人和股东之前的现金流，因此折现率必须是整体资本成本；FCFE 是支付债务之后的股东现金流，因此折现率必须是股权成本。

## 终值

自由现金流模型通常分为显性预测期和终值：

```text
企业价值 = 显性期 FCFF 现值 + 终值现值
```

终值常用稳定增长公式：

```text
TV = FCFF_{T+1} / (WACC - g)
```

终值往往占估值大部分，因此长期增长率、利润率和再投资假设非常关键。长期增长率不应高于经济长期名义增长率太多，否则公司最终会变得大于经济本身。

## 增长与再投资

自由现金流估值特别强调增长的代价。收入增长需要营运资本、资本开支、研发或并购投入。高速增长公司可能会计利润为正，但自由现金流为负，因为大量现金被用于扩张。

增长是否创造价值取决于投资回报率与资本成本：

```text
价值创造 = 投资资本回报率 > WACC
```

如果公司以低于资本成本的回报扩张，增长会降低价值。估值不能只看收入增长率，还要看增长需要多少再投资、能产生多少回报。

## FCFF 与 FCFE 的选择

FCFF 适合：

- 杠杆变化较大的公司；
- 债务结构复杂的公司；
- 并购估值；
- 需要比较不同资本结构公司的场景。

FCFE 适合：

- 资本结构稳定；
- 股东现金流清晰；
- 金融机构等债务难以与经营区分的公司。

实际估值中，两种方法理论上应一致，但输入口径和假设很容易导致差异。

自由现金流模型把公司金融的资本结构问题显性化。债务成本受信用利差影响，股权成本受 beta 和风险溢价影响，WACC 又会反过来决定哪些项目创造价值。货币政策收紧、信用利差扩大或公司杠杆上升，都会提高资本成本，使原本看似可行的增长计划变得不创造价值。

## 小结

自由现金流估值关注企业或股东真正可获得的现金流。FCFF 折现用 WACC 得到企业价值，FCFE 折现用股权成本得到股权价值。估值关键在于现金流、再投资、资本成本和终值假设。增长只有在投资回报超过资本成本时才创造价值，否则收入扩张也可能损害股东价值。
