# 33.5 市盈率与相对估值

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.18
- 相关旧笔记：本笔记 Ch.9, Ch.22

相对估值用市场上类似资产的价格来估计目标公司的价值。最常见指标是市盈率（P/E），也包括市净率、市销率、EV/EBITDA 等。相对估值简单、贴近市场，但也容易被机械误用。

## 市盈率的基本含义

市盈率是价格除以每股收益：

```text
P/E = 每股价格 / 每股收益
```

它表示投资者愿意为每 1 元当前盈利支付多少价格。高 P/E 通常意味着市场预期高增长、低风险或高盈利质量；低 P/E 可能意味着便宜，也可能意味着增长差、风险高或盈利不可持续。

## P/E 与增长模型

从 Gordon 模型出发：

```text
P_0 = D_1 / (k - g)
```

若股利 `D_1 = EPS_1 × payout`，则：

```text
P_0 / EPS_1 = payout / (k - g)
```

这说明 P/E 由三个基本因素决定：

- 股利支付率；
- 股权资本成本；
- 长期增长率。

因此，高 P/E 不一定贵，低 P/E 不一定便宜。关键是增长、风险和盈利质量是否支持这个倍数。

## 可比公司

相对估值的核心是选择可比公司。可比公司应在以下方面相近：

- 行业；
- 增长率；
- 利润率；
- 资本结构；
- 风险；
- 会计政策；
- 生命周期阶段；
- 地区和监管环境。

如果把高增长软件公司和成熟硬件制造商直接比较 P/E，结论很可能无意义。

## 不同估值倍数

常见倍数各有适用场景。

P/E 适合盈利稳定的公司，但对亏损公司无效。

P/B 适合金融机构和资产密集行业，因为账面资产与盈利能力关系较强。

P/S 适合早期或低利润公司，但忽略成本结构和利润率。

EV/EBITDA 便于比较不同资本结构公司，但忽略折旧、资本开支和营运资本需求。

没有一个倍数普遍正确。倍数选择要匹配行业经济特征。

## 相对估值的陷阱

相对估值最大的风险是把市场整体错误当作基准。如果整个行业被高估，用行业平均倍数估值也会高估目标公司。

其他常见陷阱包括：

- 使用周期高点盈利；
- 忽视一次性收益；
- 忽视会计政策差异；
- 忽视债务和少数股东权益；
- 忽视增长质量；
- 把低倍数误认为安全边际。

相对估值最好与内在价值模型结合。倍数告诉我们市场如何定价类似公司，DCF 告诉我们假设是否能支撑价格。

倍数还会随利率和风险偏好整体移动。低利率环境中，市场愿意为同样现金流支付更高倍数；风险溢价上升时，即使盈利不变，P/E 也可能压缩。相对估值如果只比较同一时期的同行，容易忽视整个市场折现率已经偏低或偏高。把倍数拆成增长、资本成本、盈利质量和周期位置，才能避免把宏观估值环境误认为公司特质。

## 小结

市盈率和相对估值通过比较类似公司价格来判断价值。P/E 可以由股利支付率、资本成本和增长率解释，因此倍数高低必须结合基本面。不同倍数适合不同公司，选择可比公司和调整会计差异非常关键。相对估值简单实用，但会继承市场整体定价错误，不能替代现金流和风险分析。
