# 20.5 货币市场工具的定价、流动性与风险

## 为什么货币市场利率通常一起移动

前面几节分别介绍了国库券、联邦基金、回购协议、大额可转让存单、商业票据、银行承兑汇票和 Eurodollar 存款。它们发行人不同，法律形式不同，交易方式也不同。但如果把这些工具的利率画在同一张图上，会发现它们往往一起移动。

原因在于，它们都是短期、低风险、流动性较高的工具，彼此具有替代关系。一个大型企业暂时有现金，可以买国库券，也可以买商业票据或回购协议；一家银行需要短期资金，可以进入联邦基金市场，也可以使用 Eurodollar 市场或发行大额 CD。只要某个工具的利率暂时明显高于其他类似工具，资金就会流向它，推动价格上升、收益率下降；如果某个工具收益率太低，资金会转向其他工具，推动其收益率回升。

这是一种套利和替代的力量。货币市场工具风险和期限相近时，利率不可能长期大幅偏离。

```mermaid
flowchart LR
    A[某货币市场工具利率偏高] --> B[投资者增加购买]
    B --> C[该工具价格上升]
    C --> D[收益率下降]
    D --> E[与其他短期利率重新靠近]
```

当然，“一起移动”不等于完全相同。不同工具仍然有信用风险、流动性、税收、抵押品和市场深度差异，因此利率会有利差。

## 货币市场工具如何定价

许多货币市场工具是短期折价证券。投资者今天支付一个低于面值的价格，到期收到面值。价格和面值之间的差额就是收益。定价的基本逻辑，就是把未来到期收到的金额折现到今天。

假设一张货币市场证券一年后支付 1000 美元，投资者要求 2% 的收益率。今天愿意支付的价格是未来 1000 美元的现值：

```text
现值 = 未来值 / (1 + 利率)^期限
价格 = 1000 / (1 + 0.02)
     = 980.39 美元
```

如果投资者要求的收益率上升到 3%，价格变成：

```text
价格 = 1000 / (1 + 0.03)
     = 970.87 美元
```

利率越高，今天价格越低。这就是债务工具定价的基本规律：未来固定支付不变时，市场要求收益率上升，当前价格下降；市场要求收益率下降，当前价格上升。

```mermaid
flowchart LR
    A[要求收益率上升] --> B[折现率上升]
    B --> C[未来固定付款的现值下降]
    C --> D[证券价格下降]
```

这个逻辑和第 7 章学习的现值、债券价格和利率反向关系完全一致。货币市场工具期限很短，所以价格波动通常小于长期债券，但方向相同。

## 利率为什么反映风险和流动性

货币市场工具收益率的差异，主要来自风险和流动性。

国库券通常收益率最低，因为它几乎没有违约风险，而且二级市场非常深、非常流动。投资者愿意接受较低收益，换取安全性和随时变现能力。

银行承兑汇票和大额可转让存单利率通常略高，因为它们依赖银行信用。大型货币中心银行信用强，违约风险低，所以利率接近国库券，但仍可能高一些。

商业票据利率通常更高，因为它是企业无担保债务，信用风险高于政府和大型银行工具。即使发行人是大公司，投资者仍然要承担企业现金流恶化或无法展期的风险。

Eurodollar 存款利率也会反映离岸银行信用、监管环境和跨境流动性风险。回购利率则受抵押品质量影响。以高质量国债为抵押的 repo 风险较低；如果抵押品质量受怀疑，利率会迅速上升，甚至市场会冻结。

| 工具 | 风险和流动性特点 | 利率倾向 |
| --- | --- | --- |
| 国库券 | 违约风险极低，二级市场最深 | 最低 |
| 回购协议 | 有抵押，风险取决于抵押品和对手方 | 较低 |
| 大额可转让存单 | 银行信用支持，二级市场较好 | 略高于国库券 |
| 银行承兑汇票 | 银行承兑，贸易背景 | 较低 |
| 商业票据 | 企业无担保信用，二级市场较弱 | 较高 |
| Eurodollar 存款 | 离岸银行信用和跨境因素 | 取决于银行和市场环境 |

这里的核心原则是：投资者要求补偿。风险越高、越难卖出、信息越不透明，投资者要求的收益率越高。

## 流动性：能不能快速、低成本变成现金

流动性指资产能否快速、容易、低成本地转换为现金。货币市场工具通常流动性较好，但不同工具之间差异很大。

国库券流动性最好，因为市场深度大、交易活跃、买卖双方众多。投资者需要现金时，通常可以快速卖出，交易成本很低。

商业票据流动性较弱。许多投资者买入商业票据后持有到期。若到期前急需出售，可能找不到活跃二级市场，必须通过交易商处理，并支付较高费用或接受价格折让。

联邦基金通常没有二级市场，因为它本来就是短期银行间借贷，常常隔夜到期，不需要再转让。Eurodollar 存款也往往二级市场较弱，因为它们是定期存款安排。银行承兑汇票和大额可转让存单则有相对较好的二级市场。

| 工具 | 通常期限 | 二级市场深度 |
| --- | --- | --- |
| 国库券 | 4、13、26、52 周 | 极好 |
| 联邦基金 | 1 到 7 天，隔夜为主 | 基本无 |
| 回购协议 | 1 到 15 天常见 | 较好 |
| 大额可转让存单 | 14 到 120 天常见 | 较好 |
| 商业票据 | 1 到 270 天 | 较弱 |
| 银行承兑汇票 | 30 到 180 天 | 较好 |
| Eurodollar 存款 | 1 天到 1 年 | 较弱 |

短期工具本身快到期，所以二级市场深度不像长期债券和股票那样关键。但对需要随时应对赎回、赔付或保证金要求的投资者来说，流动性仍然重要。货币市场基金的一个功能，就是为一些底层工具提供间接流动性：基金投资于大额工具，小投资者持有基金份额，需要现金时赎回基金。

## 风险不只是违约风险

货币市场工具常被描述为安全，但安全并不表示没有风险。至少有四类风险需要区分。

第一，违约风险。发行人或借款人可能无法按时还款。国库券违约风险最低；商业票据和银行 CD 则取决于企业或银行信用。

第二，流动性风险。即使资产最终能还本，投资者也可能在需要现金时无法以合理价格卖出。商业票据和某些 Eurodollar 存款的二级市场较弱，流动性风险更明显。

第三，利率风险。货币市场工具期限短，所以利率风险较小，但不是零。如果利率上升，已持有的短期证券价格也会下降，只是下降幅度通常有限。

第四，通胀风险。名义收益可能跟不上通胀，实际收益为负。国库券尤其常用于安全储存，但在高通胀时期，它可能保护名义本金，却不能保护购买力。

| 风险类型 | 含义 | 哪些工具更敏感 |
| --- | --- | --- |
| 违约风险 | 到期不能偿还 | 商业票据、银行 CD、Eurodollar |
| 流动性风险 | 急需卖出时价格差或买方少 | 商业票据、部分 Eurodollar |
| 利率风险 | 市场利率变化导致价格变化 | 期限越长越明显 |
| 通胀风险 | 名义收益低于通胀 | 所有固定名义收益工具 |

危机时期，风险会从“看起来很小”变成“突然很重要”。资产支持商业票据危机就是例子：平时投资者相信工具短期、安全、评级高；一旦怀疑抵押资产质量，市场迅速冻结。

## 为什么货币市场利率是宏观信号

货币市场利率不仅是工具定价结果，也是宏观金融状态的信号。

如果联邦基金利率上升，可能表示央行收紧货币政策，也可能表示银行体系准备金紧张。如果商业票据利率相对国库券利率上升，可能表示市场对企业信用风险更担忧，或者企业短期融资需求上升。如果回购利率异常上升，可能表示抵押融资市场流动性紧张。如果投资者大量涌向国库券，国库券收益率下降，可能表示避险需求增强。

这些信号会进入宏观经济。短期融资成本上升，企业可能减少库存、推迟投资、压缩营运资本；金融机构融资成本上升，可能减少贷款和证券持仓；流动性紧张会放大资产价格下跌。于是，货币市场利率变化会影响总需求和金融稳定。

```mermaid
flowchart LR
    A[货币市场利率或利差上升] --> B[短期融资成本上升]
    B --> C[企业和金融机构压缩融资]
    C --> D[投资、库存和信贷下降]
    D --> E[总需求和金融稳定受压]
```

这也是中央银行密切关注货币市场的原因。货币市场是政策利率的落点，也是金融压力最早显现的地方之一。

## 小结

货币市场工具的价格通常通过折现未来到期金额得到。要求收益率越高，当前价格越低；要求收益率越低，当前价格越高。由于货币市场工具期限短、风险低、替代性强，它们的利率通常一起移动。

不同工具之间的利率差异来自信用风险、流动性、抵押品、监管和市场深度。国库券最安全、最流动，收益率通常最低；商业票据依赖企业信用且二级市场较弱，收益率通常较高；回购协议有抵押，风险取决于抵押品质量和对手方。

货币市场工具虽安全，但仍有违约风险、流动性风险、利率风险和通胀风险。危机中，这些风险可能迅速放大。货币市场利率和利差因此是货币政策、短期融资条件和金融稳定的重要信号。
