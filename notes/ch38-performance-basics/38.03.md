# 38.3 Sharpe Ratio 的含义和局限

来源：

- 主线：Paleologo《The Elements of Quantitative Investing》Ch.3
- 相关旧笔记：本笔记 Ch.29, Ch.35

有了期望收益和波动率，就可以构造量化投资中最常用的绩效指标：Sharpe Ratio。它回答的问题很直接：每承担一单位波动，策略能获得多少超额收益？

Sharpe Ratio 的基本形式是：

```text
Sharpe Ratio = 期望超额收益 / 收益波动率
```

如果使用年化收益和年化波动率，得到的是年化 Sharpe Ratio；如果使用日收益和日波动率，得到的是日频 Sharpe Ratio。由于市场惯例通常报告年化值，研究者必须清楚时间尺度如何转换。

## Sharpe Ratio 为什么直观

假设两个策略都以现金为基准。策略 A 年化超额收益 10%，年化波动率 10%，Sharpe Ratio 为 1；策略 B 年化超额收益 15%，年化波动率 30%，Sharpe Ratio 为 0.5。策略 B 赚得更多，但每单位波动获得的补偿更少。

这个指标把收益和风险放到同一尺度下。它特别适合比较可以缩放风险的策略。如果策略 A 波动率较低，理论上可以适度增加杠杆，使其达到更高风险水平；如果 Sharpe Ratio 稳定，放大后每单位风险仍然更有效。当然，真实世界中杠杆有融资成本和容量限制，不能无限放大。

Sharpe Ratio 还和统计检验有关。在收益独立同分布、方差有限的理想条件下，样本 Sharpe Ratio 与收益均值的 t 统计量密切相关。直觉是：Sharpe 越高，样本均值相对噪声越大，越有可能不是偶然。但现实中收益自相关、厚尾和回测选择会让这个解释变复杂。

## Sharpe Ratio 的时间尺度

收益、波动率和 Sharpe Ratio 都依赖时间单位。若日收益独立、同分布且没有自相关，年化收益约等于日均收益乘以一年交易日数，年化波动率约等于日波动率乘以交易日数平方根。因此：

```text
年化 Sharpe ≈ 日 Sharpe * sqrt(一年交易日数)
```

如果一年有约 252 个交易日，日 Sharpe 乘以 sqrt(252) 得到年化 Sharpe。

这个转换看似简单，但条件很重要。若策略收益有正自相关，简单平方根年化可能高估 Sharpe；若收益被平滑估值，波动率被低估，Sharpe 也会被高估；若收益有强尾部风险，Sharpe 不能充分反映极端亏损。

## Information Ratio 与 idiosyncratic return

Sharpe Ratio 通常使用相对于现金的超额收益。Information Ratio，即信息比率，则更常用于评价相对基准或相对因子风险后的主动收益。它的基本形式是：

```text
Information Ratio = 主动收益 / 主动风险
```

主动收益可以是组合收益减基准收益；主动风险通常是跟踪误差，即主动收益的波动率。第 35 章讲绩效评价时已经介绍过这个概念。量化投资中，后面第 39 章会引入因子模型，区分系统性因子收益和特质收益。此时，Information Ratio 更接近衡量“非共同因子驱动的收益质量”。

一个策略可能有高 Sharpe，但只是因为暴露于股票市场上涨；在扣除市场、行业、风格等因子后，Information Ratio 可能并不高。反过来，一个市场中性策略总收益不惊人，但特质收益稳定，可能有较高 Information Ratio。

## Sharpe Ratio 的局限

Sharpe Ratio 的第一个局限是它依赖波动率作为风险指标。若收益分布接近对称、尾部温和，波动率可以较好描述不确定性；但若策略收益非对称，Sharpe 可能误导。

例如，卖出保险或卖出期权类策略常常平时小赚，偶尔大亏。平时波动率低，Sharpe 可能很好看；但一次极端亏损会改变长期结果。Sharpe Ratio 不会自动告诉你亏损尾部有多厚。

第二个局限出现在负收益策略。假设两个策略期望收益都是 -5%，一个波动率 5%，另一个波动率 10%。按 Sharpe 计算，前者 Sharpe 为 -1，后者为 -0.5。指标会说后者“更高”，但对投资者来说，在同样负期望下增加波动并不是好事。这个例子说明 Sharpe 不是完整的偏好排序。

第三个局限是样本不确定性。样本 Sharpe 是估计值，不是真实参数。一个三年回测 Sharpe 为 2 的策略，可能真实 Sharpe 低得多；若研究者尝试了很多信号、很多参数、很多样本区间，最高 Sharpe 很可能包含数据窥探偏差。第 43 章会详细讨论这个问题。

第四个局限是容量和成本。低容量策略在小资金下 Sharpe 很高，资金扩大后交易成本上升、机会被吃掉，Sharpe 会下降。只报告小规模回测 Sharpe，不能说明策略可投资。

## Sharpe Ratio 仍然有用

尽管有局限，Sharpe Ratio 仍然非常有用。它简单、直观、可比较，并且有成熟统计理论。更重要的是，它迫使研究者同时面对收益和风险，而不是只展示收益曲线。

Sharpe Ratio 还可以给损失概率提供粗略约束。即使不假设正态分布，只要均值和方差存在，也可以用概率不等式给极端亏损概率上界。这个上界通常比正态假设下的尾部概率保守得多，提醒我们不要过度相信正态尾部。

实际使用中，Sharpe Ratio 应与其他信息一起看：

- 最大回撤；
- 尾部亏损和偏度；
- 收益自相关；
- 成本和容量；
- 因子暴露；
- 样本外表现；
- 经济逻辑。

## 小结

Sharpe Ratio 衡量每单位波动获得的超额收益，是量化投资最常用的风险调整绩效指标。它直观、易比较，并与均值显著性检验和组合优化有密切关系。Information Ratio 则更适合评价相对基准或相对因子后的主动收益质量。

Sharpe Ratio 的局限同样重要。它继承了波动率作为风险指标的不足，可能低估尾部风险；在负期望收益情况下排序可能不符合投资者偏好；样本 Sharpe 存在估计误差和数据窥探问题；高 Sharpe 不代表高容量。正确使用 Sharpe Ratio，需要同时检查尾部、成本、容量、因子暴露和样本外稳定性。
