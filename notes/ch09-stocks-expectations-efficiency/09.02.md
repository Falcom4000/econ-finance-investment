# 9.2 一期估值模型、股利贴现模型与戈登增长模型

## 股票估值从同一个原则开始

金融估值的基本原则是：一项投资今天的价值，等于它未来能产生的现金流折成今天价值后的总和。债券如此，股票也如此。差别在于，债券未来现金流通常由合同规定，而股票未来现金流更不确定。

普通股投资者可能获得两类现金流：持有期间收到股利，未来卖出股票时收到卖出价格。因此，股票估值要回答的问题是：这些未来股利和未来卖出价格，今天值多少钱？

股票估值的困难在于未来不确定。股利可能变化，卖出价格取决于未来市场预期，权益投资要求回报率也高于安全债券利率。尽管如此，现值原则仍然提供了统一框架。

这也把股票估值和前面的债券估值连接起来。债券估值难点主要是折现率、期限和信用风险；股票估值还要处理剩余现金流不确定、增长机会难估、管理层再投资决策和股东分配政策。股票的 `ke` 不只是一个公式参数，而是投资者对无风险利率、权益风险溢价、公司风险和流动性的综合要求。

## 一期估值模型：持有一年再卖出

最简单的股票估值模型是假设投资者只持有股票一个时期。今天买入，一年后收到股利，然后卖出股票。

股票当前价格等于一年后股利和一年后卖出价格的现值：

```text
P0 = D1/(1 + ke) + P1/(1 + ke)
```

其中：

| 符号 | 含义 |
| --- | --- |
| P0 | 今天的股票价格 |
| D1 | 第 1 年末收到的股利 |
| P1 | 第 1 年末预期卖出价格 |
| ke | 权益投资要求回报率 |

这里使用的是权益要求回报率，而不是无风险利率。股票比债券风险更高，投资者要求更高补偿。`ke` 越高，同样未来现金流的现值越低。

假设某股票当前市场价为 50 美元，预计一年后支付 0.16 美元股利，一年后价格为 60 美元。若投资者要求 12% 回报，则估值为：

```text
P0 = 0.16/1.12 + 60/1.12
P0 = 0.14 + 53.57
P0 = 53.71
```

如果估算价值为 53.71 美元，而市场价格是 50 美元，投资者会认为它有吸引力。当然，别人可能对风险、股利或未来价格有不同判断，所以市场价格未必等于你的估计。

## 为什么卖出价格也来自未来股利

一期模型看起来把股票价值分成两部分：下一期股利和下一期卖出价格。但卖出价格本身又取决于下一位投资者预期未来能得到什么。

如果一年后有人愿意以 60 美元买入，是因为他预计之后还能收到股利并在更未来卖出。再往后看，未来买家也是根据更远未来的股利和卖出价格定价。这样不断向后推，股票价值最终来自未来股利流。

这不是说每家公司现在都必须支付股利才有价值。很多成长型公司当前不支付股利，因为它们把利润留在公司内部用于扩张。投资者仍愿意购买，是因为他们预期公司未来成熟后会支付股利，或通过其他方式把现金流分配给股东。只要市场相信未来某个阶段会有股东可获得的现金流，当前股票就有价值。

这一步推理把一期模型扩展为一般股利贴现模型。

## 一般股利贴现模型

一般股利贴现模型认为，股票今天的价值等于未来所有股利的现值之和：

```text
P0 = D1/(1+ke) + D2/(1+ke)^2 + D3/(1+ke)^3 + ...
```

用求和形式表示：

```text
P0 = Σ Dt/(1 + ke)^t
```

这个模型的含义很强：股票价值由未来股利流的现值决定。最终卖出价格不需要单独保留，因为卖出价格本身也是未来股利现值的反映。

这个结论常让初学者困惑：如果某股票现在不发股利，它为什么有价格？答案是，投资者预期它将来会发股利。高速成长公司常在早期把利润再投资，而不是分给股东。只要投资者相信未来利润增长最终会转化为股东现金流，股票就有价值。

股利贴现模型的难点也很明显：未来股利可能持续很多年，甚至没有终点。逐年预测无限期股利非常困难。因此，需要更简化的模型。

## 戈登增长模型：股利按固定速度增长

戈登增长模型是假设股利以固定增长率永久增长。若最近一期股利为 D0，预期股利增长率为 g，权益要求回报率为 ke，则下一期股利为：

```text
D1 = D0 × (1 + g)
```

股票价格为：

```text
P0 = D1 / (ke - g)
```

或写成：

```text
P0 = D0 × (1 + g) / (ke - g)
```

这个公式非常简洁，但依赖两个重要假设。

第一，股利长期按固定增长率增长。现实中没有公司能完美做到永久固定增长，但如果一家公司已经成熟，股利增长较稳定，这个模型可以给出有用近似。

第二，增长率 g 必须低于权益要求回报率 ke。如果 g 大于或等于 ke，公式没有合理经济含义。长期看，一家公司不可能以高于投资者要求回报率的速度无限增长，否则最终会变得不现实地巨大。

## 戈登模型怎样体现增长和风险

戈登增长模型把股票价格拆成三个核心变量：下一期股利、股利增长率、权益要求回报率。

```text
P0 = D1 / (ke - g)
```

下一期股利 D1 越高，价格越高。股利是股东直接收到的现金流，现金流越多，现值越高。

增长率 g 越高，价格越高。更高增长意味着未来股利更快增加，未来现金流更大。但注意，增长提高价格是通过缩小分母 `ke - g` 实现的，因此估值对增长假设非常敏感。

权益要求回报率 ke 越高，价格越低。投资者要求更高回报，说明股票风险更高或替代投资回报更高。同样未来股利必须以更高折现率折现，现值下降。

| 变量变化 | 对股票价格的影响 | 直觉 |
| --- | --- | --- |
| D1 上升 | 价格上升 | 未来现金流增加 |
| g 上升 | 价格上升 | 未来现金流增长更快 |
| ke 上升 | 价格下降 | 折现率和风险补偿提高 |

这个模型为后面理解市盈率、增长预期和风险打基础。高增长预期会推高股票价格，高风险或高要求回报会压低股票价格。

## 模型不是精确预言，而是估值语言

股票估值模型不应被理解为能精确算出唯一正确价格。它们更像一种分析语言，帮助我们明确价格由哪些因素决定。

一期模型告诉我们，股票价格来自未来股利和未来卖出价格的现值。一般股利贴现模型告诉我们，继续向后追溯后，股票价值最终来自未来股利流。戈登增长模型进一步说明，如果股利按固定速度增长，价格取决于下一期股利、增长率和权益要求回报率。

现实中，投资者会对未来股利、增长率和风险有不同看法。因此，同一只股票在不同投资者眼中可能有不同估值。市场价格正是在这些不同预期和交易中形成的。

还要注意，股利贴现模型中的“股利”可以扩展理解为最终可分配给股东的现金流。公司回购股票、保留利润再投资、未来提高分红，都会影响股东价值。关键不是公司今天是否支付现金股利，而是留存收益能否投资到回报率高于资本成本的项目。如果公司把利润留在低回报项目中，增长反而可能毁灭价值；如果再投资回报高，少分红也可能提高股票价值。

## 小结

股票估值遵循现值原则：股票今天的价值等于未来现金流的现值。股票投资者的现金流包括股利和未来卖出价格，但未来卖出价格本身也来自更远未来股利的现值，因此股票价值最终可以看作未来股利流的现值。

一期估值模型适用于持有一年后卖出的情形：当前价格等于下一期股利和下一期卖出价格的现值。一般股利贴现模型把这一逻辑扩展到无限未来。戈登增长模型假设股利按固定增长率增长，得到 `P0 = D1/(ke - g)`。

模型显示，股票价格随预期股利和增长率上升而上升，随权益要求回报率上升而下降。后续讨论股票价格如何反映预期、有效市场假说和行为金融时，都会以这个估值逻辑为基础。
