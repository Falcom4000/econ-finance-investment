# 9.7 行为金融对有效市场的补充

来源：

- 主线：Mishkin《货币金融学》Ch.7
- 补充：Mishkin/Eakins Ch.6, Ch.13；Bodie/Kane/Marcus《Investments》Ch.12

## 有效市场不等于市场价格永远正确

有效市场假说容易被误解为：金融市场价格总是正确，资产价格总是等于真实内在价值。这个说法比有效市场假说本身更强。

有效市场假说的基本含义是，价格充分反映可得信息，未被利用的盈利机会会被交易消除。因此，价格变化难以预测。它并不必然证明每个价格都等于基本价值。

更强的市场效率观点认为，金融资产价格总是正确反映基本面，也就是未来现金流的真实内在价值。如果这个观点成立，所有证券价格都正确，企业管理者也可以完全依赖证券价格判断资本成本和投资项目价值。

但股票市场崩盘和泡沫让许多经济学家怀疑这种更强观点。价格有时会大幅偏离难以用基本面解释的水平。1987 年“黑色星期一”、2000 年科技股崩盘等事件说明，市场心理和制度结构可能影响价格。

关键区分是：价格可能偏离基本价值，但如果这种偏离无法被稳定预测和利用，基本的有效市场逻辑仍可能成立。

## 市场崩盘说明了什么

大幅股市下跌本身并不自动推翻有效市场假说。若出现重大新信息，使投资者大幅下调未来现金流预期或提高风险补偿，股票价格剧烈下降是可以理解的。

问题在于，有些崩盘很难找到足以解释价格变化的基本面新闻。若基本面没有相应巨大变化，价格却大幅波动，就说明资产价格可能受到市场心理、投资者行为和市场制度的影响。

这使许多人怀疑“价格总是等于基本价值”的强版本市场效率。但它不一定证明普通投资者能轻松获得异常收益。即使价格偏离基本价值，如果没人能提前准确知道何时偏离、偏离多久、何时回归，套利机会仍然难以利用。

所以，崩盘和泡沫提出的问题是：市场价格是否总能准确反映基本价值？行为金融正是从这个问题出发。

## 行为金融研究什么

**行为金融**把心理学、社会学和其他社会科学的观点用于解释金融市场行为。它关心的问题是：真实投资者是否总像理性预期模型中那样处理信息？情绪、认知偏差、群体行为和制度限制会不会影响价格？

行为金融不是简单否定有效市场。它是在问：如果投资者并不完全理性，或者理性投资者无法完全纠正错误价格，那么市场价格会怎样？

有效市场假说依赖一个重要机制：聪明资金会发现并消除未被利用的盈利机会。如果价格过高，聪明投资者会卖出或做空；如果价格过低，他们会买入。行为金融指出，这个机制可能受到限制。

## 损失厌恶和卖空限制

心理学研究发现，人们对损失的痛苦通常大于同等收益带来的快乐。这称为**损失厌恶**。

损失厌恶会影响投资行为，尤其影响做空。做空是借入股票卖出，希望未来价格下跌后低价买回，从中获利。如果股票价格不跌反涨，做空者会亏损；理论上，股价上涨没有上限，做空损失可能很大。

因此，即使某些投资者认为股票被高估，也可能不愿大规模做空。做空还可能受到监管规则、借券困难、资金压力和声誉风险限制。

如果高估股票缺乏足够做空力量，聪明资金就不能轻易把价格压回基本价值。价格可能在较长时间内维持高估。

```mermaid
flowchart LR
    A[股票被高估] --> B[理性投资者想做空]
    B --> C[损失厌恶、做空成本、规则限制]
    C --> D[做空力量不足]
    D --> E[价格可能持续高于基本价值]
```

这说明，发现错误价格和纠正错误价格不是同一件事。

## 过度自信和过度交易

行为金融还强调**过度自信**。许多投资者相信自己的判断优于市场，相信自己能识别别人看不到的机会。这会导致交易频繁。

有效市场假说并不预期普通投资者频繁交易能带来稳定超额收益。如果所有公开信息已经进入价格，频繁交易往往只是增加成本。但现实证券市场交易量很大，过度自信可以解释这种现象。

投资者可能把成功归因于能力，把失败归因于运气或外部事件。几次盈利后，他们更相信自己能预测市场，于是继续交易。即使长期净收益不佳，过度自信也可能让他们高估自己的能力。

过度自信还会让投资者低估风险。估值时，他们可能过于相信自己的增长预测，低估不利情形，愿意为股票支付过高价格。

## 从众、传染和泡沫

股票价格上涨时，投资者容易把收益归因于自己的判断正确。赚钱的人会向他人讲述成功经验，媒体也会放大上涨故事。更多投资者因此相信价格还会继续上涨，买入推动价格进一步上升。

这会形成正反馈：

```text
价格上涨 -> 投资者更乐观 -> 更多买入 -> 价格继续上涨 -> 乐观进一步增强
```

这种社会传染和从众行为可能产生投机泡沫。泡沫期间，价格上涨本身成为继续买入的理由，估值逐渐脱离基本面。当价格过度偏离基本价值、信心转向或新信息出现时，泡沫可能破裂。

行为金融用过度自信、从众、媒体叙事和社会传染解释这种现象。它补充了有效市场假说中较少强调的心理机制。

## 行为金融和有效市场的关系

行为金融并不是说有效市场完全无用。有效市场仍然提供重要基准：公开信息很难带来稳定异常收益，价格会快速反映许多信息，普通投资者不应轻信简单获利机会。

行为金融的补充在于，它说明市场可能并不总能完全、立即、正确地反映基本价值。投资者心理偏差可能推动价格偏离基本面，套利限制可能阻止聪明资金迅速纠正偏差。

可以这样理解两者关系：

| 观点 | 强调什么 |
| --- | --- |
| 有效市场假说 | 信息会进入价格，异常盈利机会难以持续 |
| 行为金融 | 投资者有心理偏差，纠错交易可能受限制 |
| 综合理解 | 市场通常竞争激烈，但价格仍可能因心理和制度因素偏离基本价值 |

这种综合理解更贴近现实：市场不是完全愚蠢，也不是永远正确。它在许多时候快速处理信息，但在某些时期也可能受到泡沫、恐慌和认知偏差影响。

规模、价值、动量、盈利公告后漂移等经验现象，也可以放在这个框架里理解。一种解释是它们补偿了简单模型没有捕捉的系统性风险，另一种解释是它们来自投资者反应不足、过度反应或机构约束。判断一种策略是否真的有效，不能只看历史平均收益，还要看风险暴露、交易成本、税收、样本外表现，以及策略被广泛使用后是否仍然存在。

这也解释了为什么“市场可能犯错”和“我能稳定赚钱”之间有很长距离。即使某只股票被高估，做空者也可能面临借券成本、保证金压力、价格继续上涨和客户赎回。价格偏离价值并不自动等于可执行套利机会。

对经济学和金融学来说，行为金融还说明价格信号并非总是干净的资源配置信号。若泡沫推高某类公司估值，企业可能更容易融资并扩大投资；若恐慌压低资产价格，融资条件会收紧，真实投资也可能受影响。行为偏差不只影响投资者账户收益，也会通过资本成本影响企业融资和宏观波动。

## 对学习者的实际提醒

行为金融容易让人得出错误结论：既然市场可能非理性，我就可以战胜市场。这个推论并不成立。

发现别人可能有偏差，不等于自己没有偏差。认为市场会犯错，不等于自己知道错误在哪里、何时修正、怎样低成本获利。泡沫可以持续很久，做空可能亏损严重，交易成本和风险会吞噬收益。

因此，对普通投资者来说，行为金融更重要的提醒是认识自己的偏差：不要过度自信，不要追逐热门叙事，不要把短期盈利当作能力证明，不要在恐慌时被情绪支配。

有效市场提醒你不要轻信免费午餐；行为金融提醒你不要高估自己的理性。

## 小结

有效市场假说并不等于金融市场价格永远正确。它强调价格反映可得信息、异常盈利机会难以被系统利用。更强的市场效率观点认为价格总等于基本价值，但市场崩盘和泡沫使这一观点受到质疑。

行为金融用心理学和社会科学解释金融市场现象。损失厌恶和做空限制可能使高估价格难以及时纠正；过度自信可以解释投资者频繁交易；从众、社会传染和正反馈可以推动泡沫形成。经验异象需要同时接受风险解释、行为解释和交易成本检验，不能仅凭历史收益就认定存在免费 alpha。

行为金融补充有效市场，而不是简单取代它。市场竞争会让许多公开信息迅速进入价格，但投资者偏差和套利限制也可能使价格偏离基本价值。学习者应同时记住两点：市场很难被轻易战胜，自己也并不天然比市场更理性。
