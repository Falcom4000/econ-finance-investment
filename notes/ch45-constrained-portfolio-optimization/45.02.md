# 45.2 组合约束的类型

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 10, Beyond Simple Mean-Variance.

真实组合总是有约束。约束可能来自投资者偏好，例如不希望承担某个风格暴露；可能来自监管，例如长-only 产品不能卖空；可能来自受托责任，例如不能偏离基准太远；可能来自执行，例如某只股票流动性不足；也可能来自内部风险控制，例如单名仓位、行业暴露、杠杆和换手限制。

最常见的是线性约束。它可以写成不等式或等式：

```text
A' w <= c
A' w = c
```

长-only 约束 `w >= 0` 是最简单例子。单名仓位上限、行业权重上限、国家暴露、市场 beta、风格暴露、净敞口、总多头和总空头比例，很多都可以写成线性形式。线性约束受欢迎，是因为它们解释清楚、计算稳定，并且能表达大量实际限制。

总杠杆或 GMV 约束看起来含有绝对值：

```text
sum |w_i| <= G
```

但可以通过把持仓拆成长头寸 `x` 和短头寸 `y` 转成线性约束：`w = x - y`，`x >= 0`，`y >= 0`，再约束 `sum(x_i + y_i) <= G`。多空比例也可以类似处理，例如 130/30 组合要求多头和空头之间保持固定比例。

因子暴露约束是量化组合最常用的约束之一。若不希望组合承担市场 beta，就要求 `beta' w = 0`；若不希望价值、动量、行业或国家暴露超过范围，就对 `B' w` 加上下界和上界。这里的 `B` 可以来自风险模型，也可以是模型之外的特征。它把第 41 章的因子模型直接用于组合控制。

非线性但凸的约束也很常见。交易成本约束就是例子。若当前持仓为 `w_start`，调仓量为 `w - w_start`，交易成本可以近似为

```text
sum c_i |w_i - w_start,i|^gamma <= C
```

其中 `gamma` 介于 1 和 2 之间。`gamma = 1` 近似线性成本，`gamma > 1` 表示交易越大边际成本越高。这样的约束比简单换手约束更接近真实执行，因为不同股票的流动性和冲击成本不同。

风险约束通常是二次约束。例如对主动组合 `w_active = w - w_benchmark`，可以约束跟踪误差：

```text
w_active' Omega w_active <= sigma_active^2
```

也可以约束某组因子的风险贡献、风格因子波动或组合整体波动。二次风险约束是凸优化中的常见结构，能够被现代求解器较稳定地处理。

非凸约束更麻烦。最大持股数量约束要求组合最多持有 `Nmax` 只股票，这是组合选择问题，通常会引入整数变量，计算复杂度大幅上升。最小特质风险或最小持仓分散度之类约束也可能非凸。非凸约束不一定不能用，但它们会让问题更难求解，也更难保证全局最优。很多时候，非凸约束背后的业务目的可以用更温和的凸约束或惩罚项替代。

约束应服务真实目的，而不是为了让回测更好看。若某个约束只是事后加入，让策略避开历史中亏损的资产或时期，它本身就变成数据窥探。好的约束应能在交易前解释：它来自监管、成本、风险集中、流动性、客户授权或稳健性需求。约束不是装饰，是投资目标和现实限制的一部分。

## 小结

组合约束包括线性约束、凸非线性约束、二次风险约束和非凸约束。线性约束可表达长-only、单名上限、GMV、多空比例和因子暴露；非线性凸约束常用于交易成本；二次约束常用于波动率和跟踪误差；非凸约束如最大持股数量计算更困难。约束必须来自真实投资、监管、风险或执行需求，而不是事后优化回测。
