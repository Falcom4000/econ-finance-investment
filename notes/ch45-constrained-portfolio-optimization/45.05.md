# 45.5 Alpha 误差对 Sharpe Ratio 的影响

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 10, Beyond Simple Mean-Variance.

MVO 最脆弱的输入通常是 alpha。风险模型虽然也会错，但协方差至少可以用大量历史数据和结构模型估计；alpha 则往往很弱、非平稳、容易过拟合。一个组合按估计 alpha `alpha_hat` 和估计风险 `Omega_hat` 构造，最终实现 Sharpe 却由真实 alpha `alpha` 和真实风险 `Omega` 决定。二者之间的差距，就是优化误差的来源。

可以定义 Sharpe Ratio Efficiency，简称 SRE：

```text
SRE = realized SR using estimated inputs / optimal SR using true inputs
```

SRE 不超过 1。只有当估计输入产生的组合方向与真实最优组合方向共线时，SRE 才等于 1。这个事实来自 Cauchy-Schwarz 不等式，直觉很简单：真实最优方向已经是给定真实参数下的最佳方向，任何偏离都会降低风险调整收益。

alpha 误差可以理解为预测方向的角度误差。若 `alpha_hat` 与 `alpha` 的方向很接近，组合方向也接近真实最优；若方向偏离很大，组合可能买错资产、卖错资产，Sharpe 会下降。整体 alpha 尺度错了不一定致命，因为固定风险预算下组合方向不变；方向错才危险。

Alpha 误差造成的损害会被协方差矩阵的条件数放大。条件数是最大特征值与最小特征值之比。若协方差矩阵某些方向风险很大、某些方向风险很小，优化器会特别偏好低风险方向；一旦 alpha 在这些方向上有误，组合权重会被放大。矩阵越病态，alpha 小错误越容易变成持仓大错误。

在特质空间中，若特质协方差近似对角，条件数就是最大特质方差与最小特质方差之比。若某些股票特质波动很低，优化器会愿意给它们更大风险调整仓位；如果这些股票 alpha 估错，损害会很大。因此，对低波动、低估风险资产要特别谨慎。低风险并不等于高确定性，可能只是模型暂时低估了风险。

这个分析也解释了为什么 alpha shrinkage 和仓位约束重要。若 alpha 估计噪声大，可以把 alpha 向零、行业均值或稳健预测收缩；若某些方向因低估风险而权重过大，可以用单名上限、GMV 约束、风险集中约束和交易惩罚限制。它们本质上都在降低 alpha 误差通过优化器放大的程度。

从研究流程看，alpha 误差来自多个环节：回测数据泄漏、样本选择、信号过拟合、参数选择、交易成本低估、非平稳和生产实现差异。第 43 章的回测纪律就是为了在进入组合优化前降低这些误差。若 alpha 输入已经被严重污染，后面的优化只会更精确地放大错误。

## 小结

Sharpe Ratio Efficiency 衡量用估计输入构造的组合相对真实最优组合损失多少。Alpha 误差主要通过预测方向偏离降低实现 Sharpe，整体尺度误差在固定风险预算下相对不重要。协方差矩阵条件数会放大 alpha 误差，尤其在低风险方向上。Alpha 收缩、仓位约束和回测纪律都是为了减少错误 alpha 被优化器放大的程度。
