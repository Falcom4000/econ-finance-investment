# 30.6 用指数模型构建最优组合

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.8
- 相关旧笔记：本笔记 Ch.24

Markowitz 模型的逻辑很完整：给定所有证券的期望收益、方差和协方差，求出有效边界，再找到与无风险资产相切的最优风险组合。但完整 Markowitz 模型在实践中有一个很硬的约束：如果证券数量很多，协方差估计的数量会爆炸，而且每一个估计误差都会进入组合权重。单指数模型的价值就在这里：它把“市场共同波动”和“公司特有波动”分开，用更少的输入构造一个近似的、可操作的最优组合。

这一节要回答的问题是：如果投资经理已经用指数模型估计出若干证券的 beta、alpha 和残差风险，怎样把这些信息转化为实际组合？答案可以概括为一句话：以市场指数作为被动组合的核心，再根据证券分析形成一个主动组合，最后把两者合成为整体风险资产组合。

## 从 Markowitz 输入清单到指数模型输入清单

完整 Markowitz 模型需要三类输入：

1. 每个证券的期望收益；
2. 每个证券的方差；
3. 每两只证券之间的协方差。

如果有 `n` 只证券，协方差数量约为 `n(n-1)/2`。这意味着，当投资范围从几十只股票扩展到几百只、几千只股票时，模型对估计能力的要求会急剧上升。理论上，更多自由度可以描述更复杂的相关结构；实践上，更多自由度也意味着更多噪声。

单指数模型把每只证券的超额收益写成：

```text
R_i = alpha_i + beta_i R_M + e_i
```

其中：

- `R_i` 是证券 `i` 的超额收益；
- `R_M` 是市场指数的超额收益；
- `beta_i` 衡量证券对市场共同因子的敏感度；
- `alpha_i` 是证券分析给出的非市场预期超额收益；
- `e_i` 是公司特有残差。

在这个设定下，组合构建所需的输入变为：

1. 市场指数的期望风险溢价 `E(R_M)`；
2. 市场指数的方差 `sigma_M^2`；
3. 每只被分析证券的 `alpha_i`；
4. 每只被分析证券的 `beta_i`；
5. 每只被分析证券的残差方差 `sigma^2(e_i)`。

这就是指数模型的工程优势：宏观分析师可以负责市场风险溢价和市场波动率，行业或个股分析师可以负责 alpha、beta 和公司特有风险。它把投资组织内部的分工结构映射到模型结构上。

## 被动组合：市场指数作为分散化工具

在指数模型里，市场指数不只是一个比较基准，也是一种分散化工具。原因是公司特有风险可以通过持有大量证券被分散掉，而市场指数天然持有很多证券，已经包含了较高程度的分散化。

风险分担需要一个足够大的“池子”。保险市场通过把大量独立风险放在一起降低单个风险事件对整体的冲击；市场指数通过把许多公司放在一起降低单个公司事件对组合的冲击。差别在于，保险池重点处理相对独立的个体风险，而股票指数仍然暴露在宏观共同风险之下。

因此，指数模型构建组合时通常从一个被动市场组合开始。这个组合承担市场系统性风险，提供市场风险溢价；主动管理只在有证券分析优势的地方偏离指数。

## 主动组合：把 alpha 变成权重

主动组合由被研究证券构成。它的核心规则是：

```text
主动组合中证券权重与 alpha_i 成正比，
与残差方差 sigma^2(e_i) 成反比。
```

直观解释是：

- alpha 越高，说明证券分析认为这只证券越被低估，应该买入更多；
- alpha 越负，说明证券分析认为这只证券越被高估，可以减持或做空；
- 残差风险越大，同样的 alpha 越不可靠，权重应该被压低；
- 残差风险越小，同样的 alpha 更“干净”，可以承受更大权重。

可以把它理解为一个“信号强度与噪声”的问题。alpha 是信号，残差方差是噪声。好的主动头寸不是简单追逐最高 alpha，而是寻找 alpha 相对于公司特有风险足够高的证券。

教材中常用的形式是：

```text
w_i^* ∝ alpha_i / sigma^2(e_i)
```

这只是主动组合内部的相对权重。它告诉我们：在主动组合这一个篮子里，哪些证券应该占更大比重。但它还没有回答另一个问题：主动组合本身应该占整个风险资产组合多少？

## 主动组合与市场指数的合成

完整风险组合由两部分组成：

```text
风险组合 = 市场指数组合 + 主动组合
```

市场指数组合提供广泛分散化，主动组合提供证券分析带来的额外收益机会。问题是主动组合占比多大。这个权重取决于两个方面：

1. 主动组合的 alpha 有多强；
2. 主动组合的残差风险有多大。

如果主动组合的 alpha 很高、残差风险很低，投资经理就有理由更大幅度偏离指数；如果 alpha 很弱或残差风险很高，最优选择可能只是轻微偏离指数，甚至接近纯被动投资。

这和资本配置逻辑是同一个结构：风险补偿越高、风险越低，风险资产权重越高。这里讲的是在市场指数和主动组合之间配置：主动 alpha 越高、特有风险越低，主动偏离越大。

## beta 调整：主动组合会改变市场暴露

主动组合并不一定是市场中性的。它可能包含高 beta 股票，也可能包含低 beta 股票；可能整体 beta 大于 1，也可能小于 1。把主动组合加入市场指数后，整个风险组合的 beta 会被改变。

这一步很容易被忽视。投资经理以为自己只是在“买入有 alpha 的股票”，但实际上可能同时提高了组合的市场风险暴露。例如，很多周期性股票在经济扩张时看起来估值便宜、盈利弹性强，但它们通常 beta 也高。如果组合大幅买入这些股票，最终收益波动可能主要来自市场周期，而不是证券选择能力。

因此，指数模型构建最优组合时需要区分两种收益来源：

- 市场暴露带来的风险溢价；
- 个股分析带来的 alpha。

如果主动组合提高了整体 beta，投资经理可以通过降低市场指数头寸、增加无风险资产，或者使用指数期货等工具调整市场暴露。这样做的目的不是消除所有风险，而是把“想承担的风险”和“无意中承担的风险”分开。

高 beta 暴露本质上是对商业周期、货币政策、风险偏好变化的暴露；alpha 则应当是对个别证券错误定价的补偿。两者混在一起，就很难判断投资能力来自哪里。

## 为什么指数模型不一定输给完整 Markowitz 模型

从理论上说，完整 Markowitz 模型更一般。它允许任意两只证券之间有独立的协方差结构，而单指数模型把证券间协方差限制为：

```text
Cov(R_i, R_j) = beta_i beta_j sigma_M^2
```

这意味着单指数模型假设不同证券的残差之间不相关。现实中，这个假设并不总是成立。例如同一行业中的两家公司可能有共同的供给冲击、监管冲击或技术冲击，它们的残差就可能相关。

但实践中，完整模型的优势常被估计误差抵消。完整协方差矩阵需要估计大量参数，而历史数据有限、相关结构又会随时间变化。一个看起来精细的模型，如果输入数据噪声很大，输出的最优权重可能反而极端且脆弱。指数模型牺牲一部分结构灵活性，换来更稳定、更容易解释的输入。

这和经济学里的模型取舍很相似。一个模型不是越复杂越好，而是要看它在可获得信息条件下是否提高决策质量。完全竞争、IS-LM、货币需求、CAPM 都不是现实的完整复制，而是把关键机制抽出来，帮助我们在不确定环境中做判断。

## 约束条件下的组合构建

教材中的最优组合推导通常允许做空、允许自由调整头寸，并且忽略交易成本。但真实投资组合会面对很多约束：

- 不允许做空；
- 单只证券权重上限；
- 行业或风格偏离限制；
- 流动性约束；
- 税收和交易成本；
- 客户风险承受能力；
- 合规或 ESG 限制。

这些约束会改变指数模型给出的纯理论权重。例如，一个证券有很高的正 alpha，但成交量很低，组合无法建立足够大头寸；或者一个证券有负 alpha，但制度不允许做空，投资经理只能低配而不能负配。

这说明指数模型不是自动交易机器，而是组合构建的组织框架。它把证券分析、市场风险、公司特有风险和组合偏离程度放在一个统一语言里，然后再叠加真实世界约束。

## 一个简化例子

假设市场指数的预期超额收益为 6%，标准差为 18%。分析师研究了三只股票：

| 证券 | alpha | beta | 残差标准差 |
| --- | ---: | ---: | ---: |
| A | 3% | 1.2 | 30% |
| B | 1% | 0.8 | 15% |
| C | -2% | 1.1 | 20% |

直觉上：

- A 的 alpha 高，但残差风险也高；
- B 的 alpha 较低，但残差风险更低；
- C 的 alpha 为负，应被低配或做空。

主动组合不会简单地把最大权重给 A，而会比较 `alpha / 残差方差`。如果 B 的 alpha 虽小但更稳定，它可能获得不低的权重。C 因为 alpha 为负，在允许做空的情况下可能成为空头头寸；不允许做空时，则通常会被排除或相对指数低配。

然后，投资经理再根据主动组合整体的 alpha 和残差风险决定：是只让主动组合占整个风险资产组合的 5%、20%，还是更高。这个过程体现了主动管理的核心纪律：不是“有观点就重仓”，而是“观点强度、风险和分散化一起决定仓位”。

指数模型把风险资产配置、有效边界和主动管理放进同一个框架：风险溢价和风险共同决定最优权重，组合构建的目标仍然是提高每单位风险的补偿，市场指数则成为核心被动组合。主动管理是否真的创造 alpha，不能只看绝对收益，而要看风险调整后的表现。

在机构实践中，这会进一步变成风险预算和治理问题。投资委员会可能允许某个主动组合承担一定跟踪误差、行业偏离或 beta 偏离，但不能让一个看似个股 alpha 的策略变成无意中的宏观杠杆下注。风险预算的作用，是把主动观点放进可承受的组合框架里，让投资经理知道自己是在花费哪一种风险预算来追求 alpha。

## 小结

单指数模型把组合构建拆成市场指数和主动组合两部分。市场指数提供分散化和市场风险溢价，主动组合来自证券分析产生的 alpha。主动组合内部权重通常与 `alpha_i` 成正比、与残差方差 `sigma^2(e_i)` 成反比；主动组合在整体风险组合中的权重，则取决于它的 alpha 相对于残差风险是否足够有吸引力。指数模型虽然比完整 Markowitz 模型假设更强，但输入更少、估计风险更低，也更适合把宏观分析和证券分析组织起来。

## 自测问题

1. 为什么完整 Markowitz 模型在证券数量很多时会遇到估计困难？
2. 指数模型为什么能把协方差估计数量大幅减少？
3. 主动组合中，为什么权重不只取决于 alpha，还要除以残差方差？
4. 主动组合加入市场指数后，为什么还要关注整体 beta？
5. 在不允许做空的约束下，负 alpha 证券的组合处理方式会有什么变化？
6. 为什么一个较简单的指数模型在实践中可能优于更复杂的完整协方差模型？
7. 风险预算为什么能防止主动组合变成无意中的宏观风险押注？
