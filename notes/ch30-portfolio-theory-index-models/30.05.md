# 30.5 单指数模型：系统性风险与公司特有风险

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.8
- 相关旧笔记：本笔记 Ch.9, Ch.24

## 为什么需要指数模型

Markowitz 模型在理论上很完整，但实际使用有一个明显困难：输入太多。资产数量一多，协方差数量会迅速膨胀。500 只股票需要估计超过 12 万个方差和协方差相关参数。更麻烦的是，预期收益估计噪声很大，优化结果可能对微小估计误差非常敏感。

单指数模型就是为了解决这个问题。它不是完全替代 Markowitz 思想，而是用一个简化结构解释证券收益之间为什么相关：许多股票之所以一起涨跌，是因为它们共同受到市场指数或宏观经济因素影响。

这个想法很自然。第 30.1 节已经区分了市场风险和公司特有风险。单指数模型把这种区分写成公式：一个股票的超额收益，一部分来自市场整体波动，一部分来自该公司自己的特殊消息。

## 单指数模型的基本形式

令 `Ri` 表示证券 i 的超额收益，即证券收益率减去无风险收益率；令 `RM` 表示市场指数的超额收益。单指数模型写成：

```text
Ri = αi + βi RM + ei
```

这个式子有三个核心参数。

`βi` 是证券对市场指数的敏感度。如果 beta 为 1，市场超额收益上升 1 个百分点时，该证券平均也上升 1 个百分点；如果 beta 为 1.5，它对市场更敏感；如果 beta 为 0.5，它对市场较不敏感。

`αi` 是市场因素之外的预期超额收益。它可以理解为证券分析师认为该证券相对于市场风险暴露应有收益之外，额外提供的收益。如果一只股票被低估，分析师可能给出正 alpha；如果被高估，可能给出负 alpha。

`ei` 是公司特有的意外收益，来自该公司或行业的特殊事件，例如新产品、诉讼、管理层变化、并购消息或财报惊喜。模型假设这些公司特有意外平均为 0，并且不同公司之间大体不相关。

## beta 衡量系统性风险暴露

beta 是单指数模型中最重要的风险参数。它不是总风险，而是市场风险暴露。一个高 beta 股票在市场上涨时通常涨得更多，在市场下跌时通常跌得更多。一个低 beta 股票对市场波动反应较弱。

市场指数可以看作对宏观经济、企业利润、利率和风险偏好共同变化的汇总。某只股票的 beta 越高，说明它对这些共同因素越敏感。周期性行业、成长股或高经营杠杆公司往往 beta 较高；公用事业、防御性消费等行业 beta 可能较低。

在资产定价中，beta 的意义会更进一步。如果市场组合是有效组合，那么投资者要求补偿的不是总标准差，而是 beta 所代表的系统性风险。单指数模型先提供了测量 beta 的工具。

## alpha 表示非市场预期收益

如果一个证券的预期超额收益完全由市场风险暴露解释，那么：

```text
E(Ri) = βi E(RM)
```

单指数模型允许再加上 alpha：

```text
E(Ri) = αi + βi E(RM)
```

alpha 是证券分析的核心。一个正 alpha 证券意味着，在考虑其市场 beta 后，它仍然有额外预期收益。投资经理如果相信自己的研究能力，就会试图发现正 alpha 证券并提高权重，发现负 alpha 证券并降低权重或卖空。

但 alpha 也最难估计。历史平均收益中包含大量噪声，短期表现好不一定说明未来 alpha 为正。有效市场假说还提醒我们，如果大量投资者竞争寻找低估证券，明显 alpha 机会会被交易消除。因此，alpha 既是主动管理的目标，也是最容易被过度自信误判的部分。

## 总风险的分解

单指数模型把证券总方差分为两部分：

```text
总风险 = 系统性风险 + 公司特有风险
σi^2 = βi^2 σM^2 + σ^2(ei)
```

第一部分 `βi^2 σM^2` 来自市场指数波动，无法通过持有更多普通股票轻易消除。第二部分 `σ^2(ei)` 来自公司特有事件，可以通过分散化降低。

这个分解让第 30.1 节的直觉变得可计算。单个股票的高标准差可能来自高 beta，也可能来自高公司特有波动。两者对组合的意义不同。高公司特有风险可以被分散掉，高 beta 风险则会留在组合中。

## 协方差的大幅简化

在完整 Markowitz 模型中，每一对股票的协方差都要单独估计。单指数模型大幅简化：两只股票之所以相关，是因为它们都受市场指数影响。因此两只股票 i 和 j 的协方差为：

```text
Cov(ri, rj) = βi βj σM^2
```

这个公式说，两个股票的共同波动取决于三个因素：股票 i 对市场的敏感度、股票 j 对市场的敏感度、市场本身的方差。若两只股票 beta 都高，它们会因为共同受市场影响而高度相关；若一只 beta 低，协方差相对较低。

这使输入数量大幅下降。每只股票只需估计 alpha、beta 和公司特有方差，再加上市场风险溢价和市场方差。对 500 只股票，参数数量从十几万下降到一千多。模型因此更适合大规模证券分析和组合构建。

## 单指数模型如何解释分散化

假设投资者持有一个等权股票组合。每只股票收益都可以分成市场部分和公司特有部分。组合的 beta 是个股 beta 的平均值，组合 alpha 是个股 alpha 的平均值，公司特有误差是许多个股误差的平均值。

随着持股数量增加，公司特有误差会相互抵消。某家公司好消息带来的正误差，可能被另一家公司坏消息带来的负误差抵消。组合公司特有风险会随着证券数量增加而下降。

但市场部分不会消失。只要组合中大多数股票 beta 为正，市场下跌时组合仍会下跌。组合越分散，剩下的风险越主要来自 beta 乘以市场波动。

这就是单指数模型对分散化的清晰解释：分散化消除 `ei`，不能消除 `βi RM`。

## 模型的代价：现实风险不止一个指数

单指数模型的简化有代价。它假设股票之间的相关性只来自市场指数，忽略行业、风格、利率、汇率、商品价格等其他共同因素。现实中，同一行业股票可能因为油价、监管政策或技术变化共同波动，即使整个市场指数变化不大。

例如两家石油公司可能有相关的公司特有误差，因为它们都受油价冲击影响。单指数模型若把这些误差视为不相关，就会低估同时持有它们的风险。反过来，某些资产可能在市场之外提供额外分散化价值，单指数模型也可能低估。

因此，单指数模型适合做基础框架和大规模简化，但不是最终真理。多因子模型会把市场之外的共同风险因素纳入分析，例如规模、价值、盈利能力、投资风格、利率和信用风险等。

beta 也可以理解为证券对宏观金融条件的压缩暴露。高 beta 资产通常更依赖经济扩张、融资条件宽松和风险偏好上升；低 beta 资产通常现金流更稳定或需求更防御。货币政策、信用周期和总需求变化会通过市场指数进入 beta 风险。把 beta 解释成“对共同经济风险的敏感度”，比把它简单理解为“股票涨跌幅更大”更有用。

这能防止一个常见误解：beta 不是单纯的统计标签，而是资产现金流和折现率对共同经济状态的反应。周期股、高杠杆公司和成长股往往在经济和金融条件转弱时同时受压，所以它们的 beta 较高。理解 beta 的经济来源，才能把指数模型和宏观金融联系起来。

## 小结

单指数模型把证券超额收益写成市场部分、alpha 和公司特有误差：`Ri = αi + βi RM + ei`。beta 衡量证券对市场指数的敏感度，是系统性风险暴露；alpha 表示市场风险之外的预期超额收益；公司特有误差来自个别证券事件。

模型把总风险分解为系统性风险和公司特有风险，并把任意两只股票的协方差简化为 beta 乘积与市场方差的乘积。这大幅减少 Markowitz 模型所需输入，也让证券分析和组合优化更容易组织。

单指数模型的局限是它把共同风险来源压缩成一个市场指数，可能忽略行业和其他共同因素。它是理解 beta、alpha、分散化和多因子模型的基础。
