# 47.1 对冲问题的基本形式

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 12, "Hedging"。

对冲之所以成为一个独立问题，是因为投资组合的风险并不总是来自我们真正想承担的地方。一个股票多头组合也许真正表达的是选股能力，却不可避免地带有市场方向风险；一个能源股组合也许想赚公司经营差异，却会被原油和天然气价格牵动；一个海外资产组合也许看好当地资产，却会被汇率波动吞没。对冲的基本任务就是在不完全放弃原组合经济含义的前提下，用另一个资产或组合抵消不想承担的风险。

最简单的设定包含两个资产：核心组合和对冲工具。核心组合的收益记为 `r_c`，头寸规模为 `x_c`；对冲工具收益记为 `r_h`，头寸规模为 `x_h`。为了把问题看清楚，先假设核心组合有正期望收益 `mu_c`，对冲工具本身没有期望收益 `mu_h = 0`，两者波动率分别为 `sigma_c` 和 `sigma_h`，相关系数为 `rho_c,h`。这相当于说，我们不是因为看好对冲工具才交易它，而是因为它能改变核心组合的风险结构。

在这个设定下，组合收益为 `x_c r_c + x_h r_h`。由于对冲工具没有预期收益，加入它不会提高组合的期望收益；它唯一可能带来的好处是降低方差。最大化 Sharpe Ratio 与最小化组合方差在这里给出同一个答案：

```text
x_h* = - rho_c,h sigma_c x_c / sigma_h
```

把这个式子除以 `x_c`，就得到对冲比例：

```text
x_h* / x_c = - beta(r_c, r_h)
```

这里的 `beta(r_c, r_h)` 是核心组合收益对对冲工具收益的时间序列 beta。直觉很直接：如果核心组合对市场指数的 beta 为 0.8，那么要消掉市场方向风险，就卖空约 0.8 倍名义本金的指数；如果核心组合与对冲工具负相关，则对冲头寸方向会反过来。

这个结果也说明，对冲不是随意找一个相反方向的资产，而是在协方差结构里寻找能解释核心组合波动的那一部分。未对冲时，组合方差是 `sigma_c^2 x_c^2`；按最优比例对冲后，方差变为：

```text
(1 - rho_c,h^2) sigma_c^2 x_c^2
```

因此，对冲能降低多少风险，完全取决于相关系数的平方。相关性越高，对冲工具越能解释核心组合的波动，剩余风险越少；相关性很低时，即使方向看起来“相关”，真实的风险削减也有限。对应地，若预期收益不变，Sharpe Ratio 的改善倍数为：

```text
SR_hedged / SR_native = 1 / sqrt(1 - rho_c,h^2)
```

这个公式有两个重要含义。第一，对冲不是免费创造收益，而是通过降低无报酬风险来提高单位风险收益。第二，对冲效果高度依赖估计质量。相关系数和 beta 如果估错，名义上是在降低风险，实际可能是在引入新的风险。

在实际投资中，`beta(r_c, r_h)` 可以用两种等价思路估计。第一种是时间序列回归：用核心组合历史收益回归对冲工具历史收益，回归斜率就是 beta。第二种是从资产协方差矩阵出发。如果核心组合权重为 `w_c`，对冲工具权重为 `w_h`，资产收益协方差矩阵为 `Omega_r`，则：

```text
beta(r_c, r_h) = w_c' Omega_r w_h / (w_h' Omega_r w_h)
```

这与前面因子模型和组合风险分解相连：对冲的核心仍然是协方差。经济学里我们关心风险承担是否有补偿，金融学里我们用 beta 描述系统性暴露，投资学里则要把这些暴露变成真实头寸。对冲就是这三层逻辑的交汇点。

不过，这个“香草”对冲问题建立在几个强假设上。第一，beta 被准确估计；第二，只有一个对冲工具；第三，交易成本可以忽略；第四，对冲工具没有自身 alpha 或风险溢价；第五，问题只看一个时期。这些假设让公式非常清楚，但也提醒我们：真实对冲不是把 beta 套进公式就结束，而是要继续处理参数误差、多个风险来源、交易成本和动态调整。

## 小结

基本对冲问题把核心组合的风险拆成两部分：可以被对冲工具解释的部分，以及对冲后留下的残差风险。在单一对冲工具、零交易成本、零对冲工具期望收益的设定下，最优对冲比例等于核心组合对对冲工具 beta 的相反数。对冲提高 Sharpe Ratio 的机制不是增加期望收益，而是移除没有报酬或不想承担的波动。
