# 47.3 可交易因子的时间序列 Beta 对冲

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 12, "Hedging"。

有些风险因子并不是股票截面模型里的风格因子，而是市场上可以直接交易或近似交易的宏观变量。例如，能源股组合可能暴露于原油和天然气价格；银行股组合可能暴露于利率曲线；全球组合可能暴露于美元指数；某些通胀主题组合可能暴露于商品期货。这类风险通常可以用期货、ETF、ETN 或高度流动的篮子来对冲。

处理这类风险时，常见方法是估计每个资产对可交易因子的时间序列 beta。设组合权重为 `w`，对冲工具收益为 `r_h`，第 `i` 个资产对 `r_h` 的真实 beta 为 `beta_i`。如果真实 beta 已知，那么组合对对冲工具的 aggregate beta 就是 `beta' w`，完全对冲头寸为：

```text
x_h = - beta' w
```

问题在于，真实 beta 不可见。我们只能通过历史回归估计它：

```text
beta_hat_i = beta_i + eta_i
```

其中 `eta_i` 是估计误差。若直接把估计 beta 当成真实 beta，组合对冲头寸就变成 `- beta_hat' w`。这一步看似自然，却可能让风险变大。原因是对冲工具本身有波动，错误的对冲头寸会把估计误差放大成真实 PnL 波动。

在简化设定下，完全按估计 beta 对冲后的方差可以分解为三部分：

```text
var_hedged
= w' Omega_r w
- (sigma_h beta' w)^2
+ sigma_h^2 w' Omega_eta w
```

第一项是未对冲组合的方差。第二项是如果知道真实 beta，本来可以消掉的风险。第三项是 beta 估计误差带来的新增风险。对冲是否有效，取决于第二项是否大于第三项。若估计误差很大，或者组合过于集中，使 `w' Omega_eta w` 很大，那么对冲工具可能带来的噪声超过它消除的风险。

这点在投资上很重要。许多风险看起来“应该对冲”，但如果 beta 不稳定、样本太短、宏观关系发生结构变化，强行对冲可能只是把一个未知风险换成另一个未知风险。比如能源股对原油期货的 beta 在不同油价周期、政策环境和公司套保行为下会变化；银行股对利率的暴露也会随资产负债结构和曲线形态变化。时间序列 beta 不是物理常数，而是经济关系在特定样本中的估计。

解决办法不是放弃对冲，而是收缩对冲强度。令对冲头寸为：

```text
x_h = - y_h beta_hat' w
```

其中 `y_h` 是 0 到 1 之间的收缩系数。若 beta 估计非常可靠，`y_h` 接近 1；若估计噪声很大，`y_h` 应该下降。在简化模型里，最优收缩系数为：

```text
y_h* = 1 - (w' Omega_eta w) / (w' beta_hat)^2
```

实际使用时通常还会把它截断在 0 以上，因为负的对冲强度意味着反向加风险。这个公式的含义可以用信噪比理解：分母是组合估计 beta 的平方，是希望对冲的信号；分子是组合 beta 估计误差，是对冲会引入的噪声。信号越强、噪声越小，越接近完全对冲；信号越弱、噪声越大，越应该部分对冲甚至不对冲。

组合集中度也会影响 beta 对冲。若组合只持有少数股票，单个资产 beta 的估计误差不容易相互抵消，`w' Omega_eta w` 会较大；若组合高度分散，估计误差可以更好地平均掉。在一个更简单的情形中，如果所有资产 beta 类似，估计误差也类似，那么收缩强度与 Herfindahl 集中度有关：

```text
H(w) = ||w||_2^2 / ||w||_1^2
```

集中度越高，`H(w)` 越大，beta 估计误差对组合越危险，合理对冲比例越低。这与前面组合构建中的分散化逻辑一致：分散化不仅降低资产特质风险，也降低参数误差对组合决策的破坏。

可交易因子 beta 对冲的实践原则可以概括为三步。第一，确认风险因子是否有经济含义，而不只是历史相关。第二，估计组合对该因子的 beta，并同时估计 beta 的误差。第三，根据估计误差和组合集中度对对冲比例做收缩。这样做比简单追求“beta 为零”更稳健，因为它承认模型估计本身也是风险来源。

## 小结

可交易因子的时间序列 beta 对冲适用于用期货、ETF 或其他流动工具管理宏观风险。若真实 beta 已知，最优对冲是持有组合 aggregate beta 的相反头寸；但实际 beta 存在估计误差，完全按估计 beta 对冲可能增加风险。因此，对冲比例应随 beta 估计误差和组合集中度收缩。对冲的关键不是形式上消除暴露，而是降低真实风险。
