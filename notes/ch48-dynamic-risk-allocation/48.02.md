# 48.2 Kelly 策略的数学性质

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 13, "Dynamic Risk Allocation"。

为了理解 Kelly 策略的性质，可以从单一风险资产的近似公式开始。设风险资产收益率为 `r`，投资者每期投入财富比例 `x`。Kelly 问题是：

```text
max_x E[log(1 + x r)]
```

若收益率不太大，可以用二阶近似：

```text
log(1 + x r) approx x r - 0.5 x^2 r^2
```

取期望后得到：

```text
E[log(1 + x r)]
approx mu x - 0.5 (sigma^2 + mu^2) x^2
```

其中 `mu = E[r]`，`sigma^2 = Var(r)`。最大化这个二次式，得到：

```text
x* = mu / (sigma^2 + mu^2)
```

当 `mu` 相对 `sigma` 很小时，常用近似为：

```text
x* approx mu / sigma^2
```

这个公式与均值-方差优化非常接近。单期均值-方差里，最优头寸也与预期收益成正比、与方差成反比。区别在于，Kelly 的风险厌恶不是外生给定的，而是由复利目标内生产生：仓位越大，亏损对未来财富基数的破坏越严重，因此目标函数自动惩罚过度风险。

Kelly 的另一个重要性质是最优风险规模与 Sharpe Ratio 之间的关系。若策略 Sharpe Ratio 为：

```text
SR = mu / sigma
```

而 Kelly 权重近似为 `mu/sigma^2`，则组合的“美元波动率 / 资本”约为：

```text
x* sigma = mu / sigma = SR
```

也就是说，在 Kelly 规模下，策略承担的年度化波动占资本比例大约等于它的 Sharpe Ratio。一个年化 Sharpe 为 2 的策略，在 full Kelly 下对应的年化美元波动约为资本的 200%。这在机构投资语境下是很大的风险规模，也解释了为什么纯 Kelly 在现实中往往过于激进。

Kelly 的期望对数增长率也可以近似写成：

```text
g* approx 0.5 SR^2
```

这个结果说明，长期复利增长对 Sharpe Ratio 非常敏感。Sharpe 从 1 提高到 2，最大对数增长不是线性翻倍，而是按平方增加。这解释了为什么高质量、低相关、可扩展的 alpha 非常珍贵：它们不仅改善单期风险收益，还允许更高的长期复利速度。

Kelly 最著名的理论性质是长期最优性。在独立同分布、模型正确、没有交易摩擦和外部约束的条件下，Kelly 策略几乎必然以最快的指数速度增长财富。与任何其他固定比例策略相比，它达到足够高财富门槛的期望时间最短。这个性质非常强，因此 Kelly 常被用来描述长期成功投资者或赌徒的资本配置逻辑。

但这些数学性质也容易被误解。Kelly 最大化的是期望对数财富，不是期望终值财富。一个极端高杠杆策略可能有很高的期望终值，因为少数路径赚得极多；但大多数路径可能破产或遭受巨大回撤。Kelly 不追逐这种“均值被尾部拉高”的财富分布，而是在典型长期路径上追求最快增长。

同时，Kelly 并不控制最大回撤。即使 full Kelly 在长期增长意义上最优，它仍然可能经历非常深的回撤，而且回撤概率随着风险比例增加而迅速上升。对数效用会避免确定性破产，却不会保证投资者在现实世界中能承受路径上的损失。资产管理人还面临赎回、止损、业绩排名、投资委员会审查等约束，这些约束不是 Kelly 数学模型的一部分。

因此，Kelly 的数学性质应该这样理解：它给出了在理想条件下增长最大化的风险规模，也揭示了 Sharpe、杠杆和复利增长之间的数量关系。但它不是现实投资的最终答案。现实中更重要的问题往往是：full Kelly 的多少比例，才与投资者的回撤容忍度、参数不确定性和资金稳定性相匹配。

## 小结

单资产 Kelly 在小收益近似下给出 `x* approx mu/sigma^2`，并意味着最优美元波动率占资本比例约等于 Sharpe Ratio。Kelly 的最大长期增长率约为 `0.5 SR^2`，显示高 Sharpe 策略对复利极其有价值。它在理想条件下具有长期增长最优性，但并不最大化期望终值，也不控制回撤，因此现实应用通常需要折扣。
