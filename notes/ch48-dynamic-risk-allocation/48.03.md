# 48.3 Fractional Kelly 与风险缩放

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 13, "Dynamic Risk Allocation"。

如果 full Kelly 在理论上增长最快，为什么现实投资者很少直接使用 full Kelly？答案不是 Kelly 思想错误，而是 full Kelly 对模型正确性、资金稳定性和回撤承受能力要求太高。现实投资者面对的是估计误差、非平稳收益、交易成本、杠杆限制、赎回风险和职业风险。因此，更常见的做法是 fractional Kelly：只使用 full Kelly 仓位的一部分。

设 full Kelly 权重为 `x*`，fractional Kelly 使用：

```text
x_frac = c x*
```

其中 `0 < c < 1`。`c = 1/2` 就是 half Kelly，`c = 1/4` 就是 quarter Kelly。这个做法牺牲一部分长期增长率，换取显著更低的波动和回撤风险。由于对数增长函数在最优点附近是凹的，适度降低仓位通常损失的增长不算太多，但会明显改善路径稳定性。

Fractional Kelly 可以从三个角度理解。

第一，它等价于把资本分成风险资产和现金。假设 full Kelly 策略本身是一个组合，投资者只拿 `c` 的财富投入这个组合，其余持有无风险资产，那么整体就是 `c` 倍 Kelly。若投资者能承受的年度化财富波动上限为 `p`，而策略本身波动为 `sigma`，则风险比例不能超过 `p/sigma`。同时，full Kelly 规模近似对应 `SR/sigma`。因此实际比例会受到风险预算限制：

```text
x = min{p/sigma, SR/sigma}
```

这个表达式的核心不是形式，而是说明风险缩放应由策略质量和投资者承受能力共同决定。高 Sharpe 策略允许更高 Kelly 规模，但投资者的回撤容忍度可能仍然要求折扣。

第二，fractional Kelly 可以看作更高风险厌恶。full Kelly 对应对数效用。若投资者更不愿意承受波动，可以把二次风险惩罚项放大，相当于用更大的风险厌恶参数 `lambda > 1`。此时最优权重变为：

```text
x_frac* = x* / lambda
```

这与均值-方差框架形成直接连接。均值-方差里，风险厌恶越高，最优头寸越小；Kelly 里，折扣 Kelly 就是在增长目标上加入现实中的风险厌恶、约束和路径偏好。

第三，也是最重要的一点，fractional Kelly 是参数不确定性的自然结果。Kelly 对预期收益估计极其敏感。若真实收益为：

```text
r = theta + sigma xi
```

其中 `theta` 是未知期望收益，估计均值为 `mu`，不确定性方差为 `tau^2`，那么在近似下，Kelly 权重会从：

```text
mu / (mu^2 + sigma^2)
```

下降到类似：

```text
mu / (mu^2 + sigma^2 + tau^2)
```

预期收益越不确定，分母越大，最优仓位越小。波动率估计不确定时也有类似效果。由于 Kelly 权重对 `mu` 的误差高度敏感，任何理性的投资者都应该把 full Kelly 当作上限，而不是默认目标。

这点与前面回测和参数估计章节直接相连。回测中的 Sharpe 往往被数据挖掘、幸存者偏差、交易成本低估和样本选择抬高。如果用被高估的 `mu` 或 Sharpe 计算 Kelly，仓位会被系统性放大。换言之，overfitting 在单期回测里表现为虚假的高 Sharpe，在动态风险配置里会表现为过度杠杆和深回撤。Fractional Kelly 是对模型不确定性的一种简单防御。

在实践中，许多机构并不直接说自己使用 fractional Kelly，但实际做法非常接近：先估计策略 Sharpe 和容量，再设定目标波动率、杠杆上限和回撤阈值，然后根据近期表现和风险状态调整资本。只要投资者把“策略风险占资本的比例”控制在一个稳定范围内，就已经在做 Kelly 思想下的风险缩放。

不过，fractional Kelly 也不是越低越好。若仓位过低，策略虽然安全，却无法有效利用 alpha；若仓位过高，长期增长的理论优势可能被回撤和资金流失破坏。合理的折扣比例取决于 alpha 可信度、收益非正态性、尾部风险、交易成本、资金期限和投资者目标。这里没有一个普适常数，只有一个原则：越不确定、越非平稳、越有外部约束，Kelly 折扣越大。

## 小结

Fractional Kelly 是 full Kelly 在现实投资中的稳健版本。它可以理解为现金与 full Kelly 策略的组合，也可以理解为更高风险厌恶或参数不确定性下的最优缩放。由于预期收益和 Sharpe 估计存在误差，full Kelly 往往过于激进。合理的风险缩放应同时反映策略质量、估计可信度和投资者回撤承受能力。
