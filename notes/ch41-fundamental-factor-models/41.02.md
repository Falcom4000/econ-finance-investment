# 41.2 横截面回归与因子收益估计

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 6, Fundamental Factor Models.

基本面模型把因子暴露 `B_t` 视为已知，然后在每个日期用资产收益 `r_t` 反推出因子收益 `f_t`。这与时间序列回归估计股票 beta 的方向相反。时间序列回归通常问：某只股票过去如何随市场波动？横截面回归问：在今天这个横截面上，哪些特征解释了资产之间的收益差异？如果价值暴露高的股票今天普遍上涨，而价值暴露低的股票普遍下跌，那么价值因子的当期收益就是正的。

最基本的估计式仍是

```text
r_t = B_t f_t + epsilon_t
```

其中 `B_t` 是 `n x m` 矩阵，`n` 是资产数，`m` 是因子数。若资产数远大于因子数，横截面回归用大量资产的当期收益估计少量因子收益。普通最小二乘最小化残差平方和；加权最小二乘则最小化

```text
(r_t - B_t f_t)' W_t (r_t - B_t f_t)
```

权重矩阵 `W_t` 的作用，是让不同资产的噪声水平在估计中得到不同对待。高特质波动、低流动性、价格发现差的股票，其单日收益包含更多个体噪声；如果让它们和大型高流动性股票拥有同等权重，因子收益估计会变得更嘈杂。因此实践中常用特质协方差矩阵的逆作为权重，即 `W_t = Omega_epsilon,t^{-1}`。在特质风险近似对角时，这等价于给低特质波动资产更高权重。

在这个权重下，因子收益的估计量为

```text
f_hat_t = (B_t' Omega_epsilon,t^{-1} B_t)^(-1)
          B_t' Omega_epsilon,t^{-1} r_t
```

这条公式背后有几个重要条件。首先，`B_t` 必须满列秩，否则不同因子的暴露组合无法被区分。其次，残差需要满足均值为零、与因子无关、有限方差等基本条件。再次，权重矩阵要能合理描述残差噪声结构。真实市场不会完美满足这些条件，但公式告诉我们哪里会出问题：暴露矩阵共线会导致因子收益不可识别；残差中仍有共同结构会造成遗漏因子；错误权重会把噪声大的资产放得太重。

横截面回归的一个困难是“鸡生蛋”问题。我们希望用 `Omega_epsilon` 加权回归，但 `Omega_epsilon` 本身又是模型估计的输出。实务上通常有几种处理方式。第一种是用已有供应商风险模型的特质风险作为初始权重。第二种是两阶段估计：先用单位权重或简单权重做一次回归，得到残差后估计特质风险，再用新的权重重新回归。第三种是在滚动系统中使用前一日或上一期模型的特质风险作为当期权重。第三种在生产系统中尤其自然，因为风险模型每天更新，上一期估计通常是可得的。

横截面回归还有一个更隐蔽但更根本的问题：因子之间可能线性相关。最典型的例子是国家、行业和截距。假设每只股票都只属于一个行业，那么所有行业哑变量相加等于一列全 1。如果模型同时放入截距和所有行业因子，暴露矩阵必然不满秩。类似地，所有国家哑变量相加也等于一列全 1；如果再加入全球市场因子，就会发生重复。多国家行业模型中，国家因子、行业因子、地区因子和市场因子的关系更复杂，稍不注意就会产生不可识别性。

处理秩亏有三类方法。第一类是删除冗余因子。例如保留截距并删除一个行业，或者不设截距而保留完整行业集合。第二类是使用正则化或广义逆，例如 ridge 回归或 Moore-Penrose 伪逆。这类方法可以产生数值解，但经济解释要更谨慎，因为正则化会把不可识别部分按数学规则分摊。第三类是加入约束，例如要求行业因子收益按市值加权和为零。这样市场因子解释总体收益，行业因子解释相对市场的行业偏离。这种约束往往比任意删除一个行业更符合投资归因逻辑。

从投资角度看，横截面回归估计出来的因子收益具有两种用途。第一，它们是风险模型的原材料。只有积累一段时间的 `f_hat_t`，才能估计因子协方差。第二，它们本身也是研究对象。研究者可以观察价值、动量、质量、规模、行业、国家等因子在不同市场环境中的表现，进而判断某类策略的收益来源。比如，一个声称来自选股能力的组合，如果大部分收益其实来自长期价值暴露，那么绩效评价就必须重新解释。

不过，因子收益不是“真实因子收益”的无误读数。`f_hat_t` 是估计值，会受到暴露定义、估计宇宙、权重、异常值处理和遗漏因子的影响。假如模型没有专门的流动性因子，那么流动性冲击可能被行业、规模或特质残差吸收；假如成长股和科技行业高度重合，行业因子和风格因子的收益解释也可能彼此缠绕。这意味着横截面回归的输出既是量化对象，也是诊断对象。好的风险建模不会只看回归结果是否能跑出来，而会持续检查残差结构、因子相关性和经济解释是否稳定。

这也说明基本面因子模型与前面投资学中的绩效归因有深层联系。绩效归因不是把收益机械分摊给几个标签，而是在一个可识别、可解释、统计上尽量稳健的模型中，把组合收益分解为系统性暴露、主动因子倾斜和个股特质贡献。横截面回归是这个分解的中枢。

## 小结

横截面回归用当期资产收益和已知因子暴露估计因子收益。加权最小二乘能降低高噪声资产对估计的干扰，但需要处理特质风险权重的初始化问题。暴露矩阵的满秩和经济约束非常重要，否则因子收益不可识别或解释混乱。因子收益既是协方差估计的输入，也是绩效归因和 alpha 研究的重要对象。
