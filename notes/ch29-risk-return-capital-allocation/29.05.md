# 29.5 风险资产与无风险资产之间的资本配置

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.6
- 相关旧笔记：本笔记 Ch.7, Ch.8

## 资本配置先于证券选择

投资组合构建可以分成两个层次。第一层是资本配置：在安全资产和风险资产之间分配多少财富。第二层是风险资产内部的组合构建：股票、债券或其他风险证券之间怎样搭配。

本节先看第一层。假设投资者已经找到一个可以接受的风险组合，例如一个股票指数基金或一个混合型基金。现在的问题是：总财富中应该有多少放入这个风险组合，多少放入无风险资产？

这个问题非常实际。一个投资者可能相信股票长期有风险溢价，但仍不应该把所有资金都放入股票。另一个投资者可能很保守，但也未必应该全部持有现金。资本配置的任务，就是在风险组合和安全资产之间找到适合自己的比例。

## 无风险资产在组合中的作用

无风险资产在这里通常用短期国库券或货币市场工具近似。它的特点是短期收益较确定，标准差近似为零。无风险资产不是永远没有风险。它可能有通胀风险、再投资风险或税后实际收益为负的风险。但在一个短期投资期内，它的名义收益通常可以视为已知。

无风险资产在组合中有三种作用。

第一，它提供安全基准。任何风险投资都应解释为什么值得放弃无风险收益。

第二，它调节整体风险。风险组合本身可能有固定波动率，但投资者可以通过减少投入比例，把完整组合的波动率降到自己能承受的水平。

第三，它提供流动性。短期支出、再平衡、保证金需求和机会资金，都需要相对安全、可变现的资产。

货币市场资产在宏观金融中是短期资金市场，在投资组合中则是安全端和流动性端。

## 完整组合：风险资产和无风险资产的混合

设投资者把比例 `y` 投入风险组合 P，把剩余比例 `1 - y` 投入无风险资产 F。这个由风险资产和无风险资产合成的总组合称为完整组合。

如果风险组合的预期收益率是 `E(rP)`，无风险收益率是 `rf`，完整组合的预期收益率为：

```text
E(rC) = yE(rP) + (1 - y)rf
      = rf + y[E(rP) - rf]
```

这个公式很有直觉。完整组合的基础收益是无风险收益率 `rf`。如果投资者把一部分钱放入风险组合，就额外获得这一比例对应的风险溢价 `E(rP) - rf`。

假设无风险收益率为 2%，风险组合预期收益率为 10%。风险溢价是 8%。如果投资者把 50% 财富投入风险组合，完整组合预期收益为：

```text
2% + 0.5 x 8% = 6%
```

如果投入 100%，预期收益为 10%。如果投入 0%，就是 2%。

## 完整组合的风险怎样变化

无风险资产没有收益波动，因此完整组合的波动完全来自风险组合。若风险组合标准差为 `σP`，完整组合标准差为：

```text
σC = yσP
```

如果风险组合标准差为 22%，投资者投入 50%，完整组合标准差为 11%。投入 25%，标准差为 5.5%。投入 100%，标准差为 22%。

这个线性关系很重要。它说明投资者可以用一个既定风险组合，通过改变投入比例，获得一整条不同风险水平的完整组合。风险组合本身可以由基金经理或指数基金提供，而投资者根据自身风险偏好调整持有比例。

这也解释了为什么同一个股票指数基金可以适合不同投资者。激进投资者可以高比例持有，保守投资者可以低比例持有并搭配更多短期国库券。基金本身不需要为每个投资者重新设计，资本配置比例可以完成个性化调整。

## 资本配置线：风险收益组合的可行集合

把完整组合的预期收益率画在纵轴，标准差画在横轴，会得到一条从无风险资产出发、穿过风险组合 P 的直线。这条线称为资本配置线 CAL。

资本配置线的纵截距是无风险收益率，因为当标准差为零时，投资者全部持有无风险资产。资本配置线的斜率是：

```text
斜率 = [E(rP) - rf] / σP
```

这正是风险组合的夏普比率。斜率表示每增加一单位标准差，可以获得多少额外预期收益。斜率越高，风险收益交换越有吸引力。

假设风险组合预期收益为 10%，无风险收益为 2%，标准差为 22%。资本配置线斜率为：

```text
(10% - 2%) / 22% = 0.36
```

这意味着在这条线上，每多承担 1 个百分点标准差，大约获得 0.36 个百分点预期超额收益。投资者选择线上哪一点，取决于风险厌恶程度；但这条线本身描述了市场提供的风险收益机会。

## 借贷和杠杆

如果 `0 < y < 1`，投资者把一部分财富投向风险组合，一部分投向无风险资产。如果 `y = 1`，全部财富投向风险组合。如果 `y > 1`，投资者借入资金并投入风险组合，这就是杠杆。

例如投资者自有资金 300,000 元，又借入 120,000 元，总共 420,000 元投入风险组合。相对于自有资金，风险组合权重为：

```text
y = 420,000 / 300,000 = 1.4
```

这时 `1 - y = -0.4`，表示投资者在无风险资产上是负头寸，也就是借款。杠杆会同时放大预期收益和标准差。如果风险组合收益好，投资者获得更高回报；如果风险组合下跌，损失也按更大资产规模计算。

保证金买入本质上就是借钱提高风险资产权重。杠杆不是一种单独资产，而是一种资本配置方式：把风险资产权重提高到超过 100%。

## 借款利率通常高于贷款利率

理论上，如果投资者可以按无风险利率借入资金，那么资本配置线在 `y > 1` 后仍保持同样斜率。但现实中，普通投资者不能按国库券利率借钱。经纪商会收取更高的保证金贷款利率，银行也会因为借款人违约风险要求更高利率。

一旦借款利率高于无风险贷款利率，资本配置线在风险组合点 P 之后会变得更平。原因是杠杆部分的融资成本更高，额外投入风险组合所获得的净风险溢价下降。

例如无风险贷款利率为 2%，风险组合预期收益为 10%，标准差为 22%。不使用杠杆时，斜率为：

```text
(10% - 2%) / 22% = 0.36
```

如果借款利率为 4%，杠杆部分的斜率变成：

```text
(10% - 4%) / 22% = 0.27
```

这说明杠杆并不是免费提高收益的工具。借款成本、保证金要求和强制平仓风险，都会削弱杠杆吸引力。

资本配置线中的无风险收益率来自货币市场和中央银行政策环境。央行通过政策利率影响短期资金价格，利率变化又会影响债券供求和资产需求。现在这些机制进入投资组合：无风险收益率越高，投资者持有安全资产的吸引力越大；风险资产若不能提供足够风险溢价，就不容易吸引资金。

风险组合的预期收益和波动率则来自资本市场。股票风险溢价、债券期限溢价、信用利差和市场波动，都会影响资本配置线的斜率。投资者不是在真空中选择风险，而是在宏观利率、通胀、经济周期和市场估值共同决定的机会集中选择。

这也解释了为什么同一个投资者在不同市场环境下会改变配置。无风险利率很高时，保守资产更有吸引力；风险溢价很低或市场波动很高时，风险资产的单位风险补偿下降；市场大跌后，如果预期收益上升且风险可承受，风险资产可能重新变得更有吸引力。

不过，无风险利率上升不等于所有投资者都应机械转向现金。长期养老金和保险公司要考虑负债久期，年轻投资者要考虑长期工资收入和通胀风险，基金经理要考虑相对基准和赎回压力。资本配置线给出的是金融市场的机会价格，真正的配置还要叠加投资期限、负债结构、税收账户和流动性需求。

## 小结

资本配置是在无风险资产和风险组合之间分配财富。完整组合的预期收益率等于无风险收益率加上风险资产权重乘以风险溢价；完整组合标准差等于风险资产权重乘以风险组合标准差。

资本配置线展示了通过不同 `y` 值可以获得的风险收益组合。它的斜率等于风险组合的夏普比率，表示单位风险带来的预期超额收益。投资者沿着这条线选择适合自己的风险水平。

杠杆对应 `y > 1`，可以提高预期收益，也会放大风险。现实中借款利率通常高于无风险利率，且存在保证金和强制平仓约束，因此杠杆后的资本配置线会变平。资本配置同时受无风险利率、风险溢价、市场波动和投资者风险承受能力影响。
