# 29.1 收益率的度量

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.5
- 相关旧笔记：本笔记 Ch.7, Ch.9

## 为什么先要把“赚了多少”说清楚

投资讨论中最常见的问题是“收益率是多少”。但这个问题如果不说明口径，很容易产生误解。一个债券半年赚 2.7%，一个长期零息债到期总收益 329%，一个股票一年获得 4 元股利并涨价 10 元，它们都可以说“有收益”，但不能直接拿这些数字比较。

原因很简单：投资收益必须和投入金额、持有时间、现金分配、复利方式、通胀和税收放在一起看。今天的一元和一年后的一元不是同一个价值，名义利率、实际利率和风险利差也对应不同含义。进入投资学后，这些概念需要统一到一个可比较的收益率框架中。

本节先处理最基本的问题：如何度量一个资产在某个持有期内的收益率，如何把不同期限的收益率转化为共同口径，为什么名义收益率和实际收益率不同，以及为什么税收会让通胀对投资者更不利。

这些看似技术性的口径，会直接影响投资判断。若把半年收益率当成一年收益率，投资者会高估或低估机会；若只看名义收益而不看通胀，储蓄者可能以为自己变富，实际购买力却没有增加；若忽略税收，税后可消费财富会被夸大。投资学中的“收益”最终要回到经济学中的消费能力，而不是账户数字本身。

## 持有期收益率：最基本的收益概念

投资者买入资产后，收益通常来自两部分：资产价格变化和期间现金收入。股票可能支付股利，债券可能支付利息，基金可能分配股利和资本利得。持有期收益率就是把这两部分加总后，除以初始投资金额。

```text
持有期收益率 = (期末价格 - 期初价格 + 期间现金收入) / 期初价格
```

假设一只基金年初价格为 100 元，年末价格为 110 元，期间支付 4 元现金分配。投资者的持有期收益率是：

```text
(110 - 100 + 4) / 100 = 14%
```

这个 14% 可以拆成两部分。价格从 100 元涨到 110 元，资本利得率是 10%；期间现金分配为 4 元，现金收益率是 4%。两者相加就是持有期收益率。

这个公式看似简单，却能纠正常见误解。只看价格变化会漏掉股利和利息；只看股利率会漏掉资本利得或资本损失。债券、股票和基金的完整收益，都必须同时考虑价格变化和现金收入。

## 零息债券：用最简单资产理解收益率

为了理解不同期限收益率的比较，可以先看零息债券。零息债券没有中间利息支付，只在到期时支付面值。假设一只零息债券到期支付 100 元，今天价格为 95 元。投资者今天支付 95 元，到期收到 100 元，整个持有期的总收益率是：

```text
100 / 95 - 1 = 5.26%
```

如果这只债券一年后到期，这个 5.26% 就是一年收益率。但如果它半年后到期，问题就变了。半年赚 5.26% 不能直接和一年赚 5.26% 比，因为半年后资金还可以继续再投资。为了比较不同期限投资，必须把它们转化为同一个时间单位，通常是一年。

这就是有效年收益率的意义。有效年收益率问的是：如果每年都按某个固定年化速度增长，最终会得到和实际投资一样的财富倍数，那么这个年化速度是多少？

## 有效年收益率：把不同期限放到同一把尺上

有效年收益率 EAR 明确考虑复利。假设半年收益率为 2.71%。如果一年中可以连续经历两个相同的半年收益过程，1 元会变成：

```text
1 x (1 + 0.0271)^2
```

因此有效年收益率是：

```text
(1 + 0.0271)^2 - 1 = 5.49%
```

如果投资期限长于一年，也要反过来求每年的等价增长率。假设一只 25 年零息债价格为 23.30 元，到期支付 100 元。总财富倍数是：

```text
100 / 23.30 = 4.2918
```

这意味着 25 年总收益率为 329.18%。但这个数字不能直接和一年收益率比较。我们要找一个年收益率，使得：

```text
(1 + EAR)^25 = 4.2918
```

解出来 EAR 约为 6%。也就是说，25 年总共赚 329.18%，等价于每年复利增长约 6%。

现值把未来现金流折回今天，收益率则把今天投资滚到未来。两者是同一件事的两个方向：折现问“未来 100 元今天值多少钱”，年化收益率问“今天投入多少钱，以什么速度增长到未来 100 元”。

## APR 和 EAR：为什么信用卡利率看起来会低估真实成本

市场上常见另一个收益率口径：年百分比利率 APR。APR 通常把每期利率简单乘以一年中的期数，不完全反映复利。

例如，信用卡每月利率为 1.5%。APR 会被写成：

```text
1.5% x 12 = 18%
```

但如果欠款每月按 1.5% 增长，12 个月后不是只增长 18%，而是：

```text
(1 + 0.015)^12 - 1 = 19.56%
```

19.56% 才是有效年利率。APR 没有把“利息继续产生利息”的复利效应完整算进去。借款人如果只看 APR，可能低估真实融资成本；投资者如果只看 APR，也可能误判不同投资工具的实际收益。

APR 和 EAR 的关系可以概括为：

```text
1 + EAR = (1 + APR / n)^n
```

其中 n 是一年中的复利次数。复利越频繁，EAR 和 APR 的差距越大。在连续复利的极限下，就会使用自然指数和连续复利收益率。连续复利在期权定价和资产收益统计中很常见，因为它在数学上更方便。

## 名义收益率和实际收益率

收益率还要区分名义和实际。名义收益率表示货币金额增长了多少，实际收益率表示购买力增长了多少。如果投资收益为 8%，但物价也上涨 5%，投资者的购买力并没有增加 8%，而是大约增加 3%。

近似关系是：

```text
名义收益率 ≈ 实际收益率 + 通胀率
```

更精确的关系是：

```text
1 + 名义收益率 = (1 + 实际收益率) x (1 + 通胀率)
```

名义 GDP 增长可能只是物价上涨，名义工资增长也可能被通胀抵消；投资收益同样如此。投资者真正关心的是未来能多买多少商品和服务，而不是账户上的数字增长多少。

因此，当预期通胀上升时，投资者会要求更高的名义利率，以维持相同的实际收益。费雪假说正是表达这个想法：

```text
名义利率 = 实际利率 + 预期通胀率
```

如果实际利率稳定在 2%，预期通胀从 4% 上升到 5%，名义利率就应大致从 6% 上升到 7%。名义利率上升并不一定意味着实际投资机会更好，它可能只是通胀补偿提高。

## 实际利率如何由储蓄和投资决定

实际利率不是凭空出现的。家庭愿意储蓄，形成资金供给；企业和政府希望借入资金，形成资金需求。实际利率调节储蓄和投资，使资金供给与资金需求相等。

如果家庭更愿意推迟消费，储蓄供给增加，实际利率倾向下降。若企业发现更多高回报实物投资机会，资金需求增加，实际利率倾向上升。政府财政赤字扩大，也会增加借款需求，推高均衡实际利率。中央银行通过货币政策影响短期名义利率和金融条件，也可能在一段时间内影响实际利率。

这个框架把投资收益率和整个经济连接起来。股票和债券收益不是孤立数字，而是建立在真实资本回报、储蓄意愿、财政政策、货币政策和通胀预期之上。安全实际利率也因此成为所有风险资产预期收益的基础。

## 税收为什么会放大通胀伤害

投资者最终拿到的是税后实际收益。税制通常按名义收入征税，而不是只对扣除通胀后的真实收益征税。这会使通胀对储蓄者更不利。

假设投资名义收益率为 12%，通胀率为 8%，税率为 30%。税前实际收益大约是：

```text
12% - 8% = 4%
```

如果只对真实收益征税，税后实际收益应为：

```text
4% x (1 - 30%) = 2.8%
```

但现实中税收按名义收益 12% 征收。税后名义收益是：

```text
12% x (1 - 30%) = 8.4%
```

再扣除 8% 通胀，税后实际收益只剩：

```text
8.4% - 8% = 0.4%
```

也就是说，通胀补偿那部分名义收益也被征税了。投资者明明只是保持购买力，却要为这部分“看起来像收入”的金额缴税。通胀越高、税率越高，税后实际收益被压低得越厉害。

这也是为什么投资者在比较资产时要看税后收益。免税特征能让名义收益较低的市政债对高税率投资者更有吸引力。对长期投资者来说，税收和费用一样，会通过复利持续影响最终财富。

## 收益率口径怎样影响投资判断

把这些概念放在一起，可以看到“收益率”至少有几层口径：

| 口径 | 回答的问题 | 容易误用的地方 |
| --- | --- | --- |
| 持有期收益率 | 某一持有期内总共赚了多少 | 不同期限不能直接比较 |
| 有效年收益率 EAR | 折成年复利口径后赚多少 | 要确认复利假设 |
| 年百分比利率 APR | 按简单年化报价是多少 | 可能低估复利后的真实成本或收益 |
| 名义收益率 | 货币金额增长多少 | 没有扣除通胀 |
| 实际收益率 | 购买力增长多少 | 需要通胀数据或预期 |
| 税后实际收益率 | 纳税后购买力增长多少 | 最接近投资者最终关心的结果 |

投资分析通常从持有期收益率开始，但不能停在那里。长期资产配置、退休规划、债券定价、基金评价和风险溢价估计，都需要把收益率调整到可比较口径。一个看似高收益的资产，如果期限更长、通胀更高、税负更重、费用更高，实际对投资者的吸引力可能并不高。

这一点也把投资学和宏观经济联系起来。宏观统计中的名义变量和实际变量不能混用，投资收益也一样；货币政策改变的是名义短端利率，但投资者最终关心的是税后实际购买力；金融机构报出的贷款利率、债券收益率和基金收益率，则分别嵌入了期限、信用、流动性、税收和费用。收益率口径统一，是后面比较风险溢价、评估基金绩效和做资产配置的前提。

## 小结

收益率度量是投资学的基础。持有期收益率把价格变化和现金收入合并，反映某一持有期内的完整投资结果。为了比较不同期限投资，需要把收益率转化为有效年收益率，明确考虑复利。APR 是常见报价口径，但它不完全等同于 EAR，复利越频繁，两者差距越明显。

投资者真正关心的是购买力，因此需要区分名义收益率和实际收益率。费雪关系说明，名义利率大致等于实际利率加预期通胀率。实际利率可以从可贷资金市场理解，由储蓄供给、投资需求、财政政策和货币政策共同影响。

税收进一步改变投资者的最终结果。由于税收通常按名义收益征收，通胀补偿部分也可能被征税，导致税后实际收益显著下降。比较投资机会时，最有意义的是在统一期限、统一复利、扣除通胀和税收后的可比收益。
