# 29.4 风险厌恶、效用与投资者选择

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.6
- 相关旧笔记：本笔记 Ch.1, Ch.4

## 为什么最高预期收益不一定是最好选择

如果只看预期收益，投资问题似乎很简单：选择预期收益最高的资产。但现实中，投资者不会这样决策。一个预期收益 15%、但可能亏掉一半本金的组合，未必比一个预期收益 6%、波动很小的组合更适合所有人。

原因在于投资者通常是风险厌恶的。风险厌恶不是说投资者永远不承担风险，而是说在其他条件相同的情况下，投资者不喜欢不确定性；如果要承担更多风险，就需要更高的预期收益作为补偿。

投资者不是单纯最大化收益，而是在收益和风险之间做选择。更高预期收益通常不是免费得到的，它伴随更大波动、更大亏损可能、更长恢复时间和更强心理压力。

## 投机和赌博的区别

投资学区分投机和赌博。两者都涉及不确定结果，但含义不同。

**投机**是为了获得足够补偿而承担风险。投资者知道某项资产有风险，但认为它提供的风险溢价足以弥补这种风险。例如股票市场波动很大，但如果长期预期收益显著高于国库券，风险厌恶投资者仍可能愿意持有股票。

**赌博**是为了风险本身或娱乐而承担风险，通常没有足够的风险溢价。一个公平游戏的期望收益为零，参与者承担风险却没有正的预期补偿。风险厌恶投资者会拒绝公平赌博，因为它增加不确定性，却不提高预期财富。

这个区分很重要。说投资者风险厌恶，并不意味着他们只买无风险资产。只要风险溢价足够高，风险厌恶投资者也会持有风险资产。风险厌恶要求的是“风险必须有价格”，而不是“风险必须为零”。

## 用效用函数表达风险收益权衡

为了比较不同投资组合，可以把预期收益和风险合成一个效用分数。常用形式是：

```text
U = E(r) - 1/2 x A x σ^2
```

其中 `E(r)` 是组合预期收益率，`σ^2` 是收益率方差，`A` 是投资者风险厌恶系数。这个公式的含义很直接：预期收益越高，效用越高；风险越高，效用越低；投资者越厌恶风险，同样的风险带来的效用扣分越大。

如果组合是无风险资产，方差为 0，效用就等于确定收益率。风险资产则要从预期收益中扣除一项风险惩罚。风险惩罚的大小取决于两个因素：资产本身有多波动，以及投资者有多讨厌波动。

这个公式不是说人的心理真的能被一个简单公式完整描述。它的作用是给风险收益权衡一个可操作框架。只要投资者偏好“高预期收益、低风险”，类似效用函数就能帮助我们比较组合。

## 确定性等价：把风险组合换算成确定收益

效用分数还有一个直观解释：它可以看成风险组合的**确定性等价收益率**。确定性等价是这样一个无风险收益率：如果投资者能确定获得这个收益，他会觉得它和某个风险组合一样有吸引力。

假设一个风险组合预期收益为 10%，标准差为 20%。如果某投资者风险厌恶系数为 3，那么效用为：

```text
U = 0.10 - 1/2 x 3 x 0.20^2
  = 0.10 - 0.06
  = 0.04
```

这个组合对该投资者来说，等价于一个确定收益率为 4% 的无风险资产。如果当前无风险收益率为 2%，他会愿意持有这个风险组合，因为它的确定性等价高于无风险收益率。如果另一个投资者风险厌恶系数更高，例如 A = 5，同一组合效用为：

```text
U = 0.10 - 1/2 x 5 x 0.20^2
  = 0
```

对这个更保守的投资者来说，同一风险组合只等价于 0% 的确定收益，低于 2% 无风险收益率，因此他会拒绝。

这说明，同一资产没有对所有人都“好”或“坏”。资产是否合适，取决于风险溢价、波动率和投资者风险厌恶程度。

## 风险厌恶、风险中性和风险偏好

不同投资者对风险的态度不同。

风险厌恶者的 `A > 0`。他们会给风险扣分，要求风险资产提供正的风险溢价。大多数投资者、养老金、保险公司和家庭储蓄者都可以看作风险厌恶者。

风险中性者的 `A = 0`。他们只看预期收益，不关心波动。如果两个资产预期收益相同，他们对安全资产和高风险资产无所谓。风险中性在某些定价模型中很有用，但并不适合描述多数真实投资者的财富选择。

风险偏好者的 `A < 0`。他们喜欢风险本身，会把波动当作正面效用来源。纯赌博行为更接近这种偏好。但在严肃投资理论中，风险偏好不是基准假设，因为如果投资者普遍偏好风险，就很难解释为什么低风险资产仍被大量持有。

在现实中，一个人也不是固定不变的风险厌恶系数。年龄、财富、收入稳定性、家庭负债、健康状况、投资经验和市场环境都会影响风险承受能力。接近退休的人通常比年轻且收入稳定的人更难承受长期亏损；有高杠杆房贷和不稳定收入的人，也可能需要更保守的金融资产配置。

## 均值-方差准则

如果用预期收益率和标准差描述组合，那么投资者偏好的方向很清楚：预期收益越高越好，标准差越低越好。在一张横轴为标准差、纵轴为预期收益的图上，越往西北方向越好，因为那意味着更低风险和更高收益。

这就是均值-方差准则。组合 A 支配组合 B，如果：

```text
E(rA) >= E(rB)
σA <= σB
```

并且至少一个不等号是严格的。也就是说，A 的预期收益不低于 B，风险不高于 B，并且至少在一个维度上更好，那么没有理由选择 B。

难点在于，许多组合之间不存在这种简单支配关系。一个组合预期收益更高但风险也更高，另一个组合预期收益较低但更稳定。这时就需要投资者的风险厌恶程度来决定选择。保守投资者可能选择低风险组合，激进投资者可能选择高风险组合。

## 无差异曲线：同样满意的风险收益组合

无差异曲线表示让投资者获得同样效用的不同风险收益组合。沿着同一条无差异曲线，风险越高，就必须有更高预期收益来补偿，否则投资者不会觉得同样满意。

风险厌恶程度越高，无差异曲线越陡。因为同样增加一点标准差，保守投资者要求更多额外预期收益作为补偿。风险承受能力较强的投资者无差异曲线较平，同样的风险增加只需要较少补偿。

这个图形语言可以帮助理解资产配置。市场提供一组可选的风险收益组合，投资者有自己的无差异曲线。最优选择是市场机会集和投资者最高可达到无差异曲线相切的位置。换成直觉说法，就是在市场能提供的投资机会中，选择最符合自己风险偏好的那一点。

## 风险承受能力不只是心理问题

很多投资问卷会问：如果市场下跌 30%，你会卖出、持有还是买入？这类问题试图衡量心理风险承受能力。但真正的风险承受能力还包括经济承受能力。

一个投资者可能心理上很激进，但如果他未来两年需要用钱买房、支付学费或偿还债务，那么他不适合把大部分资金放在高波动资产中。相反，一个年轻投资者收入稳定、短期支出压力小，即使心理上不喜欢波动，也可能有能力承担较多长期股票风险。

因此，风险厌恶至少有两个维度：

| 维度 | 关注的问题 | 例子 |
| --- | --- | --- |
| 风险容忍度 | 心理上能否承受波动 | 亏损时是否恐慌卖出 |
| 风险承受能力 | 财务上能否承受损失 | 是否接近退休、是否有短期现金需求 |

合理投资建议必须同时考虑这两者。只看心理问卷可能忽略现金流约束，只看年龄和收入也可能忽略投资者在市场下跌时的行为反应。

风险厌恶可以从边际效用递减理解。财富越少时，一元钱对生活的影响越大；财富越多时，额外一元钱带来的满足感较小。因此，同样金额的损失带来的痛苦，通常大于同样金额收益带来的快乐。一个公平赌博虽然期望财富不变，但会增加财富波动，降低风险厌恶者的预期效用。

这也解释了保险市场为什么存在。购买保险通常是负期望收益交易：保费中包含保险公司成本和利润。但风险厌恶者愿意支付保费，把少概率大损失转化为确定的小成本。投资中的风险管理也类似，投资者有时愿意牺牲一部分预期收益，换取更平稳的财富路径。

从宏观角度看，风险厌恶也是风险溢价存在的基础。如果所有投资者都风险中性，风险资产只需要按预期现金流折现，不需要额外风险补偿。但现实中投资者要求补偿，因此股票、长期债券、信用债和其他风险资产才会相对于安全资产提供风险溢价。

风险厌恶还解释了保险、养老金和资产配置为什么是同一类问题的不同制度形式。保险把少概率大损失转化为确定保费，养老金把工作期收入转化为退休期消费，资产配置则在安全资产和风险资产之间安排未来财富路径。三者都不是追求单期最高收益，而是在不确定世界中平滑消费、降低坏状态下的损失。

## 小结

投资者通常是风险厌恶的：他们愿意承担风险，但要求获得足够风险溢价。投机是为了获得相应补偿而承担风险，赌博则是承担没有足够补偿的风险。风险厌恶者会拒绝没有正风险溢价的公平游戏。

效用函数把预期收益和风险结合起来。常用形式 `U = E(r) - 1/2 x A x σ^2` 表示预期收益提高效用，风险降低效用，风险厌恶系数越高，风险惩罚越大。效用值可以理解为确定性等价收益率。

均值-方差准则说明，投资者偏好更高预期收益和更低标准差。无差异曲线则刻画不同投资者在风险和收益之间的取舍。实际风险承受能力既取决于心理容忍度，也取决于收入、财富、投资期限和现金需求。
