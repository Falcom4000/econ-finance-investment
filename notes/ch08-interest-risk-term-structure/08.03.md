# 8.3 预期通胀、费雪效应与商业周期

来源：

- 主线：Mishkin《货币金融学》Ch.5, Ch.6
- 补充：Mishkin/Eakins Ch.4, Ch.5

## 为什么预期通胀会影响利率

利率是跨时间借贷的价格。借款人今天得到购买力，未来用货币金额偿还；贷款人今天放弃购买力，未来收到货币金额。只要未来物价水平可能变化，借贷双方真正关心的就不只是货币金额，而是这些金额能买到多少商品和服务。

如果预期未来通胀较高，固定金额的债券付款在未来购买力会下降。对债券投资者来说，这降低了债券的实际吸引力。对借款人来说，未来用贬值后的货币偿还债务，真实成本下降。因此，预期通胀会同时影响债券需求和债券供给。

这个机制最终会推高名义利率。预期通胀上升，名义利率通常也会上升。这种关系称为**费雪效应**。

## 预期通胀如何降低债券需求

债券通常承诺未来支付固定货币金额。假设一只债券未来每年支付固定利息，并在到期时偿还固定面值。如果投资者预期未来物价上涨更快，那么这些固定金额的实际购买力会下降。

例如，一年后收到 1000 元。如果预期通胀率是 2%，这 1000 元的购买力损失较小；如果预期通胀率是 10%，同样 1000 元能买到的商品和服务更少。债券的名义付款不变，但实际收益下降。

从资产需求角度看，预期通胀上升会降低债券相对于实物资产的预期收益。房屋、汽车和其他实物资产的名义价格可能随通胀上升，持有这些资产可能获得名义资本利得。相比之下，固定金额债券的真实回报变差。

因此，在每一个给定债券价格和利率下，投资者愿意持有的债券数量减少，债券需求曲线向左移动。

```mermaid
flowchart LR
    A[预期通胀上升] --> B[固定债券付款的实际购买力下降]
    B --> C[债券相对预期收益下降]
    C --> D[债券需求减少]
    D --> E[需求曲线左移]
```

需求减少会给债券价格带来下行压力。由于债券价格和利率反向变动，价格下降意味着利率上升。

## 预期通胀如何增加债券供给

预期通胀不仅影响投资者，也影响借款人。

企业或政府发行债券时，承诺未来支付固定货币金额。如果预期通胀上升，未来偿还这些固定金额的实际成本下降。借款人用未来购买力较低的货币还债，真实负担变轻。

因此，在每一个给定债券价格和利率下，借款人更愿意发行债券。债券供给增加，供给曲线向右移动。

```mermaid
flowchart LR
    A[预期通胀上升] --> B[未来固定债务的实际偿还成本下降]
    B --> C[借款更有吸引力]
    C --> D[债券供给增加]
    D --> E[供给曲线右移]
```

供给增加会给债券价格带来下行压力。同样，由于价格和利率反向变动，债券价格下降意味着利率上升。

## 费雪效应：预期通胀上升推高名义利率

把需求和供给放在一起，预期通胀上升有两个方向明确的影响：

第一，债券需求减少，需求曲线左移。

第二，债券供给增加，供给曲线右移。

两个变化都会压低债券价格。债券价格下降，对应的均衡利率上升。因此，预期通胀上升会导致名义利率上升。这就是费雪效应。

可以用一句话概括：

```text
预期通胀上升 -> 债券需求减少、债券供给增加 -> 债券价格下降 -> 名义利率上升
```

这一结论与费雪方程相互呼应。费雪方程说明：

```text
名义利率 = 实际利率 + 预期通胀率
```

如果实际利率大体不变，预期通胀率上升，名义利率就会上升。债券市场供求分析则解释了这个结果背后的市场过程：投资者因实际回报下降而减少债券需求，借款人因实际借款成本下降而增加债券供给，最终利率上升。

## 费雪效应不是说所有利率只由通胀决定

费雪效应说明预期通胀和名义利率有重要正向关系，但不能误解为“利率只由通胀决定”。利率还受储蓄、投资机会、货币政策、风险、流动性、财政赤字和经济周期等因素影响。

如果预期通胀上升，同时经济衰退导致投资需求大幅下降，利率变化可能受到多种力量共同影响。债券市场中，需求曲线和供给曲线可能同时受其他因素推动。实际观察到的利率，是所有因素综合作用的结果。

费雪效应的意义在于，它指出了一个方向明确、非常重要的机制：若其他因素不变，预期通胀上升会推高名义利率。历史数据中，短期国债利率通常与预期通胀率同向变动，也体现了这一关系。

政策含义也很直接：如果希望长期保持较低名义利率，就不能长期容忍较高预期通胀。通胀预期一旦上升，市场会要求更高名义利率来补偿购买力损失。

## 商业周期扩张如何影响债券供给

商业周期扩张时，经济中生产的商品和服务增加，收入上升，企业销售前景改善。企业更可能看到有利可图的投资机会，例如购买设备、扩建工厂、增加库存、开发新产品。

投资机会增加时，企业需要更多外部融资。发行债券是融资方式之一。因此，在每一个给定债券价格下，企业愿意发行更多债券。债券供给曲线向右移动。

供给曲线右移会压低债券价格，推高利率。直觉上，经济扩张时企业借款需求更强，资金需求上升，借贷价格也容易上升。

```mermaid
flowchart LR
    A[商业周期扩张] --> B[投资机会增加]
    B --> C[企业融资需求上升]
    C --> D[债券供给增加]
    D --> E[债券价格下降]
    E --> F[利率上升]
```

这条机制解释了为什么经济扩张时期利率常有上升压力。

## 商业周期扩张也会影响债券需求

经济扩张不仅提高企业融资需求，也提高家庭和机构财富。收入增加、利润改善、资产价值上升，会让投资者有更多资源购买金融资产。按照资产需求理论，财富增加会提高债券需求。

因此，商业周期扩张也会使债券需求曲线向右移动。需求增加会推高债券价格，压低利率。

这与供给效应方向相反：

| 商业周期扩张带来的变化 | 对债券价格的影响 | 对利率的影响 |
| --- | --- | --- |
| 企业投资机会增加，债券供给右移 | 价格下降 | 利率上升 |
| 财富增加，债券需求右移 | 价格上升 | 利率下降 |

因此，从理论上说，商业周期扩张对利率的影响并不必然只有一个方向。关键取决于供给曲线和需求曲线哪个移动更大。

如果企业融资需求增加非常强，债券供给右移幅度大于需求右移幅度，债券价格下降，利率上升。如果财富增加带来的债券需求上升更强，债券价格可能上升，利率下降。

原书图示通常把商业周期扩张画成供给右移幅度大于需求右移幅度，于是均衡债券价格下降、利率上升。这符合许多扩张期中投资需求强、利率上升的直觉。但严格地说，供求分析本身告诉我们：方向取决于两条曲线相对移动幅度。

## 衰退时的反向逻辑

经济衰退时，机制大体反向。

企业投资机会减少，融资需求下降，债券供给减少，供给曲线向左移动。这会推高债券价格、压低利率。

同时，收入和财富下降，投资者可配置财富减少，债券需求下降，需求曲线向左移动。这会压低债券价格、推高利率。

因此，衰退对利率的影响同样取决于两种力量的相对大小。很多衰退中，投资需求下降、中央银行放松政策、避险需求上升等因素会使利率下降，但具体结果要结合当时市场条件分析。

本节的重点不是背诵“扩张一定升息、衰退一定降息”，而是掌握供求分析方法：先判断需求曲线和供给曲线分别如何移动，再根据债券价格和利率反向关系得出结论。

## 小结

预期通胀上升会降低固定金额债券付款的实际购买力，使债券相对预期收益下降，债券需求曲线左移。同时，预期通胀上升会降低借款人未来偿还固定债务的实际成本，使债券供给曲线右移。两种力量都会压低债券价格、推高名义利率，这就是费雪效应。

费雪效应说明预期通胀和名义利率通常同向变化，但不意味着利率只由通胀决定。利率还受经济周期、投资机会、财富、风险、流动性和政策因素影响。

商业周期扩张会同时影响债券供给和需求。投资机会增加使企业更愿意发行债券，供给右移，利率有上升压力；财富增加使投资者更愿意持有债券，需求右移，利率有下降压力。最终利率变化取决于两种力量谁更强。
