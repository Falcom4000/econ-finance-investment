# 8.6 收益率曲线与期限结构理论

来源：

- 主线：Mishkin《货币金融学》Ch.5, Ch.6
- 补充：Mishkin/Eakins Ch.4, Ch.5

## 什么是期限结构

同一发行人、同样违约风险、同样流动性、同样税收待遇的债券，仍然可能因为到期期限不同而有不同利率。短期国债、三年期国债、十年期国债和三十年期国债通常不会有完全相同的收益率。

不同期限债券利率之间的关系，称为**利率的期限结构**。把相同风险、流动性和税收特征的债券收益率按期限画出来，就得到**收益率曲线**。

更精确地说，期限结构最好用零息债的**即期利率**来描述。即期利率是从今天到某个未来期限的一次性投资收益率。附息债券的到期收益率会把不同期限现金流压缩成一个平均数，而即期利率曲线可以分别折现每一期现金流。

收益率曲线常见形状有三种：

| 曲线形状 | 含义 |
| --- | --- |
| 向上倾斜 | 长期利率高于短期利率 |
| 平坦 | 长短期利率接近 |
| 向下倾斜或倒挂 | 长期利率低于短期利率 |

收益率曲线为什么通常向上倾斜？为什么有时会平坦或倒挂？为什么不同期限利率常常一起变动？这些问题需要期限结构理论解释。

## 一个好理论必须解释三件事

期限结构理论至少要解释三个经验事实。

第一，不同期限债券的利率通常会一起变动。短期利率上升时，长期利率也往往上升；短期利率下降时，长期利率也常下降。

第二，当短期利率很低时，收益率曲线更可能向上倾斜；当短期利率很高时，收益率曲线更可能向下倾斜或倒挂。

第三，收益率曲线通常向上倾斜。长期利率多数时候高于短期利率。

三种理论试图解释这些事实：预期理论、分割市场理论、流动性溢价理论。前两种各有优点和缺陷，第三种把两者的关键思想结合起来，因此更常被接受。

## 预期理论：长期利率是未来短期利率的平均

**预期理论**认为，长期债券利率等于该长期债券存续期间市场预期的短期利率平均值。

假设投资者有两个选择：

第一，买一只两年期债券并持有到期。

第二，先买一只一年期债券，到期后再买下一只一年期债券。

如果不同期限债券是完全替代品，投资者不会接受其中一个策略的预期回报低于另一个策略。于是，两年期利率必须等于今天一年期利率和预期明年一年期利率的平均。

例如，今天一年期利率是 9%，市场预期明年一年期利率是 11%。那么两年期利率应为：

```text
(9% + 11%) / 2 = 10%
```

更一般地，n 年期利率等于未来 n 个一年期利率预期值的平均。

```text
n 年期利率 = 未来 n 个短期利率预期的平均值
```

这条理论给收益率曲线一个清晰解释。如果收益率曲线向上倾斜，说明市场预期未来短期利率会上升；如果收益率曲线倒挂，说明市场预期未来短期利率会下降；如果曲线平坦，说明市场预期短期利率大体不变。

同一逻辑也可以用**远期利率**表达。远期利率是今天价格中隐含的未来某一段期间利率。例如，一年期即期利率 `s1` 和两年期即期利率 `s2` 隐含第二年的一年期远期利率 `f2`：

```text
(1 + s2)^2 = (1 + s1)(1 + f2)
```

远期利率不是对未来短期利率的纯预测，它还可能包含期限溢价。但它把收益率曲线中“市场如何给未来资金定价”这件事表达得更清楚。

## 预期理论能解释什么，不能解释什么

预期理论能解释不同期限利率为什么一起变动。如果今天短期利率上升，人们通常也会提高对未来短期利率的预期。长期利率是未来短期利率平均值，因此长期利率也会上升。

它也能解释为什么短期利率低时曲线更可能向上倾斜。短期利率很低时，人们往往预期它未来回到正常水平，未来短期利率平均值高于当前短期利率，长期利率就高于短期利率。短期利率很高时，人们往往预期它未来下降，长期利率可能低于当前短期利率，收益率曲线倒挂。

但预期理论解释不了一个重要事实：收益率曲线通常向上倾斜。按预期理论，如果短期利率未来上升和下降的可能性差不多，曲线平均来说应该接近平坦，而不是经常向上倾斜。

问题出在它的关键假设：不同期限债券是完全替代品。现实中，投资者往往并不把所有期限债券看成完全一样。

## 分割市场理论：不同期限各有自己的市场

**分割市场理论**走向另一个极端。它认为不同期限债券的市场彼此分割，某一期限债券利率由该期限债券自身的供给和需求决定，不受其他期限债券预期收益影响。

这个理论的关键假设是，不同期限债券不是替代品。投资者有明确的期限偏好。例如，需要短期资金周转的人偏好短期债券，因为短期债券利率风险较小；为孩子多年后教育储蓄的人可能更愿意持有长期债券以匹配未来支出时间。

如果多数投资者厌恶利率风险并偏好短期债券，长期债券需求相对较低，长期债券价格较低，长期利率较高。这样，收益率曲线通常向上倾斜。

分割市场理论能解释第三个事实：收益率曲线为什么通常向上倾斜。但它无法很好解释不同期限利率为什么常常一起变动。如果各期限市场完全分割，短期利率变化没有理由系统影响长期利率。它也难解释短期利率高低与曲线形状之间的规律关系。

## 流动性溢价理论：长期利率等于预期平均值加期限溢价

**流动性溢价理论**把预期理论和分割市场理论结合起来。它承认不同期限债券可以相互替代，但不是完全替代。长期利率既受未来短期利率预期影响，也包含投资者持有长期债券所要求的额外补偿。

投资者通常更偏好短期债券，因为短期债券利率风险较小。长期债券价格对利率变化更敏感，持有长期债券需要承担更大价格风险。因此，投资者通常要求正的**流动性溢价**或**期限溢价**，才愿意持有长期债券。

因此：

```text
长期利率 = 未来短期利率预期平均值 + 流动性溢价
```

期限越长，流动性溢价通常越大。这样，即使市场预期未来短期利率不变，长期利率也可能高于短期利率，收益率曲线通常向上倾斜。

与它接近的是**优先栖息地理论**。这个理论认为投资者偏好某些期限的债券，只有当其他期限债券提供足够高的预期回报时，投资者才愿意离开自己偏好的期限。它得出的结论与流动性溢价理论相近：长期债券通常需要提供期限溢价。

## 三种理论的比较

可以把三种理论放在同一张表里。

| 理论 | 核心假设 | 能解释 | 主要不足 |
| --- | --- | --- | --- |
| 预期理论 | 不同期限债券是完全替代品 | 不同期限利率一起变动；短期利率高低与曲线形状有关 | 解释不了曲线通常向上倾斜 |
| 分割市场理论 | 不同期限债券完全不是替代品 | 曲线通常向上倾斜 | 解释不了不同期限利率一起变动 |
| 流动性溢价理论 | 不同期限债券可替代但不完全替代 | 三个经验事实都能解释 | 需要判断期限溢价大小 |

流动性溢价理论之所以更有说服力，是因为它同时保留了两点：长期利率反映未来短期利率预期；长期债券还必须补偿投资者承担的期限和利率风险。

## 如何用曲线形状读市场预期

在流动性溢价理论下，收益率曲线仍然包含市场对未来短期利率的预期，但读法要考虑期限溢价。

如果收益率曲线陡峭向上，说明长期利率远高于短期利率。由于期限溢价本来就使曲线有上升倾向，陡峭上升通常表示市场预期未来短期利率会上升。

如果收益率曲线只是温和向上，可能表示市场预期未来短期利率变化不大。曲线向上的部分可能主要来自期限溢价。

如果收益率曲线平坦，说明市场可能预期未来短期利率会适度下降。因为正的期限溢价本应让长期利率高一些，若曲线仍然平坦，预期短期利率下降抵消了期限溢价。

如果收益率曲线倒挂，说明市场可能预期未来短期利率大幅下降。只有预期短期利率下降幅度足够大，才能压过正的期限溢价，使长期利率低于短期利率。

| 曲线形状 | 按流动性溢价理论的含义 |
| --- | --- |
| 陡峭向上 | 预期短期利率上升 |
| 温和向上 | 预期短期利率大体稳定 |
| 平坦 | 预期短期利率适度下降 |
| 倒挂 | 预期短期利率大幅下降 |

这为下一节理解收益率曲线的宏观含义打基础。

## 小结

期限结构描述不同到期期限债券利率之间的关系，收益率曲线是期限结构的图形表示。期限结构理论要解释三个事实：不同期限利率通常一起变动；短期利率低时曲线更可能向上，短期利率高时曲线更可能倒挂；收益率曲线通常向上倾斜。

预期理论认为长期利率等于未来短期利率预期平均值，能解释前两个事实，但解释不了曲线通常向上倾斜。分割市场理论认为不同期限市场相互分割，能解释曲线通常向上倾斜，但解释不了不同期限利率一起变动。流动性溢价理论把两者结合，认为长期利率等于未来短期利率预期平均值加期限溢价，因此能更好解释现实。

收益率曲线的形状可以帮助判断市场对未来短期利率的预期。陡峭向上通常表示预期短期利率上升，平坦或倒挂则常表示预期短期利率下降。即期利率和远期利率进一步把曲线转化为现金流折现工具，使期限结构可以直接用于债券定价、久期管理和利率风险对冲。
