# 64.1 Tokenization 如何把资产权利搬到链上

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.9；补充参考本笔记 Ch.21、Ch.22、Ch.24、Ch.33、Ch.35、Ch.57-Ch.63。

资产代币化指把某种资产的权利，用区块链上的 token 表示出来。这里的关键不是“把房子、债券或基金真的放进链里”，而是把与这些资产相关的所有权、收益权、转让权或使用权，变成可以在链上记录、转让和组合的数字凭证。

## 代币化首先是权利映射

任何金融资产都不是单纯的名称，而是一组权利。债券代表收取利息和本金的债权，股票代表剩余收益和治理权，基金份额代表对组合净资产的一部分索取权，房地产权益可能代表租金和处置收益，碳信用代表某种减排权益。

Tokenization 的第一步，是说清楚 token 对应哪一种权利。持有人拥有底层资产所有权吗？拥有收益分配权吗？拥有赎回权吗？能否参与治理？能否转让？如果发行人破产，持有人在法律上排第几位？

若这些权利没有定义清楚，链上 token 只是一个交易符号，而不是可靠资产凭证。第 57 章讲 token 经济权利时已经看到，估值必须从权利结构开始。

这里还要区分现金流权利和控制权利。一个 token 可能只让持有人分享租金、利息或基金收益，却不允许持有人决定资产出售、贷款展期或管理人更换；另一个 token 可能附带治理投票，但投票结果是否能约束链下公司或基金，又取决于公司章程、合伙协议和当地法律。金融学中常把资产看成未来现金流的索取权，但现实投资还要看谁有权改变这些现金流的分配方式。

因此，代币化文档中最重要的部分往往不是合约代码，而是权利说明、资产文件和违约处理规则。读者可以把它看成一份数字化的证券说明书：token 只是入口，真正决定价值的是入口背后的合同。

## 链上凭证和链下资产的桥

原生数字资产如 ETH、NFT 或某些协议 token，本身就在链上存在。转移 token 就等于转移链上状态。但现实世界资产不同。房屋、债券、贷款、碳信用和知识产权存在于法律、登记、合同和物理世界中。链上 token 只能代表对这些资产的权利，而不能自动改变链下事实。

因此，RWA 代币化必须建立一座桥：链上 token 与链下法律文件、托管安排、登记系统、审计、资产管理人和监管规则对应起来。谁保管资产？谁负责收入分配？谁确认资产真实存在？谁处理违约？谁执行赎回？

这正是代币化最容易被误解的地方。区块链可以提高登记和转让效率，但不能替代所有权法律、资产托管和违约执行。

桥的强弱决定 token 的信用质量。若底层资产由独立托管人保管，定期审计，持有人权利写入法律文件，并且存在明确赎回路径，token 与资产之间的对应关系较强。若项目只声称“有资产支持”，却没有独立托管、审计和法律追索，token 与资产之间的桥就很弱。稳定币章节已经展示过同一逻辑：链上余额看似清楚，但储备资产、托管银行和赎回承诺才决定它能否稳定。

## 为什么要代币化

代币化的潜在价值来自几个摩擦的降低。第一，登记和转让可以更自动化，减少人工对账。第二，结算可以更快，从传统 T+1 或 T+2 接近实时或同日。第三，资产可以被分割成更小单位，降低投资门槛。第四，链上凭证可以和 DeFi 协议组合，用作抵押品或进入交易池。

这些优势对债券、私募基金、房地产、碳信用和 IP 等资产尤其有吸引力。许多这类资产在传统市场中流动性差、交易单位大、转让流程复杂、信息不透明。代币化试图把它们变得更容易登记、分割和交易。

但效率改善不等于风险消失。底层资产仍然可能违约、估值错误、租金下降、碳信用质量不足或 IP 收入不达预期。Token 只是权利载体，不是收益保证。

从经济学角度看，代币化的价值来自交易成本下降。交易成本包括寻找交易对手、确认产权、谈判合同、结算交割、监督履约和处理纠纷。区块链最擅长改善的是登记、转让和部分自动执行，最不擅长的是判断链下资产质量、处理欺诈和解决法律争议。理解这一边界，才能避免把所有市场摩擦都想象成一个技术问题。

## 代币化和证券化的关系

证券化把贷款、应收账款或其他现金流打包成证券出售给投资者。代币化有时像证券化，因为它也把资产权利变成可转让凭证。但两者重点不同。证券化强调现金流打包、分层和信用结构；代币化强调数字登记、链上转让、可组合和自动结算。

两者可以结合。一个贷款组合可以先被证券化，再用 token 表示证券份额；一个基金份额可以本来就是证券，再用链上 token 做登记和转让；一个债券可以原生以数字证券形式发行。

因此，代币化不是完全新发明，而是把既有证券、基金和资产权利放入新的技术和结算环境。

## 小结

资产代币化的核心是把资产权利映射到链上 token。对原生数字资产，token 本身就是资产状态；对现实世界资产，token 必须连接链下法律、托管、登记、审计和执行。代币化可能降低登记、转让、结算和分割成本，但不能消除底层资产风险。分析任何 RWA token，第一步都要问：它到底代表什么权利，这些权利如何被法律和运营流程保障？
