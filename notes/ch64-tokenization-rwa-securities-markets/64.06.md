# 64.6 代币化资产与传统资本市场的连接方式

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.9；补充参考本笔记 Ch.5、Ch.21-Ch.24、Ch.33、Ch.35、Ch.58-Ch.63。

代币化资产的长期价值，不取决于它是否听起来新，而取决于它能否和传统资本市场连接。资本市场已经有发行人、投资者、托管、清算、监管、评级、会计和法律体系。RWA 要扩大规模，必须和这些制度接口兼容，而不是只在加密圈内部循环。

## 发行人为什么会使用代币化

发行人使用代币化，通常希望降低发行成本、缩短结算时间、扩大投资者基础、提高二级流动性或实现更自动化的合规管理。数字债券、链上基金份额、房地产收益权和贷款 token 都可能围绕这些目标设计。

但发行人会比较成本。若传统债券市场已经便宜、流动、投资者基础广，代币化必须提供额外价值。若代币化需要额外法律结构、技术开发、审计和合规成本，而投资者需求不足，发行人未必愿意采用。

因此，RWA 采用不会只由技术推动，而由成本收益决定。

不同发行人的动机会不同。大型优质发行人可能把代币化当作结算和登记试验，因为它们已经能在传统市场低成本融资；中小发行人可能更看重触达新投资者；资产管理人可能希望把基金份额嵌入链上现金管理；房地产或私募项目可能希望降低最低投资门槛。动机不同，判断标准也不同，不能把所有 RWA 项目放在同一个框里评价。

经济学上，这就是采用新技术的边际收益与边际成本比较。只有当代币化节省的发行、结算、分销或流动性成本超过新增的法律、技术、合规和声誉成本时，发行人才有持续采用的理由。

## 投资者为什么会购买

投资者购买代币化资产，可能是为了获得更小面额、更快交易、链上抵押用途、全球可达性或新的风险暴露。机构投资者还可能看重自动结算、透明登记和与链上现金管理工具的整合。

但投资者同样会比较替代品。若可以买传统国债 ETF、货币基金或债券基金，为什么要买链上版本？链上版本必须在结算、抵押、可组合、费用或准入上有真实优势。

对普通投资者，风险理解尤其重要。一个高收益 RWA token 可能只是高信用风险、低流动性或弱保护资产。收益率高不等于技术先进。

投资者还要分清“获得原本难以获得的资产”和“以更高风险获得同类资产”。如果链上国债基金提供的收益、费用和风险接近传统货币基金，但能更方便地用于链上支付和抵押，它可能有真实用途。如果某个房地产 token 的收益显著高于同类地产债务或 REITs，投资者要问高收益来自哪里：是运营效率，还是更高杠杆、更弱保护、更差流动性和更少披露？

资产定价的基本原则没有变。更高预期收益通常对应更高风险、流动性折价或信息不对称补偿。代币化不能让投资者无成本获得超额收益。

## 中介不会消失，而会改变

RWA 代币化常被说成减少中介。但现实中，发行人、律师、托管商、审计师、资产管理人、预言机、合规服务商、做市商和交易平台仍然存在。它们的角色可能变化，但不会全部消失。

传统中介处理法律、信息和风险问题。区块链可以减少部分登记和对账中介，但无法替代资产评估、信用分析、法律执行和投资者服务。真正的变化，是哪些环节可以自动化，哪些环节仍必须由可信机构承担。

这和 DeFi 金融中介地图一致：金融中介功能会被拆解和重组，而不是凭空消失。

例如，链上登记可能减少某些人工对账，但托管人仍要保管底层资产，审计师仍要验证资产和报告，做市商仍要提供二级市场流动性，合规服务商仍要处理 KYC/AML 和投资者资格。预言机甚至会成为新的关键中介，因为它把链下资产价格、NAV 或事件带入链上系统。

所以，判断一个 RWA 项目是否更高效，不应只看中介数量是否减少，而要看每个中介承担的功能是否更清楚、成本是否更低、责任是否更可追究。

## 公链、许可链和混合架构

代币化资产可以在公链、许可链或混合系统中运行。公链开放、可组合、透明，但合规和隐私管理更难。许可链可控、适合机构和监管要求，但开放性和外部流动性较弱。混合架构可能把发行和合规放在许可环境，把某些凭证或结算接口连接到公链。

选择哪种架构，取决于资产类型、投资者范围、监管要求和流动性目标。面向散户的合规证券、机构债券、私募基金和碳信用，可能需要不同技术路径。

技术架构不是价值本身。它服务于法律权利、市场流动性和风险控制。

因此，投资者不应因为一个 RWA 项目部署在公链上就自动认为它更透明，也不应因为它部署在许可链上就自动认为它更安全。真正要看的是：谁能查看什么信息，谁能转让，谁能冻结，谁能赎回，谁在争议中执行权利。架构选择只有和这些制度安排结合起来，才有金融含义。

## 从试点走向规模的条件

RWA 从试点走向规模，需要几个条件。第一，法律承认数字证券和链上登记。第二，可靠的托管、审计和披露。第三，合规的投资者准入和转让规则。第四，可信结算资产。第五，足够二级市场流动性。第六，传统机构愿意接入。

缺少其中任何一项，RWA 可能停留在小规模试验。原书也强调，监管差异、KYC/AML、持有人权利和投资者信心，是代币化扩张的关键障碍。

规模化还需要标准化。传统资本市场之所以能处理大量交易，是因为证券代码、清算规则、会计标准、托管流程和披露格式高度标准化。RWA 如果每个项目都使用不同法律结构、不同 token 标准、不同赎回规则和不同披露口径，投资者和机构很难批量接入。标准化并不意味着所有资产相同，而是关键字段和流程要可比较。

最后，RWA 还要经历周期检验。牛市中，投资者愿意接受新结构；压力时期，赎回、清算、法律执行和资产隔离才会真正暴露质量。只有穿越压力期仍能履约的结构，才可能被传统资本市场长期接受。

## 小结

代币化资产要真正改变资本市场，必须连接发行人、投资者、托管、清算、合规和二级流动性。发行人会比较成本，投资者会比较替代品，中介功能会重组而不是消失。公链、许可链和混合架构各有权衡。RWA 扩大规模需要法律承认、披露、托管、合规转让、可信结算资产和传统机构接入。技术只是工具，资本市场最终仍由权利、现金流、风险和信任支撑。
