# 64.3 分割性、可组合性和流动性改善

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.9；补充参考本笔记 Ch.5、Ch.21、Ch.24、Ch.35、Ch.58-Ch.61。

代币化最常被宣传的好处，是让原本昂贵、难交易的资产更容易买卖。房地产、私募基金、艺术品、碳信用和某些债券，往往交易单位大、流程复杂、二级市场薄。Token 可以降低面额、简化转让，并让资产进入链上组合。问题是：流动性改善到底来自哪里？

## 分割性降低参与门槛

许多现实资产天然金额很大。一栋楼、一个私募基金份额、一笔企业贷款或一组碳信用，普通小投资者很难直接参与。代币化可以把权益拆成较小份额，让更多人以较低金额持有。

这和共同基金的面额中介功能相似。第 24 章讲过，共同基金把许多投资者的钱集中起来，让小投资者间接持有大额证券组合。代币化则把某些资产份额变成更细小、更容易转让的数字单位。

分割性提高了可达性，但不自动提高投资质量。小额投资者更容易参与，也可能更容易买到自己不了解的复杂资产。

分割性还会改变投资者的心理感受。原本需要一百万美元才能参与的资产，若被拆成一百美元一份，看起来更亲近，也更容易被平台包装成普通投资品。但资产风险不会因为面额变小而变简单。高杠杆地产项目、低评级贷款、早期私募股权和复杂收益权，即使被拆成很小份额，仍然需要专业分析。

这与投资学中的分散化概念有关。小份额让投资者更容易分散持有多种资产，但分散化只有在资产相关性、估值和风险暴露被理解时才有效。若投资者买入许多看似不同、实则都依赖同一轮加密流动性或同一类高风险借款人的 token，分散化可能只是表面分散。

## 可组合性让资产进入 DeFi

一旦资产权利变成标准化 token，它理论上可以进入链上金融：作为抵押品、进入流动性池、被收益聚合器持有、用于 DAO 国库管理或参与自动结算。碳信用 token 可以作为抵押品，链上国债基金 token 可以用于现金管理，房地产收益权 token 可以在二级市场转让。

可组合性提高资本效率。传统市场中，某些资产很难快速抵押、转让或嵌入其他产品；链上 token 若被协议接受，就能更快参与金融活动。

但可组合性也放大风险。若 RWA token 的底层资产估值不透明，却被多个 DeFi 协议接受为抵押品，一旦估值下调或赎回受阻，风险会传导到借贷、稳定币和收益产品。

这里可以把 RWA token 想成一种新的抵押品输入。第 59 章讲过，DeFi 借贷最怕抵押品价格突然失真。原生加密资产的价格至少可以从交易所连续获取，虽然波动很大；很多 RWA 的估值更新慢，且依赖链下机构。如果协议用一个更新缓慢的 NAV 作为抵押品价格，清算可能滞后；如果用薄弱二级市场价格，又可能被操纵。可组合性越强，价格输入越重要。

## 流动性需要买方，不只是技术

流动性不是“可以转账”那么简单。一个 token 可以在链上 24 小时转账，但若没有买方、没有做市商、没有可信估值、没有信息披露，真正卖出时仍会大幅折价。

传统高收益债市场的发展说明，风险资产能否融资，不只取决于资产本身，还取决于二级市场、做市、信息和投资者基础。代币化资产同样如此。智能合约能降低转让摩擦，但不能保证市场深度。

因此，代币化带来的流动性改善通常是条件性的：资产标准化程度高、信息透明、法律权利清楚、投资者基础足够大、做市和赎回机制可靠时，流动性才可能真正改善。

还要区分交易流动性和融资流动性。交易流动性是投资者能否以接近合理价格买卖资产；融资流动性是投资者能否把资产作为抵押借到钱。一个 RWA token 可能二级交易很少，却因为被某个协议接受为抵押品而短期看起来有用。但如果抵押融资依赖高估值，一旦市场怀疑底层资产，借贷额度会迅速收缩，融资流动性也会消失。

这与金融危机章节中的经验一致：危机前很多资产看起来可融资，危机时折扣上升、抵押品被拒收，流动性突然枯竭。代币化不能改变这一金融约束。

## 流动性溢价会重新定价

如果代币化确实提高流动性，投资者要求的流动性溢价可能下降，发行人成本可能降低。例如，数字债券若结算更快、投资者基础更广、二级交易更便利，发行人可能用更低融资成本获得资金。

但如果流动性只是表面改善，投资者最终会要求更高折价。一个房地产 token 若交易很少、估值不透明、法律执行复杂，即使可以链上转让，也仍然是不流动资产。市场不会因为它叫 token 就取消流动性溢价。

投资学上，流动性是资产定价的一部分。代币化真正有价值时，应当体现在更低交易成本、更窄折价、更快融资和更广投资者基础中，而不是只体现在宣传中的“24/7 可交易”。

## 小结

代币化通过分割性降低投资门槛，通过可组合性让资产进入链上金融，通过更快登记和转让潜在改善流动性。但流动性需要真实买方、做市、信息披露和法律确定性，不是 token 标准自动创造的。若代币化真正降低交易和结算摩擦，流动性溢价可能下降；若底层资产仍不透明、买方不足，token 只是给不流动资产套上可转让外壳。
