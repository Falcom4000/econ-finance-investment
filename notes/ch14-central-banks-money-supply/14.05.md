# 14.5 货币乘数与货币供给过程

来源：

- 主线：Mishkin《货币金融学》Ch.14, Ch.15
- 补充：Mankiw Ch.30；Mishkin/Eakins Ch.9

简单存款乘数说明，准备金可以支持多倍存款。但现实货币供给不只由法定准备金率决定。中央银行、银行和存款人都会影响货币供给。中央银行影响货币基础和准备金率；银行决定是否向中央银行借款、持有多少超额准备金；存款人决定持有多少现金、多少存款。

货币乘数把这些因素放到一个框架中。它回答的问题是：给定货币基础变化，货币供给会变化多少？

## 三个参与者

货币供给过程有三个参与者。

第一，中央银行。它通过公开市场操作影响非借入货币基础，通过贴现贷款影响借入准备金，并通过准备金要求影响存款扩张能力。

第二，银行。银行决定是否向中央银行借入准备金，也决定持有多少超额准备金。银行越愿意贷款、越少持有超额准备金，存款创造越强；银行越谨慎，货币扩张越弱。

第三，存款人。公众决定现金和存款的比例。现金是货币供给的一部分，但不能支持多轮存款创造。存款越多地留在银行体系，越能支持贷款和存款扩张。

| 参与者 | 影响变量 | 对货币供给的作用 |
| --- | --- | --- |
| 中央银行 | 非借入货币基础、准备金率 | 增加基础货币通常增加货币供给；提高准备金率降低乘数 |
| 银行 | 借入准备金、超额准备金 | 借入准备金增加货币供给；超额准备金增加降低乘数 |
| 存款人 | 现金持有 | 更多现金通常削弱存款扩张 |

## 货币乘数公式的直觉

以 M1 为例，货币供给由现金和支票存款构成：

```text
M = C + D
```

货币基础由现金和准备金构成：

```text
MB = C + R
```

准备金又包括法定准备金和超额准备金：

```text
R = RR + ER
```

如果法定准备金率是 `rr`，现金与存款之比是 `c = C / D`，超额准备金与存款之比是 `e = ER / D`，货币乘数为：

```text
m = (1 + c) / (rr + e + c)
```

货币供给与货币基础的关系是：

```text
M = m × MB
```

这个公式看起来抽象，但含义清楚。分子 `1 + c` 表示每 1 单位存款对应的货币供给包括存款本身和公众持有的现金。分母 `rr + e + c` 表示支持这些存款和现金需要多少货币基础：法定准备金、超额准备金和现金都要占用基础货币。

## 为什么乘数不等于简单存款乘数

简单存款乘数只考虑法定准备金率，假设公众不持有现金，银行不持有超额准备金。因此，准备金全部用于支持存款扩张。

现实中，公众会持有现金。每 1 美元基础货币如果变成现金，就不会变成银行准备金，也不会支持下一轮贷款。银行也会持有超额准备金。每 1 美元准备金如果被银行闲置为超额准备金，就不会用于贷款和存款创造。

因此，现实货币乘数通常小于简单存款乘数。2008 年前，美国银行超额准备金比率很低，货币乘数通常大于 1。全球金融危机和新冠疫情期间，中央银行资产购买和贷款工具使准备金大幅上升，银行持有大量超额准备金，超额准备金比率变得很高，货币乘数甚至可以低于 1。

## 五个变量如何影响货币供给

货币供给与非借入货币基础正相关。中央银行公开市场购买增加非借入货币基础，准备金增加，存款创造扩大，货币供给上升。公开市场出售则相反。

货币供给与借入准备金正相关。银行向中央银行借款取得准备金，货币基础增加，存款创造能力上升。银行减少借款则货币基础下降。

货币供给与法定准备金率负相关。准备金率越高，每 1 美元存款需要的准备金越多，同样准备金支持的存款越少。

货币供给与超额准备金负相关。银行持有更多超额准备金，意味着更少准备金用于贷款和存款创造。

货币供给与现金持有通常负相关。公众把存款换成现金，银行准备金下降，存款创造能力减弱。不过在超额准备金极高的特殊情况下，现金比率变化对乘数的影响可能出现反直觉结果，因为现金增加也可能减少大量超额准备金占用。

## 小结

货币乘数说明货币基础如何转化为货币供给。现实货币供给由中央银行、银行和存款人共同决定。中央银行控制非借入货币基础并影响准备金要求；银行决定借入准备金和超额准备金；存款人决定现金和存款比例。货币乘数 `m = (1 + c) / (rr + e + c)` 表明，法定准备金率、超额准备金比率和现金比率都会影响货币扩张。货币供给不是简单由中央银行单方面决定，而是多个主体行为共同作用的结果。
