# 31.1 CAPM 与证券市场线

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.9
- 相关旧笔记：本笔记 Ch.9, Ch.29, Ch.30

资本资产定价模型（CAPM）是现代投资学的核心模型之一。组合理论从个体投资者角度解释如何在风险资产之间构建有效组合；CAPM 往前走一步，问的是：如果所有投资者都这样做，市场均衡下每一种风险资产应该提供怎样的预期收益？

CAPM 的答案很有力量：资产的风险溢价不取决于它的总波动率，而取决于它对市场组合的系统性风险暴露，也就是 beta。不能被分散掉的风险才有风险溢价；可以通过组合分散掉的公司特有风险，在均衡中不应获得额外补偿。

## CAPM 的基本假设

CAPM 建立在一组强假设之上。它并不声称现实世界完全满足这些条件，而是用这些条件把风险定价的基本机制剥离出来。

关于投资者行为，CAPM 假设：

1. 投资者是理性的均值-方差优化者；
2. 投资者有相同的单期投资期限；
3. 投资者使用相同的信息和相同的输入清单，即同质预期。

关于市场结构，CAPM 假设：

1. 所有风险资产都可以公开交易；
2. 投资者可以按同一无风险利率借入和贷出；
3. 不存在税收和交易成本；
4. 投资者可以自由做空。

这些假设显然很强。但它们的作用是让我们先看清一个干净的均衡逻辑：如果投资者面对同样的信息和同样的投资机会集，他们会选择同一个最优风险组合。

## 从有效边界到市场组合

给定无风险资产和风险资产有效边界，投资者会选择与资本配置线相切的最优风险组合。不同风险厌恶程度的投资者持有同一个风险组合，只是在这个风险组合和无风险资产之间配置比例不同。

CAPM 把这个逻辑推广到整个市场。因为所有投资者拥有同质预期，他们看到同一条有效边界和同一个切点组合。因此，所有人都想持有同一个风险资产组合。

均衡要求所有风险资产都被持有，所有资产市场都出清。于是，这个共同持有的最优风险组合必须等于市场组合。

市场组合是所有风险资产按市值加权构成的组合：

```text
某资产在市场组合中的权重 = 该资产市值 / 所有风险资产总市值
```

这一步非常关键。CAPM 不是随便选一个指数当作最优组合，而是在理论上推出：如果假设成立，所有投资者共同持有的切点组合就是市场组合。现实中我们用宽基股票指数、全球市场指数等作为市场组合的近似。

## 风险溢价为什么只奖励系统性风险

单个证券的总风险可以分成两部分：

```text
总风险 = 系统性风险 + 公司特有风险
```

公司特有风险来自个别企业事件，如产品失败、管理层更替、诉讼、会计问题等。系统性风险来自全市场共同因素，如商业周期、利率、通胀、风险偏好和宏观政策。

在充分分散的组合里，公司特有风险会被大量证券之间的相互抵消显著降低。理性投资者不需要为承担单个公司的特有风险获得补偿，因为他们可以近乎免费地把它分散掉。相反，系统性风险无法通过持有更多证券消除，因为它影响整个市场。

这和微观经济学里的机会成本思维一致：市场只会补偿稀缺且无法轻易替代的风险承受能力。公司特有风险可以通过分散化处理，不是定价中的稀缺资源；系统性风险无法被整个市场消除，因此需要风险溢价。

## beta：系统性风险的度量

CAPM 中，系统性风险用 beta 度量：

```text
beta_i = Cov(R_i, R_M) / Var(R_M)
```

其中 `R_i` 是资产 `i` 的超额收益，`R_M` 是市场组合的超额收益。

beta 衡量资产收益对市场组合收益的敏感度：

- `beta = 1`：资产与市场同等敏感；
- `beta > 1`：资产比市场更激进，市场上涨或下跌时反应更大；
- `0 < beta < 1`：资产比市场更防御；
- `beta < 0`：资产在市场下跌时倾向于上涨，具有对冲价值。

注意 beta 不是总风险。一个小公司股票可能标准差很高，但如果其波动主要是公司特有风险，beta 未必很高。CAPM 关心的不是“这个资产波动大不大”，而是“这个资产在市场整体坏状态中是否也一起坏”。

这点和风险厌恶的经济含义有关。投资者最不愿意承担的是在整体财富下降、消费机会变差、就业和收入也可能受压时同时亏钱的资产。与市场同跌的资产需要更高风险溢价；能在市场坏状态中提供保护的资产可以接受较低预期收益。

## 证券市场线

CAPM 的核心公式是证券市场线（SML）：

```text
E(r_i) = r_f + beta_i [E(r_M) - r_f]
```

也可以写成超额收益形式：

```text
E(R_i) = beta_i E(R_M)
```

其中：

- `r_f` 是无风险利率；
- `E(r_M) - r_f` 是市场风险溢价；
- `beta_i` 是资产对市场风险的暴露。

证券市场线的横轴是 beta，纵轴是期望收益。它表示：均衡中，资产的预期收益应当随 beta 线性增加。

例如，若无风险利率为 3%，市场预期收益为 9%，某股票 beta 为 1.5，则 CAPM 要求收益为：

```text
3% + 1.5 × (9% - 3%) = 12%
```

如果分析师估计该股票预期收益为 14%，它高于 SML，具有正 alpha，可能被低估；如果预期收益只有 10%，它低于 SML，可能被高估。

## alpha：偏离证券市场线的收益

CAPM 给出一个风险调整后的基准收益。资产实际预期收益与 CAPM 要求收益之间的差，就是 alpha：

```text
alpha_i = E(r_i) - {r_f + beta_i [E(r_M) - r_f]}
```

正 alpha 表示资产提供的预期收益超过其系统性风险所要求的补偿；负 alpha 表示补偿不足。指数模型把 alpha 用于主动组合构建，CAPM 则解释了 alpha 的基准来自哪里。

不过，现实中的 alpha 是估计值。它依赖预期收益、beta、无风险利率和市场风险溢价的估计。任何一个输入出错，都可能造成虚假的 alpha。

## 资本市场线与证券市场线的区别

资本市场线（CML）和证券市场线（SML）容易混淆。

CML 描述的是有效组合的期望收益与总风险标准差之间的关系。它只适用于由无风险资产和市场组合构成的有效组合。

SML 描述的是任何资产或组合的期望收益与 beta 之间的关系。它适用于单个证券、低效组合和有效组合。

简化地说：

- CML 用标准差衡量风险，只评价有效组合；
- SML 用 beta 衡量风险，可以评价所有资产；
- CML 是组合选择工具；
- SML 是资产定价工具。

## 与企业资本预算的关系

CAPM 不只用于证券投资，也用于公司金融中的资本成本估计。企业评估一个项目时，应使用与项目系统性风险相匹配的折现率，而不是简单使用公司的平均融资成本。

例如，一个公用事业公司如果投资高风险新能源项目，这个项目的 beta 可能高于公司原有业务；如果仍用低 beta 公用事业业务的资本成本折现，就会高估项目价值。反过来，一个高科技公司投资稳定现金流基础设施项目，若用公司整体高资本成本折现，可能会错过正 NPV 项目。

折现率不是会计成本，而是承担同等系统性风险的资本机会成本。

这个资本机会成本会随宏观金融环境变化。无风险利率上升会直接提高 CAPM 折现率；市场风险溢价扩大，通常意味着投资者风险承受能力下降或宏观不确定性上升；项目 beta 提高，则说明项目现金流更依赖商业周期和金融条件。CAPM 因此把货币政策、风险偏好和企业投资决策连接在同一条折现率链条上。

## 小结

CAPM 从 Markowitz 组合理论出发，在同质预期和理想市场假设下推出市场组合是均衡中的最优风险组合。资产的风险溢价只补偿系统性风险，系统性风险由 beta 衡量。证券市场线给出 `E(r_i) = r_f + beta_i [E(r_M) - r_f]`，成为证券估值、主动管理和企业资本预算的重要基准。CAPM 的现实假设很强，但它确立了现代资产定价的核心原则：可分散风险不应获得风险溢价，不能分散的共同风险才应被定价。
