# 31.4 Fama-French 模型与风险因子

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.10
- 相关旧笔记：本笔记 Ch.9

Fama-French 模型是多因子资产定价中最有影响力的一类模型。它的起点是一个经验事实：单一市场 beta 不能充分解释不同股票组合的平均收益差异。小市值股票、账面市值比高的价值股，在很长历史样本中表现出高于 CAPM 解释的收益。Fama 和 French 将这些特征转化为因子，构建了著名的三因子模型，后来又扩展为五因子模型。

这一节要理解两个层面：第一，Fama-French 模型怎样构造；第二，这些因子到底是风险补偿，还是市场无效或行为偏差的结果。

## 从 CAPM 到三因子模型

CAPM 只有一个风险因子：市场组合。三因子模型加入两个额外因子：

```text
R_i = alpha_i + beta_iM MKT + beta_iS SMB + beta_iH HML + e_i
```

其中：

- `MKT` 是市场超额收益；
- `SMB` 是规模因子，代表小市值股票相对大市值股票的收益；
- `HML` 是价值因子，代表高账面市值比股票相对低账面市值比股票的收益；
- `beta_iS` 和 `beta_iH` 分别衡量证券对规模和价值因子的暴露。

`SMB` 通常理解为 small minus big，即小盘组合收益减大盘组合收益。`HML` 通常理解为 high minus low，即高账面市值比组合收益减低账面市值比组合收益。

三因子模型的定价关系可以理解为：

```text
E(R_i) = beta_iM lambda_M + beta_iS lambda_S + beta_iH lambda_H
```

如果一个资产组合对小盘因子和价值因子有正暴露，它的预期收益可能高于 CAPM 只用市场 beta 所预测的水平。

## 规模因子：小公司为什么可能有更高收益

小市值股票长期表现出较高平均收益，这被称为规模效应。可能解释包括：

- 小公司经营更脆弱，对融资环境更敏感；
- 信息透明度较低，投资者要求更高补偿；
- 流动性较差，交易成本更高；
- 小公司更容易受经济衰退和信用收缩影响；
- 市场关注度低，可能存在错误定价。

如果规模效应是风险补偿，那么小盘股高收益不是免费午餐，而是补偿投资者承担小公司在坏状态中的更高风险。如果规模效应是市场无效，那么它可能来自投资者忽视、机构约束或行为偏差。

实践中，两种解释可能同时存在。一部分高收益来自风险，一部分来自市场结构和投资者行为。

## 价值因子：便宜股票为什么可能有更高收益

价值因子基于账面市值比。高账面市值比股票通常价格相对于账面资产较低，被称为价值股；低账面市值比股票通常价格反映高增长预期，被称为成长股。

价值股可能有更高平均收益，原因包括：

- 它们往往处于困境、成熟或低增长行业；
- 盈利前景不确定，投资者要求更高风险补偿；
- 市场可能过度悲观，低估反转可能性；
- 投资者偏好热门成长叙事，导致成长股过贵。

价值因子的争议正体现了资产定价和行为金融之间的边界问题。若价值溢价补偿的是困境风险，它是理性风险溢价；若价值溢价来自投资者过度外推成长股好消息、过度厌恶坏公司，它是行为偏差。

## 五因子模型：盈利能力与投资

Fama-French 后来提出五因子模型，在市场、规模、价值之外加入：

- 盈利能力因子：盈利能力强的公司相对盈利能力弱的公司；
- 投资因子：投资保守的公司相对投资激进的公司。

简化写法为：

```text
R_i = alpha_i + beta_M MKT + beta_S SMB + beta_H HML
      + beta_R RMW + beta_C CMA + e_i
```

其中：

- `RMW` 表示 robust minus weak，即高盈利减低盈利；
- `CMA` 表示 conservative minus aggressive，即保守投资减激进投资。

这些因子与公司估值的基本公式有关。股票价值来自未来现金流折现。盈利能力越强，在同等价格下预期现金流越高；投资越激进，若边际投资回报不高，可能损害未来收益。因子模型把这些基本面特征转化为系统性收益模式。

## 因子暴露不是 alpha

Fama-French 模型对主动管理有一个重要影响：很多看起来像 alpha 的收益，可能只是因子暴露。

例如，一个基金长期买入小盘价值股，跑赢市场指数。如果只用 CAPM 评价，它可能有正 alpha；但如果加入 SMB 和 HML 后，alpha 可能消失。这说明基金收益可能来自承担小盘和价值因子风险，而不一定来自独特选股能力。

真正的证券选择 alpha 应当是在控制常见因子之后仍然存在的收益。否则，投资者可以用更低成本的因子基金或指数产品获得类似暴露。

## 风险解释与行为解释

Fama-French 因子的根本争议在于：它们到底是风险，还是异象？

风险解释认为，高平均收益来自投资者不愿承担的风险。小盘、价值、低盈利或高投资公司可能在经济坏状态下表现更差，因此需要风险溢价。

行为解释认为，高平均收益来自系统性错误定价。例如投资者过度外推成长、过度追逐热门股票、忽视无聊公司，导致价值股被低估、成长股被高估。

这两种解释在实证上很难完全区分，因为风险和行为会同时影响价格。一个资产在坏状态中表现差，可能是风险；投资者对坏消息反应过度，也可能造成低估。投资学中的许多重要争论都在这个边界上。

## 因子投资的实践风险

因子被发现以后，不等于投资者可以轻松赚钱。因子投资至少有几类风险。

第一，因子溢价会长期低迷。价值因子可能多年跑输成长因子，小盘可能长期落后大盘。投资者必须承受追踪误差和心理压力。

第二，因子会拥挤。当大量资金追逐同一因子，估值被推高，未来收益可能下降。

第三，因子定义不同，结果不同。账面市值比、盈利能力、动量、质量等指标有多种构造方式，组合细节会影响表现。

第四，交易成本会侵蚀收益。小盘、低流动性和高换手因子尤其如此。

第五，因子不是宏观免疫工具。多个因子在危机中可能同时下跌，相关性在压力时期上升。

因子投资也需要宏观解释。价值股、小盘股、高盈利公司和保守投资公司在不同利率、通胀和信用周期下表现可能差异很大。低利率和高成长预期环境可能压制价值因子，信用收缩可能伤害小盘因子，市场恐慌可能让低流动性因子同时承压。因子不是脱离经济周期的永久机器，而是风险、行为和资金拥挤共同作用的结果。

## 小结

Fama-French 模型把 CAPM 扩展为多因子框架，用市场、规模、价值以及后来的盈利能力和投资因子解释股票收益差异。它的重要意义不只是提高解释力，更是改变了 alpha 的定义：跑赢市场不一定代表选股能力，可能只是承担了特定因子暴露。Fama-French 因子的解释存在风险补偿和行为偏差两种路径，实践中的因子投资还必须面对拥挤、交易成本、长期失效和模型定义风险。
