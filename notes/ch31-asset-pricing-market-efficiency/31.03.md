# 31.3 APT 与多因子风险模型

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.10
- 相关旧笔记：本笔记 Ch.9

套利定价理论（APT）和多因子模型延续了 CAPM 的核心思想：只有系统性风险应该获得风险溢价。但它们不再坚持“市场组合这一个因子足以解释所有系统性风险”。现实中，资产收益可能同时暴露于多个宏观或金融因子：利率、通胀、经济增长、信用风险、汇率、能源价格、风格因子等。

APT 的出发点不是所有投资者都持有市场组合，而是一个更一般的无套利逻辑：如果两个充分分散的组合对所有系统性因子的暴露相同，它们就应该有相同的预期收益；否则投资者可以构造套利或近似套利。

## 多因子收益结构

在多因子模型中，证券收益可以写成：

```text
R_i = E(R_i) + beta_i1 F_1 + beta_i2 F_2 + ... + beta_iK F_K + e_i
```

其中：

- `F_k` 是第 `k` 个系统性风险因子；
- `beta_ik` 是证券 `i` 对该因子的敏感度；
- `e_i` 是公司特有风险；
- `K` 是因子数量。

例如，一个银行股可能同时暴露于：

- 市场整体风险；
- 利率期限结构变化；
- 信用利差变化；
- 房地产周期；
- 监管政策冲击。

单一 beta 很难把这些风险都概括进去。多因子模型允许资产因不同风险来源而获得不同风险溢价。

## APT 的无套利逻辑

APT 的关键不是均值-方差优化，而是无套利。

设想两个充分分散的组合 A 和 B，它们对所有系统性因子的 beta 完全相同。如果 A 的预期收益高于 B，投资者可以做多 A、做空 B。由于两者系统性风险暴露抵消，公司特有风险又因充分分散而接近消除，这个头寸几乎没有风险，却有正预期收益。

竞争性市场中，这种机会不应长期存在。因此，充分分散组合的预期收益必须由其因子暴露决定。

APT 的多因子定价式可以写成：

```text
E(R_i) = beta_i1 lambda_1 + beta_i2 lambda_2 + ... + beta_iK lambda_K
```

其中 `lambda_k` 是第 `k` 个因子的风险溢价。它表示每承担一单位该因子风险，市场要求多少预期超额收益。

## APT 与 CAPM 的关系

CAPM 可以看作一种单因子模型：

```text
E(R_i) = beta_iM E(R_M)
```

APT 则允许多个因子：

```text
E(R_i) = beta_i1 lambda_1 + ... + beta_iK lambda_K
```

两者的共同点是：

- 都区分系统性风险和公司特有风险；
- 都认为可分散风险不应获得风险溢价；
- 都用 beta 衡量风险暴露；
- 都可以产生风险调整后的基准收益和 alpha。

两者的差异是：

- CAPM 从投资者优化和市场组合出发；
- APT 从无套利和因子暴露出发；
- CAPM 理论上给出唯一市场因子；
- APT 允许多个因子，但不自动告诉我们哪些因子最重要。

APT 更灵活，但也更开放。灵活性带来实用价值，也带来模型选择风险。

## 因子可以是什么

多因子模型中的因子大致有三类。

第一类是宏观经济因子，如通胀、工业生产、利率、期限利差、信用利差、汇率、油价等。这些因子直接对应经济状态。

第二类是可交易组合因子，如市场组合、价值组合、规模组合、动量组合、质量组合等。它们通过构造投资组合来追踪某类风险或特征。

第三类是统计因子，如主成分分析从大量收益数据中提取的共同波动因子。它们可能有强解释力，但经济含义不一定直观。

投资管理中更常使用可交易组合因子，因为它们可以被复制、对冲和归因。例如，如果一个基金收益来自价值因子暴露，投资者可以判断这到底是经理能力，还是一种可用低成本策略获得的风险暴露。

## 因子风险溢价的经济含义

不是所有共同波动都应有风险溢价。一个因子要被定价，通常需要满足两个条件：

1. 它代表投资者不愿承担的系统性风险；
2. 它不能被轻易分散或免费对冲。

例如，信用利差扩大通常发生在经济压力上升、违约风险增加、融资环境恶化的时候。暴露于信用利差风险的资产会在坏状态中下跌，因此投资者要求补偿。

又如，通胀上升可能侵蚀固定现金流资产的实际购买力。如果某资产在通胀冲击下表现很差，它也可能需要通胀风险溢价。

这与宏观经济学中的总需求、货币政策和价格水平相连。资产价格不是孤立数字，而是不同经济状态下现金流和边际效用的折现结果。

## 因子模型在组合管理中的用途

多因子模型在实践中有几类用途。

第一，风险分解。投资者可以知道组合收益来自市场、行业、利率、价值、规模、动量等哪些因子，而不是只看总波动。

第二，业绩归因。基金经理跑赢基准，可能来自真正选股 alpha，也可能来自对某些因子的系统性暴露。因子模型帮助区分两者。

第三，风险控制。机构投资者可以限制某些因子暴露，避免组合在不知不觉中集中押注单一宏观或风格风险。

第四，组合构建。投资者可以有意识地配置不同因子，形成更均衡的风险来源。

第五，成本比较。如果某个主动基金的收益主要来自廉价因子暴露，投资者就不应为它支付高额主动管理费。

## APT 的局限

APT 的局限也很明显。

第一，它不直接告诉我们正确因子集合。不同研究者和机构可能选择不同因子，导致模型结论不同。

第二，因子溢价会随时间变化。某个历史上有效的因子，未来可能因拥挤交易、市场结构变化或宏观环境改变而减弱。

第三，因子之间可能相关。价值、质量、低波动、盈利能力等因子并非总是独立，解释时要避免重复归因。

第四，数据挖掘风险很高。研究者可以在大量变量中找到历史显著的因子，但这些因子可能只是样本偶然结果。

所以，多因子模型比 CAPM 更接近现实，但并不意味着它自动更可靠。模型的经济解释、样本外表现、交易可实现性和风险归因稳定性都必须检验。

多因子模型还把资产管理和金融产品连接起来。因子 ETF、智能 beta 产品和风险平价策略，本质上都是把某些因子暴露包装成可交易组合。投资者购买这类产品时，买到的不一定是管理人 alpha，而可能是价值、动量、低波动、期限、信用或流动性等系统性风险暴露。理解因子模型，才能判断费用是否合理、风险是否重复、组合是否真正分散。

## 小结

APT 和多因子模型把 CAPM 的单一市场 beta 扩展为多个系统性风险暴露。APT 基于无套利逻辑：充分分散且因子暴露相同的组合应有相同预期收益。多因子模型可用于风险分解、业绩归因、组合构建和风险控制，但它也面临因子选择、时间变化、相关性和数据挖掘问题。它的核心贡献，是把“风险溢价来自不可分散的系统性风险”这一思想从单一市场风险推广到多个经济风险来源。
