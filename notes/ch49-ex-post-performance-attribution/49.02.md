# 49.2 带误差的绩效归因与归因悖论

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 14, "Performance Attribution"。

基本绩效归因假设因子收益和特质收益都已知。但在真实风险模型中，因子收益本身也是估计出来的。这个细节会引出一些看似矛盾的现象，也会改变我们解释 factor PnL 和 idiosyncratic PnL 的方式。

以因子模拟组合为例。若用回归或加权最小二乘估计因子收益，模型会构造出某些组合，使它们对目标因子有单位暴露、对其他因子中性。按照估计方法，这些因子模拟组合的估计特质收益可能为零，因为残差被设定为与因子组合正交。于是会出现一个奇怪现象：一个因子模拟组合明明有很高特质风险，但事后归因却显示它没有特质 PnL。

另一个悖论来自因子中性组合。假设组合 `w` 满足：

```text
B' w = 0
```

也就是模型认为它因子暴露为零。现在加入某个因子模拟组合 `v` 的倍数，得到：

```text
w + lambda v
```

随着 `lambda` 改变，组合的总风险会变化，特质风险也可能变化；但在某些机械归因口径下，估计的 idiosyncratic PnL 可能保持不变。这显然不合理：如果风险真的改变了，归因的不确定性也应该改变。

这些悖论的根源是把“估计值”误当成“真实值”。真实因子收益 `f` 不可观察，我们看到的是估计因子收益：

```text
f_hat = f + eta
```

其中 `eta` 是估计误差。在加权最小二乘的高斯设定下，估计误差方差通常与下面的矩阵有关：

```text
(B' Omega_epsilon^{-1} B)^{-1}
```

这个矩阵说明，因子收益估计的精度取决于因子载荷矩阵和特质风险结构。若某个因子只有少数股票能识别，或者这些股票特质风险很大，那么该因子收益估计就不可靠。

对一个组合来说，用估计因子收益计算的 factor PnL 为：

```text
b' f_hat
```

但真实 factor PnL 是：

```text
b' f
```

二者差异为：

```text
b' eta
```

因此，估计 factor PnL 等于真实 factor PnL 加上一项估计误差；估计 idiosyncratic PnL 则会相应减去这项误差。也就是说，factor PnL 和 idiosyncratic PnL 不是两个确定数字，而是两个带标准误的估计量。

跨多个时期时，真实 factor PnL 可以被看作围绕估计 factor PnL 的随机变量：

```text
true factor PnL
~ N(sum_t b_t' f_hat_t,
    sum_t b_t' (B_t' Omega_epsilon,t^{-1} B_t)^{-1} b_t)
```

真实 idiosyncratic PnL 也有类似的不确定性，并且与 factor PnL 的估计误差方向相反。这解释了为什么 factor PnL 和 idiosyncratic PnL 的时间序列有时会显著负相关：同一段收益被模型多解释给因子一些，就会少解释给残差一些。

正确的归因报告因此不应只给出点估计。它至少应给出标准误、置信区间或不确定性说明。比如某组合的行业因子 PnL 为 100 万美元，但标准误为 120 万美元，那么不能自信地说收益来自行业暴露；若另一个风格因子 PnL 为 80 万美元，标准误只有 10 万美元，则这个归因更可信。归因的统计显著性和经济显著性都重要。

这个问题也提醒我们，风险模型不是显微镜，不能无限精确地把收益切成干净小块。因子定义、载荷估计、协方差估计、回归权重和样本窗口都会影响归因结果。成熟的归因实践不是追求一个貌似精确的数字，而是承认估计误差，并围绕不确定性做解释。

## 小结

因子收益和特质收益在真实模型中都是估计量，因此绩效归因带有误差。若把估计值当成真实值，会出现因子模拟组合有特质风险却无特质 PnL、因子中性组合风险变化却归因不变等悖论。解决办法是把 factor PnL 和 idiosyncratic PnL 视为带标准误的估计量，并在归因报告中呈现不确定性。
