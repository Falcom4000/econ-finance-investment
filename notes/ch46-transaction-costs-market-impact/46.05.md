# 46.5 确定性 alpha 与 AR(1) 信号下的交易

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 11, Market-Impact-Aware Portfolio Management.

交易成本感知优化最终要回答一个信号问题：给定 alpha 的时间结构，今天该交易多少？如果 alpha 是一次性脉冲，最优策略应提前建仓、在信号附近持有、之后逐步清算。如果 alpha 会持续衰减，交易就应更谨慎，因为晚到的收益越来越弱。如果 alpha 高度持久，则可以更慢地建立仓位，因为机会不会立刻消失。

确定性 alpha 是最容易理解的情形。假设未来某个时间 `t0` 会出现一次收益机会。若没有交易成本，最优做法可能是在 `t0` 前一刻直接建仓；有交易成本时，临近时突然交易太贵，所以模型会提前逐步建仓。`t0` 之后，如果没有后续 alpha，组合再按清算规则逐渐回到零。交易路径因此围绕 alpha 的时间分布展开，而不是只看当前信号。

AR(1) 信号更接近日常量化策略。设预期收益满足

```text
mu_{t+1} = Phi mu_t + eta_t
```

其中 `Phi` 的对角元素在 0 和 1 之间，表示信号持久性。若 `Phi` 接近 0，信号很快消失，今天不交易就可能错过机会；若 `Phi` 接近 1，信号很持久，交易可以更平滑。最优交易规则仍可写成

```text
delta x_t = (-Gamma x_t + K mu_t) delta t
```

其中 `-Gamma x_t` 是清算项，`K mu_t` 是投资项。

这个式子把交易直觉讲得很清楚。当前持仓若太大，清算项会拉低仓位；当前 alpha 若为正，投资项会推高仓位。最终交易是二者合力。成本越高，`K` 的有效作用越弱，交易越慢；风险越高，清算项越强，组合越不愿持有大仓位；信号越持久，未来 alpha 的折现价值越高，模型越愿意建立仓位但不必猛追。

AR(1) 模型还帮助区分“信号强度”和“信号速度”。两个信号当前 alpha 一样大，但一个明天就消失，一个能持续数周。无成本 MVO 会给出相同目标仓位；交易成本模型会区别对待。短命信号若成本高，可能根本不值得交易；持久信号即使单期强度不高，也可能值得慢慢建仓。

这正是交易成本和 alpha 研究的连接点。一个信号不能只报告 IC 或 Sharpe，还要报告换手、半衰期、容量和成本后的收益。第 43 章说 alpha 回测必须贴近生产；第 46 章说明，生产中的 alpha 价值取决于它能否在成本允许的速度下被捕捉。

## 小结

确定性 alpha 和 AR(1) 信号说明，最优交易取决于 alpha 的时间结构。一次性脉冲 alpha 会诱导提前建仓和事后清算；持久 AR(1) alpha 允许更平滑地交易。交易规则由清算项和投资项组成，分别反映风险持有成本和未来预期收益。信号强度、持久性、交易成本和风险共同决定一个 alpha 是否值得交易。
