# 37.6 Kalman Filter 在波动率估计中的直觉

来源：

- 主线：Paleologo《The Elements of Quantitative Investing》Ch.2
- 相关旧笔记：本笔记 Ch.29, Ch.35

Kalman Filter 常被包装成一套线性代数和概率递推公式，但它的核心问题很朴素：真实状态看不见，观测数据有噪声，我们怎样随着新数据到来不断修正估计？

金融中到处都是这样的问题。真实价格不一定等于最后成交价，真实波动率不能直接观察，真实因子收益需要从大量资产收益中估计，真实 alpha 也混在噪声里。Kalman Filter 提供的是一种过滤思想：不要完全相信最新观测，也不要死守旧估计，而是根据两边的不确定性分配权重。

## 从一个普通例子开始

想象我们要估计一辆车的位置。车的位置是真实状态，但 GPS 测量有误差。如果 GPS 信号很好，每次新读数都很可靠，我们应该快速更新位置估计；如果 GPS 信号很差，读数跳来跳去，我们就不应完全相信最新读数，而要更多依赖上一期估计和运动规律。

波动率估计也一样。真实波动率是看不见的状态，收益平方或实现方差是带噪声的读数。某天收益平方很大，可能说明风险真的升高，也可能只是一次偶然冲击。某天收益平方很小，也不能说明风险已经消失。

Kalman Filter 的精神就是：先用旧状态预测今天，再用今天观测修正预测。修正幅度取决于新观测相对旧预测有多可靠。

## 状态方程和观测方程

状态空间模型通常有两部分。

第一部分是状态方程，描述真实状态如何变化：

```text
今天真实状态 = 昨天真实状态的延续 + 状态变化噪声
```

在波动率问题中，真实状态可以是当前方差或对数方差。状态变化噪声表示市场风险环境会变化，例如新闻、流动性、风险偏好和杠杆调整都会改变波动率。

第二部分是观测方程，描述我们看到的数据如何由真实状态产生：

```text
今天观测值 = 今天真实状态 + 观测噪声
```

观测值可以是收益平方、实现方差，或其他风险代理变量。观测噪声表示单个观测并不完美。收益平方尤其嘈杂，因为一次大收益可能只是偶然方向冲击，不一定代表长期方差永久上升。

## 更新权重来自两种噪声的比较

Kalman Filter 中最重要的直觉是比较两种不确定性。

第一种是状态预测的不确定性。如果市场风险状态变化很快，昨天的估计今天可能已经过时，旧预测不可靠。

第二种是观测的不确定性。如果收益平方非常嘈杂，今天的新观测可能包含大量偶然噪声，新数据不可靠。

当状态变化噪声大、观测噪声小时，模型应给新观测更高权重；当状态变化噪声小、观测噪声大时，模型应给旧估计更高权重。这就是过滤器权重的经济含义。

EWMA 可以看作这种思想的简单版本：

```text
新估计 = 一部分旧估计 + 一部分新观测
```

但 Kalman Filter 进一步说明，权重不只是经验平滑参数，而可以由状态变化噪声和观测噪声推导出来。

## 为什么波动率不能简单随机游走

最简单的状态方程可能假设方差像随机游走一样变化：今天等于昨天加一个随机扰动。但方差有两个特殊问题。

第一，方差不能为负。普通线性高斯状态模型可能在数学上允许负值，这在经济上不合理。实际模型常对波动率或方差取对数，使状态可以在实数上变化，而映射回来的方差始终为正。

第二，波动率通常有均值回复。危机时波动率上升，但不会永远维持极端高位；长期平静时波动率下降，但也不会永远接近零。更合理的状态方程会让波动率围绕长期水平波动：偏离越大，未来回归压力越强。

这和 GARCH 的直觉一致。GARCH 中方差受过去冲击影响，也受长期水平约束；状态空间模型用另一种语言表达类似思想。

## Harvey-Shephard 模型的启发

更精细的状态空间波动率模型会直接建模对数波动率。这样做有几个好处。对数方差可以是任意实数，但方差本身通过指数变换保持为正；波动率分布可以自然右偏，因为风险水平偶尔会非常高；收益在给定波动率状态时可以更接近标准化随机冲击。

这类模型的技术细节不必在第一次学习时完全掌握，但它带来一个重要观念：波动率不是固定参数，而是一个隐藏的、随时间变化的随机过程。我们通过收益率数据不断推断它。

在量化投资中，这种观念比具体公式更重要。它会反复出现在后续主题中：因子收益不可直接观察，需要从资产收益中估计；风险暴露可能随时间变化，需要更新；alpha 也不是直接看见的，需要从噪声中识别。

## Kalman Filter 与投资流程

Kalman Filter 的意义不仅在于估计波动率。它代表一种处理金融数据的基本态度：

- 真实对象通常不可直接观察；
- 观测数据有噪声；
- 状态会随时间变化；
- 新数据应该更新估计，但更新幅度要受噪声控制；
- 模型要在灵敏和稳健之间权衡。

这和第 36 章的量化投资流程完全一致。交易前的风险模型、信号模型和成本模型都在估计不可直接观察的对象；交易后绩效归因也在从 noisy PnL 中识别真实能力。过滤思想贯穿整个系统。

## 小结

Kalman Filter 解决的是隐藏状态估计问题。在波动率估计中，真实方差或对数方差是不可直接观察的状态，收益平方或实现方差是带噪声的观测。过滤器根据状态变化噪声和观测噪声的相对大小，决定新观测和旧估计各占多大权重。

简单 EWMA 可以看作过滤思想的特例。更丰富的状态空间模型可以加入均值回复、正方差约束和对数波动率。学习 Kalman Filter 的重点不是背公式，而是理解金融建模中的核心问题：我们总是在用不完美数据推断不断变化的隐藏状态。
