# 37.4 GARCH 与条件异方差

来源：

- 主线：Paleologo《The Elements of Quantitative Investing》Ch.2
- 相关旧笔记：本笔记 Ch.29, Ch.35

上一节的经验事实告诉我们：收益方向很难用简单历史收益预测，但波动率有明显持续性。高波动之后，下一期仍然高波动的概率更高；低波动之后，市场也常常继续平静。这就引出一个自然想法：也许收益率的均值可以近似为零或很小，但收益率的方差不应被看成常数。

**条件异方差**模型就是这样产生的。异方差指方差会变化；条件异方差指在给定过去信息后，下一期收益的条件方差会随历史状态变化。GARCH 是这类模型中最经典、最常用的形式之一。

## 从“同一个噪声分布”到“随时间变化的风险状态”

最简单的收益模型可以写成：

```text
收益 = 平均收益 + 固定波动率 * 随机冲击
```

这个模型假设每一期波动率相同。它的好处是简单，坏处是明显不符合市场。危机后的第二天和长期平静期的一天，风险不可能相同；财报公告日和普通交易日，单只股票波动也不可能相同。

GARCH 的核心不是直接预测涨跌方向，而是预测随机冲击的尺度。可以把收益写成：

```text
收益_t = 当期波动率_t * 标准化冲击_t
```

标准化冲击可以近似看作均值为零、方差为一的随机变量；当期波动率决定这次冲击会被放大多少。于是，模型的重点变成：当期波动率如何由过去信息决定？

## GARCH(1,1) 的基本机制

最常见的 GARCH(1,1) 可以用一句话概括：

```text
今天的方差 = 长期基础水平 + 昨天收益冲击的影响 + 昨天方差的延续
```

其中，“昨天收益冲击”通常用昨天收益率平方表示。昨天绝对涨跌越大，说明市场刚经历较强冲击，今天方差估计应上调。“昨天方差的延续”表示风险状态有惯性，高波动不会立刻消失。

这个结构同时包含两个力量：

- 冲击效应：大收益平方会把波动率推高；
- 均值回复：波动率不会无限上升，会逐渐回到长期水平。

如果只有冲击效应，模型会对每一次大波动过度反应；如果只有长期平均，模型又无法及时反映危机。GARCH 把两者结合起来。

## 为什么 GARCH 能解释波动率聚集

假设某天市场大跌，收益率平方很大。GARCH 会提高下一期方差估计。方差提高后，下一期收益率出现大绝对值的概率也提高。若下一期又出现大波动，方差继续维持高位。这就形成波动率聚集。

反过来，在长时间小波动后，收益率平方较小，方差估计逐渐下降。低方差状态下，大幅波动概率较低，市场看起来继续平静。直到新的大冲击出现，模型再上调风险估计。

这种机制非常适合解释风险管理中的一个经验：危机时期不能用长期平均波动率估计短期风险，平静时期也不能用危机波动率永久外推。

## 为什么 GARCH 会产生厚尾

GARCH 还可以解释收益率厚尾。给定某一期波动率时，标准化冲击也许接近正态；但长期来看，收益率来自不同波动率状态的混合：低波动时期收益集中在零附近，高波动时期收益分散得多。把这些时期混合在一起，整体分布就会比单一正态分布更尖峰、更厚尾。

可以用一个直观类比：如果每天都从同一个正态分布抽样，尾部较薄；如果有些日子从低方差正态分布抽样，有些日子从高方差正态分布抽样，整体样本中就会同时出现很多接近零的小收益和更多极端大收益。这正是金融收益率常见形态。

因此，GARCH 的一个重要检验是看标准化残差。若用估计波动率把收益率除掉，得到的残差应比原始收益更接近稳定分布。若残差仍有强烈波动聚集或厚尾，说明模型没有充分解释风险状态。

## GARCH 和 EWMA 的关系

GARCH 和 EWMA 都使用过去收益平方估计当前方差。EWMA 用指数衰减权重平均过去收益平方，近期权重更大、远期权重更小。GARCH 则在类似指数加权思想上加入长期方差水平，并通过参数估计决定冲击和延续的强度。

粗略地说，EWMA 是一种简单、透明、工程上稳定的风险更新规则；GARCH 是更完整的统计模型，能描述长期方差、冲击效应和持续性。实践中，商业风险模型常偏好 EWMA 或其变体，因为它容易解释、参数少、部署稳定；研究中 GARCH 更常用于理解波动率动态和做统计分析。

这不是谁绝对优越的问题。若策略需要可解释、稳健、跨大量资产统一应用，EWMA 可能更合适；若研究目标是刻画某个资产的波动过程，GARCH 更有表达力。

## GARCH 的局限

GARCH(1,1) 虽然重要，但不能包打天下。

第一，它主要处理单资产方差，不直接解决多资产协方差。组合风险需要知道资产之间如何共同波动，仅有每只资产的 GARCH 不够。

第二，标准 GARCH 对正负收益冲击通常对称处理。但许多股票市场中，下跌可能比上涨更显著提高未来波动率，这被称为杠杆效应或不对称波动。需要 EGARCH、GJR-GARCH 等扩展模型才能处理。

第三，GARCH 对极端跳跃、制度变化和流动性崩溃的处理有限。一次央行意外决策、战争、违约或交易暂停，可能不是平滑波动率过程能充分解释的。

第四，估计误差不可忽视。对每只股票单独估计 GARCH 参数，可能不稳定；对大量资产统一使用参数，又可能忽略个体差异。实际系统必须在复杂度和稳健性之间取舍。

## 在投资流程中的位置

GARCH 的主要作用是帮助我们理解和预测风险，而不是直接产生 alpha。它告诉我们，当市场刚经历大波动时，应提高短期风险估计；当市场长期平静时，可以逐渐降低风险估计，但不能忘记尾部风险。

在组合管理中，波动率估计会影响：

- 仓位大小：波动率上升时，同样名义仓位风险更高；
- 风险预算：不同资产风险状态变化会改变配置；
- 杠杆控制：高波动时期应降低杠杆；
- 回测评估：策略收益要按当时风险状态解释；
- 交易成本：高波动通常伴随更高价差和冲击成本。

因此，GARCH 不只是一个计量经济学模型，而是连接收益率经验事实和实际风险管理的桥梁。

## 小结

GARCH 属于条件异方差模型，核心思想是收益率的条件方差随时间变化，并由过去收益冲击和过去方差状态更新。GARCH(1,1) 同时包含冲击效应和风险状态延续，因此能解释波动率聚集；由于不同波动率状态混合，它也能解释无条件收益率厚尾。

GARCH 的价值在于说明风险可以动态估计：收益方向很难预测，但波动率状态有持续性。它也有局限，包括单资产为主、对不对称波动和跳跃处理有限、参数估计可能不稳定。实际量化投资会把 GARCH 直觉与 EWMA、实现波动率和多资产因子风险模型结合使用。
