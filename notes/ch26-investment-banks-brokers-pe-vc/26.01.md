# 26.1 投资银行的功能

## 投资银行为什么不是普通银行

投资银行这个名字容易误导。普通商业银行的核心业务是吸收存款、发放贷款，并通过资产负债表承担信用风险。投资银行的核心业务不是吸收公众存款，而是帮助企业在证券市场发行股票和债券，帮助投资者交易证券，并为公司并购、资产出售和融资结构提供服务。

如果一家企业需要资金，它可以向银行借款，也可以发行债券或股票。银行贷款属于间接融资，企业向银行借，银行再用存款或其他资金提供贷款。发行证券属于直接融资，企业直接向投资者筹资。投资银行的主要位置，就在企业和投资者之间。

```mermaid
flowchart LR
    A[企业需要长期资金] --> B[发行股票或债券]
    B --> C[投资银行设计、定价、销售]
    C --> D[投资者购买证券]
    D --> E[资金流向企业]
```

投资银行存在的根本原因，是证券发行很复杂。企业不经常发行股票和债券，不一定知道什么时候发、发什么、定价多少、怎样满足监管披露要求、怎样找到投资者。投资银行每天与证券市场打交道，积累了信息、销售网络和定价经验。

## 一级市场中的核心中介

投资银行最典型的功能，是帮助企业在一级市场发行新证券。一级市场是新证券首次出售给投资者的市场。企业通过一级市场获得资金；投资者通过购买新发行证券，把资金提供给企业。

这与二级市场不同。二级市场中，投资者之间买卖已经发行的证券，交易本身通常不把新资金交给企业。例如，你在证券账户里买入某公司股票，钱通常流向卖出股票的另一个投资者，而不是流向公司。只有公司发行新股或新债时，资金才进入公司。

投资银行让一级市场更容易运转。它们帮助企业决定融资方式，准备文件，接受监管审查，向投资者分发招股说明书，组织销售，并在承销中承担一定价格风险。

从宏观角度看，一级市场是储蓄转化为投资的重要通道。家庭、基金、保险公司和养老金的储蓄，通过购买企业证券，变成企业扩张、设备投资、研发和收购的资金。投资银行提高这个通道的效率。

## 信息生产和信誉背书

证券发行面临严重信息不对称。企业内部管理层最了解公司真实状况，外部投资者信息较少。如果投资者担心企业隐瞒坏消息，就会要求更低价格，甚至拒绝购买。好企业不愿以低价发行证券，坏企业留下来，市场可能出现柠檬问题。

投资银行的作用之一，是生产信息并提供信誉背书。它们研究企业财务状况、行业前景、竞争环境和风险，帮助确定合理价格。承销时，投资银行还可能先买下整批证券，再卖给公众。它用自己的资本承担风险，向市场传递一个信号：它相信这次发行可以被投资者接受。

这种信誉很重要。投资银行如果反复把质量差、定价不合理的证券卖给客户，会失去投资者信任，以后很难销售新项目。因此，声誉约束迫使投行在发行人和投资者之间保持一定平衡。

## 给企业的建议

企业发行证券前，首先要决定融资工具。发行股票会增加股本，摊薄原股东持股，但不需要固定还本付息；发行债券不会稀释股权，但增加固定利息负担，提高财务风险。

投资银行会结合市场条件给出建议。如果企业所在行业股票价格较高，市盈率处于高位，发行股票可能更有利；如果债券利率较低，发行债券的融资成本可能更低。企业也需要判断融资时机。例如，竞争对手刚公布糟糕业绩，市场对整个行业悲观，企业可能暂缓发行股票。

最困难的是定价。发行人希望价格越高越好，因为高价格意味着同样股份能筹到更多钱。投资银行却不希望价格过高，因为如果承销证券卖不出去，它要承担损失。定价过低，发行人会觉得自己损失了资金；定价过高，投资银行卖不掉证券并损害声誉。

这个定价问题在首次公开发行中尤其困难。已经上市公司的再次发行，可以参考现有市场价格；首次公开发行没有历史交易价格，需要依靠估值模型、可比公司、市场需求和投资者反馈。

## 文件、披露和监管

证券公开发行必须向监管者提交注册文件。注册文件包括公司财务状况、管理层、竞争环境、行业背景、融资用途和证券风险。监管者审查的重点，是必要披露是否完整，而不是替投资者保证这是一项好投资。

招股说明书是注册文件中面向投资者传播的部分。投资者在购买新发行证券前，应获得招股说明书，以便了解公司和发行风险。

对债券发行，投资银行还要协助获得信用评级、聘请债券律师确认发行合法性、选择受托人监督发行人履行债券契约。对股票发行，投资银行可能帮助公司安排上市交易所。

披露制度的经济含义，是降低信息不对称。它不能消除投资风险，也不能保证信息完全真实，但能让发行人承担披露责任，让投资者获得基本判断材料。

## 投资银行的业务边界

投资银行不只做证券发行。它们还帮助公司出售业务部门、进行并购、安排私募发行，并通过关联的经纪和交易部门参与二级市场。

历史上，美国商业银行和投资银行曾被法律分开。大萧条后，监管者担心商业银行用存款支持证券承销和投机，导致银行体系风险上升，于是建立了商业银行与投资银行分业框架。后来金融创新和监管放松使边界逐渐模糊，许多商业银行收购投行业务，证券公司也提供类似银行的现金管理、贷款和支付服务。

这条边界变化说明，金融机构会围绕客户需求不断扩张业务。企业需要融资、交易、风险管理和并购顾问；家庭和机构需要投资、账户、贷款和支付便利。不同金融机构会争夺这些业务，监管则需要在竞争、效率和风险控制之间做权衡。

## 小结

投资银行的核心功能，是帮助企业在一级市场发行股票和债券，把投资者储蓄转化为企业长期资金。它们提供融资建议、证券定价、文件准备、监管披露、销售网络和承销服务。

投资银行存在的经济理由，是证券发行中的信息不对称、定价困难和交易成本。它们通过信息生产和信誉背书，降低投资者对新发行证券的疑虑，也帮助企业更有效筹资。

投资银行与商业银行不同，但历史上二者边界不断变化。理解投资银行，首先要抓住它在一级市场中的位置，再理解它如何延伸到并购、经纪、交易和私募股权等业务。
