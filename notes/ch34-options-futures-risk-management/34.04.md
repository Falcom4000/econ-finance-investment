# 34.4 二叉树期权定价

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.21
- 相关旧笔记：本笔记 Ch.27

二叉树模型是理解期权定价最清楚的方法。它假设在一个短期间内，股票价格只有两种可能：上涨或下跌。虽然现实价格不只两个状态，但二叉树展示了期权定价的核心逻辑：通过股票和无风险借贷复制期权现金流。

## 单期二叉树

假设股票现价为 `S`，下一期可能上涨到 `uS`，也可能下跌到 `dS`。看涨期权到期价值为：

```text
C_u = max(uS - X, 0)
C_d = max(dS - X, 0)
```

目标是构造一个组合：持有 `Delta` 股股票，并借入或贷出无风险资产，使它在上涨和下跌状态下的价值都等于期权价值。

如果复制组合与期权到期现金流完全相同，则今天成本必须等于期权价格。

## Delta 对冲

复制组合中的股票数量称为 delta：

```text
Delta = (C_u - C_d) / (uS - dS)
```

Delta 衡量期权价值对股票价格变化的敏感度。看涨期权 delta 通常为正，看跌期权 delta 通常为负。

期权卖方可以通过持有 delta 股股票对冲风险。若卖出看涨期权，就买入一定数量股票，使组合在上下状态中无风险。

## 风险中性定价

二叉树也可以用风险中性概率表达。设风险中性上涨概率为 `q`：

```text
q = (1 + r - d) / (u - d)
```

期权价值为风险中性期望 payoff 的折现：

```text
C = [q C_u + (1-q) C_d] / (1+r)
```

风险中性概率不是真实概率。它是使资产预期收益等于无风险利率的定价工具。期权价格不需要直接使用投资者风险厌恶，因为复制组合已经把风险价格包含在股票价格中。

## 多期二叉树

多期二叉树把单期模型重复多次。到期时计算各节点期权价值，再向前倒推到今天。

美式期权可以在每个节点比较：

```text
继续持有价值 vs 立即执行价值
```

取较大者。这使二叉树特别适合处理美式期权、股利、可赎回债券和其他带提前执行特征的证券。

## 模型直觉

二叉树模型说明，期权价值来自动态复制。期权不是因为投资者主观看涨才有价值，而是因为它的非线性 payoff 可以被股票和无风险资产复制。

这与无套利思想一致。若期权价格高于复制成本，卖出期权并买入复制组合可套利；若低于复制成本，买入期权并卖出复制组合可套利。

## 小结

二叉树模型通过股票和无风险资产复制期权现金流，推出无套利期权价格。Delta 是复制组合中的股票头寸，也是期权价格对标的价格的敏感度。风险中性概率是定价工具，不是真实概率。多期二叉树可以处理美式期权和提前执行问题，是理解复杂衍生品定价的基础。
