# 34.3 看跌-看涨平价与期权边界

来源：

- 主线：Bodie/Kane/Marcus《Investments》Ch.20
- 相关旧笔记：本笔记 Ch.27

期权定价的第一层不是复杂模型，而是无套利边界。只要两个组合在未来所有状态下现金流相同，它们今天价格就应相同。看跌-看涨平价就是这种复制和无套利逻辑的经典应用。

## 看跌-看涨平价

对不支付股利股票的欧式期权，看跌-看涨平价为：

```text
C + PV(X) = P + S
```

其中：

- `C` 是看涨期权价格；
- `P` 是看跌期权价格；
- `S` 是股票现价；
- `PV(X)` 是执行价格现值。

左边是持有看涨期权和无风险债券，右边是持有看跌期权和股票。到期时，两边价值都等于 `max(S_T, X)`，因此今天价格必须相等。

可以分两种状态验证。若到期股票价格高于执行价，看涨期权会被执行，左边的看涨期权给出 `S_T - X`，无风险债券给出 `X`，合计 `S_T`；右边的股票价值是 `S_T`，看跌期权到期为 0，合计也是 `S_T`。若到期股票价格低于执行价，看涨期权为 0，无风险债券给出 `X`；右边股票价值是 `S_T`，看跌期权价值是 `X - S_T`，合计仍是 `X`。两边在所有状态中都相同，所以今天价格必须相同。

这个推理比记公式更重要。期权定价经常看起来复杂，但第一原则非常朴素：如果两个组合未来无论发生什么都给出同样现金流，它们今天就不能有不同价格。否则市场就允许投资者买便宜组合、卖昂贵组合，不承担风险地赚钱。

## 平价的含义

看跌-看涨平价说明，看涨、看跌、股票和无风险债券之间存在严格关系。知道其中三个价格，就能推出第四个无套利价格。

它也说明期权不是凭空创造收益。保护性看跌和信托看涨在到期收益上可以复制；不同期权组合之间的价格必须由复制关系约束。

如果平价被违反，套利者可以做多便宜组合、做空昂贵组合，锁定无风险收益。交易会推动价格回到平价附近。

这个过程体现了金融市场的价格发现。看似不同的证券，只要最终现金流相同，就必须服从同一价格约束。股票、债券、看涨和看跌期权之间的关系不是经验规律，而是由可复制现金流决定的逻辑关系。后面的二叉树和 Black-Scholes 模型，都是把这种复制思想推广到更多状态和连续时间中。

看跌-看涨平价还帮助读者理解许多期权策略其实是同一风险结构的不同包装。保护性看跌是股票加看跌期权，根据信托看涨关系，它与持有债券加看涨期权在到期 payoff 上非常接近。换句话说，投资者以为自己在买“股票保险”，也可以被理解为买一个保底资产再加上上涨参与权。不同包装的经济实质，需要用无套利关系拆开看。

## 股利调整

如果股票在期权到期前支付确定股利，平价需要调整：

```text
C + PV(X) + PV(股利) = P + S
```

股利会降低除息后股价，因此对看涨期权不利、对看跌期权有利。

## 期权价格边界

无套利还给出期权价格上下界。

看涨期权价格不能超过股票价格：

```text
C ≤ S
```

看跌期权价格不能超过执行价格现值或执行价格附近的合理上限。

欧式看涨期权下界为：

```text
C ≥ max(S - PV(X), 0)
```

欧式看跌期权下界为：

```text
P ≥ max(PV(X) - S, 0)
```

若价格低于下界，投资者可通过复制策略套利。

## 提前执行与边界

美式期权因可提前执行，价值不低于欧式期权。美式看跌可能提前执行，因为深度价内时提前收到执行价现金有利。美式看涨在无股利股票上通常不应提前执行，因为提前执行会放弃时间价值。

这些结论来自同一个原则：比较立即执行与继续持有期权的价值。

无套利关系还依赖融资、做空和交易执行。理论上，套利者可以同时买入便宜组合、卖出昂贵组合；现实中，借券成本、保证金、买卖价差、税收和模型误差会让价格偏离在一定范围内存在。看跌-看涨平价因此既是定价基准，也是检查市场摩擦和交易限制的工具。

## 小结

看跌-看涨平价用无套利复制关系连接看涨、看跌、股票和无风险债券。它说明期权价格不能任意变化，必须满足相同到期现金流对应相同价格的原则。期权价格边界进一步限制期权价值范围。复杂期权定价模型可以不同，但都必须尊重这些无套利关系。
