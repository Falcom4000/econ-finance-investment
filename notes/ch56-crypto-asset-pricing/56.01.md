# 56.1 加密资产的分类：货币型、平台型、治理型和证券型

来源：Marco Di Maggio, *Blockchain, Crypto and DeFi: Bridging Finance and Technology*, Chapters 1-4 and 9；补充参照本笔记第 29 章至第 33 章关于收益、风险、资产定价和证券估值的讨论。

加密资产不能混成一类。BTC、ETH、稳定币、治理代币、NFT、RWA token 和证券型 token 的金融属性差异很大。如果把它们都叫“币”，就无法做严肃定价。

投资学的第一步是识别资产权利。股票代表剩余索取权，债券代表固定或约定现金流，商品代表实物或稀缺资源，货币市场工具代表短期信用。加密资产也要先问：它给持有人什么权利？收益来自哪里？风险由谁承担？

这个问题看似基础，却决定后面所有估值方法。一个没有现金流的资产不能硬套市盈率；一个只是治理投票权的 token 不能直接当作股权；一个代表国债的 RWA token 也不能因为上链就脱离利率和信用风险。分类不是贴标签，而是确定现金流、权利、负债和风险的归属。

## 货币型资产

BTC 是最典型的货币型或数字稀缺资产。它没有发行人负债，不承诺现金流，也不代表公司股权。其价值主要来自固定供给、网络共识、价值储藏叙事、支付和转移功能，以及市场愿意为非主权稀缺资产支付价格。

货币型资产难以用贴现现金流估值，因为没有现金流。它更接近黄金、外汇或收藏性稀缺资产。定价依赖需求、稀缺性、流动性、信任和宏观环境。

这类资产的核心风险是需求坍塌、监管限制、技术替代和叙事失效，而不是违约风险。

所以，货币型资产的“基本面”不是利润表，而是接受度和信任网络。持有人是否愿意长期持有，交易所和托管基础设施是否成熟，监管是否允许机构参与，宏观环境是否支持非主权稀缺资产需求，都会影响价格。它不像债券那样违约，而是可能失去市场愿意赋予它的货币溢价。

## 平台型资产

ETH 更接近平台型资产。Ethereum 是智能合约平台，ETH 用于支付 Gas、质押验证、DeFi 抵押和生态结算。网络使用增加，可能提高 ETH 的费用需求和燃烧数量。

平台型资产不像股票那样拥有明确股东权利，但也不像 BTC 那样完全没有使用需求。它的价值部分来自网络活动，部分来自质押收益，部分来自货币溢价。

分析平台型资产，要看交易量、费用、活跃开发者、应用生态、质押率、发行和销毁机制，以及竞争链压力。

## 治理型资产

许多 DeFi 协议发行治理代币。治理代币通常赋予投票权，可以决定协议参数、费用分配、抵押品类型、升级和资金库使用。

治理权不等于现金流权。如果治理代币不能分享协议收入，也没有强制价值捕获机制，那么其价值更多来自治理影响力、未来分红预期或投机需求。若协议收入最终可以分配给代币持有人，治理代币会更接近股权。

治理型资产的关键问题是委托代理。谁真正投票？大户是否控制治理？团队和投资人解锁如何影响价格？治理是否能把协议收入转化为持有人价值？

## 证券型和 RWA 资产

证券型 token 或 RWA token 代表某种链下资产或现金流，例如国债、基金份额、私募信贷、房地产权益或碳信用。它们最接近传统证券。

这类资产的定价应首先回到传统金融。代币化国债的价值来自底层国债收益率和信用；代币化房地产来自租金、空置率、折现率和法律产权；链上基金份额来自底层组合净值。

区块链可能提高流动性、可分割性和结算效率，但不能改变底层资产现金流本质。一个差资产上链后不会自动变成好资产。

## 稳定币和包装资产

稳定币通常不应按高风险投资资产理解。法币抵押稳定币更像短期发行人负债或货币市场工具；加密抵押稳定币像超额抵押债务系统；包装资产像桥发行的凭证。

它们的目标不是价格上涨，而是维持锚定。投资者关心的是储备质量、赎回机制、流动性和脱锚风险。若稳定币提供高收益，还要问收益来自储备利息、借贷需求、做市风险还是补贴。

这也提醒我们，加密资产分类不是按价格波动大小分类，而是按经济功能分类。稳定币价格波动小，但一旦出事可能是挤兑和信用事件；治理代币价格波动大，但可能没有明确经济权利；RWA token 看起来在链上交易，底层却是传统现金流。只有先把经济功能分清，后面的风险模型和估值模型才不会错位。

## 小结

加密资产定价必须先分类。货币型资产如 BTC 主要依赖稀缺性和需求共识；平台型资产如 ETH 与网络使用、Gas、质押和费用燃烧相关；治理型资产取决于治理权和价值捕获；证券型和 RWA token 应回到底层现金流估值；稳定币和包装资产更像短期负债或凭证，重点是赎回和储备风险。不同类型不能用同一估值语言。
