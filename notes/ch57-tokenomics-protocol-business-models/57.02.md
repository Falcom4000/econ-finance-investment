# 57.2 发行、解锁、通胀和销毁机制

来源：主线参考 Di Maggio《Blockchain, Crypto and DeFi》Ch.2、Ch.6、Ch.9-Ch.10；补充参考本笔记 Ch.15-Ch.17、Ch.29-Ch.33、Ch.52-Ch.56。

理解 token 的权利之后，下一步要看供给。传统金融中，股票投资者关心股份稀释，债券投资者关心新增债务优先级，宏观经济学关心货币发行和通胀。token 也一样：即使一个协议有真实需求，如果供给持续增加、早期投资者大量解锁、激励排放超过真实收入，持有人也可能被稀释。

## 初始发行：谁先拿到 token

Token 的初始发行通常决定早期利益分配。常见分配对象包括团队、投资人、基金会或国库、社区激励、流动性挖矿、生态基金、空投用户和公开销售参与者。每一类分配对象的行为都不同。团队和基金会可能更关注长期建设，早期投资人可能有退出压力，做市和流动性激励参与者可能更短期，真实用户则可能既使用协议也持有 token。

经济学中的产权分配会影响激励。若团队没有足够权益，长期开发动力可能不足；若团队和早期投资人占比过高，社区会担心未来抛压和治理集中；若社区激励过多但缺少锁定，短期挖卖可能压低价格并消耗国库。一个看似“去中心化”的发行方案，仍可能在实际控制权上高度集中。

初始发行还影响市场流通量。市值常用价格乘以总供应量或流通供应量。若流通供应量很小，总估值很高，少量买盘就能推高价格，但未来大量解锁可能造成稀释。投资者不能只看当前流通市值，还要看完全稀释估值，即所有 token 释放后的总价值。

## 解锁：未来供给进入市场的时间表

解锁机制是 token 投资中特别重要的风险来源。团队、投资人和生态基金的 token 通常不会一次性进入市场，而是按月、按季度或按里程碑释放。这个设计可以降低早期抛售，也能约束团队长期投入。但从二级市场角度看，解锁就是未来潜在卖压。

解锁并不必然导致价格下跌。若协议增长很快，新增需求可以吸收新增供给；若解锁对象继续质押、参与治理或长期持有，实际卖压也可能较小。问题在于，解锁改变了供需平衡和预期。市场会提前定价未来抛压，做市商也会调整库存和报价。

这和股票市场的限售股解禁类似。公司上市后，早期股东持股在锁定期结束时可以出售。解禁本身不是基本面恶化，但会影响流通盘、交易压力和投资者心理。token 市场的差异在于，它全天候交易、信息披露不统一、做市深度更薄，因此解锁事件对价格的冲击可能更剧烈。

## 通胀：激励安全、流动性和增长的成本

许多协议通过新增发行 token 来支付验证者奖励、质押奖励、流动性挖矿奖励、生态补助或用户补贴。表面上，这些奖励给参与者带来收益；实质上，它们通常由全体 token 持有人承担稀释成本。

以 PoS 网络为例，新发行 token 奖励验证者，可以鼓励他们质押资产、维护网络安全。若所有持有人都按比例质押，通胀主要是在质押者之间重新分配；若一部分持有人不质押，他们会被质押者稀释。这个机制不是免费收益，而是一种激励设计：协议用新增供给购买安全服务。

流动性挖矿也是类似逻辑。协议向 LP 或用户发放 token，换取交易深度、借贷资金、用户增长和网络效应。短期内，这能让新协议迅速启动市场；长期看，如果补贴停止后用户离开，通胀只是把国库资源转移给短期资本。判断通胀是否合理，要看它买到了什么：安全、真实用户、长期流动性，还是只有临时 TVL。

## 销毁：减少供应不等于创造价值

销毁机制是把 token 永久移出流通。Ethereum 的 EIP-1559 把基础费销毁，使网络使用和 ETH 供给产生联系；某些协议用收入回购并销毁 token；算法稳定币或合成资产系统也可能通过铸造和销毁调节供给。

销毁容易被误解成“只要少了就会涨”。在供求分析中，供给减少确实可能提高价格，但前提是需求不变或上升，并且销毁不是以损害网络增长为代价。如果协议为了销毁而提高费用，用户可能迁移到替代平台；如果销毁来源只是新发行再销毁，净供给未必下降；如果销毁不对应真实收入，只是财务工程，价值创造也有限。

更严谨的看法是：销毁是一种价值分配方式，类似公司回购股票。高质量回购来自真实自由现金流，并在价格合理时减少股份；低质量回购可能只是用借债或一次性收入制造每股指标改善。token 销毁也应问：销毁资金来自真实用户支付，还是来自新投资者资金或通胀排放？

## 净供给：发行、解锁、销毁要合在一起看

单看发行、单看销毁或单看解锁都不够。真正影响持有人的是净供给路径：

```text
净供给变化 = 新发行 + 解锁进入流通 - 销毁 - 长期锁定或质押沉淀
```

其中，“长期锁定或质押沉淀”不是永久减少供应，因为这些 token 将来仍可能退出。但在短期流动性分析中，它会影响可交易筹码。若大量 token 被质押、锁仓或用于 DeFi 抵押，市场浮动供应会下降；一旦收益率下降或风险上升，这些沉淀供应又可能重新流入市场。

投资学中，价格由边际买卖决定。一个项目的完全稀释估值可能很高，但若流通盘小、锁仓多、交易深度薄，短期价格会对边际资金非常敏感。反过来，若解锁密集、奖励排放高、长期持有人开始退出，即使协议使用量仍在增长，价格也可能承压。

## 供给机制和宏观货币的类比边界

很多 token 社区喜欢把供给机制类比成中央银行货币政策。这个类比有启发，但不能过度。中央银行发行的是法定货币，背后有税收、法律偿付、银行体系和宏观稳定目标；token 发行通常是协议规则和社区治理的结果，缺少最后贷款人和财政支持。

不过，宏观经济学仍提供了重要语言。通胀不是单纯“发行变多”，而是供给增长、需求变化、流通速度和预期共同作用的结果。token 价格也不是只由供应量决定，还取决于网络使用、风险偏好、流动性、监管和替代资产收益率。供给机制告诉我们稀释压力，不能单独给出价值。

## 小结

Token 供给机制包括初始发行、解锁、通胀奖励和销毁。初始分配决定早期利益和治理结构，解锁决定未来潜在卖压，通胀奖励是协议购买安全、流动性和增长的成本，销毁则是价值分配方式而非自动创造价值。分析 token 不能只看最大供应量或销毁叙事，而要看净供给路径、需求增长和持有人稀释。
