# 42.2 最大似然、SVD 与统计因子估计

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 7, Statistical Factor Models.

PCA 最初看起来像一个纯线性代数工具：找最大方差方向，或者找最佳低秩近似。风险模型还需要一个概率解释，因为组合风险最终来自协方差矩阵。概率 PCA 把谱方法与标准因子模型连接起来。

考虑线性因子模型

```text
r = B f + epsilon
```

假设因子 `f` 服从均值为零、协方差为单位矩阵的正态分布，残差 `epsilon` 服从均值为零、协方差为 `sigma^2 I` 的正态分布，并且二者独立。于是资产收益的协方差矩阵为

```text
Sigma_r = B B' + sigma^2 I
```

这个结构说：共同风险由低秩矩阵 `BB'` 表示，剩余风险是同方差、相互独立的白噪声。这个假设显然比真实市场简单，但它提供了理解 PCA 的桥梁。

若用多元正态分布的似然函数估计 `B` 和 `sigma^2`，最大似然解与样本协方差矩阵的特征分解有关。设样本协方差矩阵分解为 `U S U'`，则载荷矩阵可以写成前 `m` 个特征向量乘以收缩后的特征值：

```text
B_hat = U_m (S_m - sigma_hat^2 I)^{1/2}
```

而 `sigma_hat^2` 可以用剩余 `n - m` 个特征值的平均值估计。换句话说，概率 PCA 不是直接把前 `m` 个样本特征值全部当作因子方差，而是把公共噪声水平扣掉。若 `sigma^2` 趋近于零，模型就退化为普通 PCA：所有被保留的方差都归因于因子，残差风险消失。

这条连接很有价值，因为它解释了为什么直接使用样本主成分会高估因子方差。样本协方差的前几个特征值不仅包含真实共同风险，也包含噪声被排序后推到前面的影响。概率模型通过估计噪声底座，把特征值分成“高于噪声的部分”和“噪声本身”。这与前一章基本面模型中因子收益估计误差会抬高样本方差的逻辑一致：观测到的统计量并不等于真实风险本身。

SVD 还揭示了横截面回归和时间序列回归之间的对称关系。设 `R = U S V'`，取前 `m` 个左奇异向量作为载荷 `B = U_m`。若在每个日期做横截面回归，用 `B` 解释资产收益，那么估计出来的因子收益为

```text
F_hat = U_m' R = S_m V_m'
```

这正是 SVD 中对应的因子收益部分。反过来，如果只知道这些因子收益，再把每只资产的收益时间序列回归到这些因子收益上，又能恢复出 `U_m`。因此，在 PCA/SVD 框架中，载荷和因子收益是相互一致的：载荷是资产收益对统计因子收益的时间序列 beta，因子收益又是横截面对载荷的回归系数。

这个结果把统计模型和基本面模型连接起来。基本面模型先给定 `B`，再横截面回归求 `F`；统计模型通过 SVD 同时给出 `B` 和 `F`。如果某些资产没有进入统计模型估计宇宙，但有完整收益历史，我们可以先用估计宇宙得到统计因子收益，再对这些资产做时间序列回归来补全载荷。这正是生产系统中处理非估计宇宙资产的一种方法。

不过，概率 PCA 的同方差残差假设很强。真实资产的特质波动率差异巨大：大盘高流动性股票与小盘低流动性股票的残差噪声不可能相同。如果直接对原始收益做 PCA，高波动股票会支配主成分，模型可能更多反映波动大小而不是相关结构。后面的两阶段 PCA 和特质波动率重加权，正是为了解决这个问题。

最大似然框架也提醒我们，统计因子估计不能只看解释方差。一个模型多加几个因子总能提高样本拟合，但未必提高未来风险预测。统计模型必须同时处理因子方差估计、残差方差估计、因子数量选择和样本外表现。这与投资学中的过拟合问题完全一致：样本内解释力越强，不代表组合风险预测越可靠。

## 小结

概率 PCA 把 PCA 与线性高斯因子模型连接起来：收益协方差被分解为低秩因子风险和同方差残差风险。最大似然解使用样本协方差的特征向量，但会从特征值中扣除噪声水平。SVD 说明统计载荷和统计因子收益可通过横截面回归与时间序列回归相互恢复。这个框架清楚展示了 PCA 的力量，也暴露了同方差残差和样本内过拟合的限制。
