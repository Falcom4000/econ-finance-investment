# 42.1 PCA 与最佳低秩近似

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 7, Statistical Factor Models.

基本面因子模型假设我们事先知道因子暴露是什么：行业、国家、价值、动量、规模等变量进入暴露矩阵 `B`，然后用横截面回归估计因子收益。统计因子模型从更困难也更少先验的情形出发：我们既不知道因子收益，也不知道资产对因子的暴露。唯一可靠输入可能只有一张收益矩阵。问题变成：如果资产之间确实存在少数共同风险来源，能不能只从收益数据中把它们估出来？

设 `R` 是 `n x T` 的收益矩阵，`n` 是资产数，`T` 是时间期数。统计因子模型寻找两个矩阵 `B` 和 `F`，使得

```text
R ≈ B F
```

其中 `B` 是 `n x m` 的载荷矩阵，`F` 是 `m x T` 的因子收益矩阵，`m` 远小于 `n` 和 `T`。如果用平方误差衡量未解释部分，问题可以写成

```text
min ||R - B F||_F^2
```

这里的 Frobenius 范数就是把所有资产、所有日期的残差平方加总。由于 `BF` 的秩最多为 `m`，这个问题等价于寻找一个秩不超过 `m` 的矩阵 `R_hat`，让它尽量接近原始收益矩阵 `R`。这就是最佳低秩近似问题。

奇异值分解给出了这个问题的标准解。若

```text
R = U S V'
```

其中 `U` 和 `V` 是正交矩阵，`S` 的对角线上是从大到小排列的奇异值，那么最优的 `m` 阶近似就是保留前 `m` 个奇异值，丢弃其余奇异值。可以写成

```text
R_hat = U_m S_m V_m'
```

这一步的直觉非常重要：收益矩阵中最强的共同变化方向被前几个奇异向量捕捉，弱小而分散的变化被放入残差。统计因子模型并不先给因子命名，而是先问数据中哪些方向解释了最多共同波动。

PCA 与这个低秩近似紧密相连。若从收益协方差矩阵出发，第一主成分可以被定义为一个单位长度的组合权重 `w`，使组合收益 `w' r` 的方差最大：

```text
max w' Sigma w
s.t. ||w|| <= 1
```

这个问题的解是协方差矩阵最大特征值对应的特征向量。第二主成分是在与第一主成分正交的方向中寻找方差最大的组合，依此类推。因此，主成分也可以被理解为一组相互正交的最大方差组合。用投资语言说，PCA 先找“最像市场整体波动的组合”，再找与它正交的下一个最大共同波动方向。

这种正交性是数学上的，不一定是经济上的。基本面模型中的行业因子和价值因子可以相关，因为现实中的经济风险本来会相关；PCA 因子则被构造成互相正交。这使它在协方差压缩上很方便，却让解释变难。第一主成分通常容易解释为市场因子，因为几乎所有股票会随市场共同波动。第二、第三主成分有时能对应利率曲线中的斜率和曲率，或股票市场中的行业/风格分化；更高阶主成分通常需要额外工具解释。

PCA 还有一个旋转不唯一问题。若 `B F` 是一个有效分解，那么对任意可逆矩阵 `C`，`B C` 和 `C^{-1} F` 给出的乘积完全相同。因子模型本身只识别由因子张成的子空间，而不一定唯一识别每个因子的名字和方向。当特征值相等或非常接近时，这种不唯一性更加明显：单个特征向量可以大幅旋转，但整个子空间仍然稳定。

从金融角度看，最佳低秩近似的意义在于降维。完整协方差矩阵包含大量参数，很难在短样本中稳定估计；若资产共同风险主要由少数方向驱动，则用少数统计因子近似能显著降低估计误差。它与前面金融学中“系统性风险不能被分散、特质风险可以被分散”的思想相连：PCA 尝试从收益数据中直接找出那些无法通过持有更多资产轻易消失的共同方向。

但 PCA 不是免费的午餐。它只看收益矩阵本身，不知道公司行业、会计特征、宏观状态和投资者结构。它可能表现很好，也可能把样本噪声误认为因子；它可能揭示基本面模型遗漏的共同风险，也可能产生难以解释、难以交易的方向。统计因子模型应被看作基本面模型的补充、诊断工具和在缺少特征数据时的替代方案，而不是无条件更高级的模型。

## 小结

统计因子模型在因子收益和因子暴露都未知时，从收益矩阵中估计共同风险结构。PCA/SVD 给出最佳低秩近似，前几个主成分解释收益矩阵中最大的共同变化方向。它的优势是只需要收益数据、能降维并捕捉未知共同风险；劣势是解释性较弱，单个因子方向可能不稳定。PCA 找到的是统计子空间，不一定直接等同于经济因子。
