# 44.4 新因子的估计、加入与交易

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 9, Portfolio Management: The Basics.

研究者经常发现一个新的资产特征，想知道它是否值得交易。例如，一个新的质量指标、替代数据变量、文本情绪指标或流动性度量。问题不是简单把它加入 alpha 列表，而是要判断它相对现有因子是否真的提供新增信息，并且如何把它转化为组合。

第一步是正交化。新特征 `a` 可能已经被已有因子解释。比如一个“高股息”特征可能与价值、低波动、防御行业和大市值高度相关；一个“文本乐观”特征可能与动量或盈利修正相关。若不先剔除已有因子暴露，新因子的收益可能只是旧因子收益的另一种表达。正交化把 `a` 中能由现有 `B` 解释的部分推出去，留下相对于现有模型的新方向。

在加权几何下，正交化可以写成类似投影残差：

```text
b_new = (I - B (B' Omega_epsilon^{-1} B)^{-1} B' Omega_epsilon^{-1}) a
```

这里使用 `Omega_epsilon^{-1}` 加权，是因为低特质噪声资产对因子识别更可靠。这个步骤与前面“push-out”思想一致：新因子应当解释现有模型没有解释的收益，而不是重复已有暴露。

第二步是估计新因子收益。用现有模型的残差收益 `epsilon_t` 回归到正交化后的新暴露上，可以得到每期新因子收益：

```text
f_new,t = b_new' Omega_epsilon^{-1} epsilon_t
          / (b_new' Omega_epsilon^{-1} b_new)
```

然后计算其平均收益、波动率和 Sharpe。若新因子收益均值不稳定、主要由少数日期驱动、或在扣除交易成本后消失，就不应轻易交易。

第三步是更新风险。若新因子已经对旧因子正交，并且因子组合足够分散，那么新因子与旧因子的协方差可以近似视为零，新因子方差约为

```text
(b_new' Omega_epsilon^{-1} b_new)^{-1}
```

这个近似在真实生产中需要小心。因子暴露和协方差会随时间变化，新因子可能与旧因子在危机中相关上升，也可能改变残差结构。因此，解析公式是理解工具，实际系统通常还要用历史因子收益数值估计协方差。

第四步是构造新因子的 FMP。新因子组合为

```text
v_new = Omega_epsilon^{-1} b_new
        / (b_new' Omega_epsilon^{-1} b_new)
```

这个组合的收益等于用总收益或残差收益估计出来的新因子收益。若新因子的预期收益为正，因子方差可估，就可以把它加入因子空间 MVO。若协方差近似块对角，旧因子仓位不变，只新增一笔与新因子预期收益除以方差成比例的仓位。

这个流程看起来线性而干净，但每一步都有实践风险。正交化可能把特征中经济上有意义但与旧因子重叠的部分剔除掉；残差收益可能含有模型遗漏和估计误差；新因子收益可能过拟合历史；FMP 可能集中在低流动性股票；交易成本可能吃掉全部收益。第 43 章的回测纪律在这里尤其重要：新因子加入模型前，应记录理论、定义、样本、参数和所有失败变体。

新因子还必须回答“为什么现在已有模型没有捕捉到”。如果它只是价值、动量或行业暴露的变体，它可能没有新增组合价值。真正有用的新因子通常能改善风险解释、提高样本外信息系数、降低残差结构、或在组合层面提供可交易的独立收益来源。统计显著只是入口，不是结论。

## 小结

加入新因子通常包括正交化、收益估计、风险更新和 FMP 构造四步。正交化确保新特征提供相对现有模型的新增信息；残差回归估计新因子收益；风险更新判断它如何进入因子协方差；FMP 把它转成可交易组合。新因子必须经受回测纪律、交易成本、容量、流动性和样本外稳定性检验，否则可能只是旧因子或历史噪声的包装。
