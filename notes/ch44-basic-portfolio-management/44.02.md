# 44.2 均值-方差最优组合

来源：Giuseppe A. Paleologo, *The Elements of Quantitative Investing*, Chapter 9, Portfolio Management: The Basics.

有了风险模型和 alpha 之后，最基本的问题是：在给定风险预算下，哪一个组合的预期收益最高？设 `alpha` 是资产预期超额收益向量，`Omega` 是资产收益协方差矩阵，`w` 是组合权重或美元持仓。最简单的问题写成

```text
max alpha' w
s.t. w' Omega w <= sigma^2
```

约束中的 `sigma` 是组合允许承担的最大波动率。这个问题也可以理解为最大化 Sharpe Ratio。因为 Sharpe Ratio 对组合规模不敏感，单独最大化会出现尺度不确定；一旦固定波动率预算，最大化 Sharpe 就等价于在风险预算内最大化预期收益。

一阶条件给出最优组合方向：

```text
w* ∝ Omega^{-1} alpha
```

若要满足精确的风险预算，则

```text
w* = sigma * Omega^{-1} alpha / sqrt(alpha' Omega^{-1} alpha)
```

这个公式是组合管理中最重要的公式之一。它说，最优持仓不是直接等于 alpha，而是等于精度矩阵 `Omega^{-1}` 作用在 alpha 上。协方差矩阵告诉我们风险如何共同波动；精度矩阵则告诉我们在控制其他资产后，某个 alpha 应该如何转化为边际持仓。

一个直接推论是：如果波动率约束固定，alpha 的绝对尺度误差不一定致命。把所有 alpha 同时乘以一个正数，最优组合方向不变，只是未约束问题中的风险偏好参数会改变。真正重要的是相对 alpha：哪些资产比哪些资产更有吸引力。若 alpha 排序和相对强弱错了，组合方向会错；若只是整体单位估大或估小，在固定风险预算下影响较小。

这个结果也可以用资产 Sharpe 和相关矩阵重写。设资产波动率构成对角矩阵 `V`，相关矩阵为 `C`，则 `Omega = V C V`。把持仓换成“美元波动率持仓” `v = V w`，资产 alpha 换成资产级 Sharpe `s = V^{-1} alpha`，最优解变成

```text
v* ∝ C^{-1} s
```

这比原公式更直观：资产的风险持仓由其单资产 Sharpe 决定，但还要用相关矩阵的逆修正。如果两个资产高度相关，不能简单都买；若一个资产的 alpha 已经被另一个相关资产解释，新增持仓价值会下降。

相关性和 Sharpe 分散度决定组合上限。如果资产完全不相关，多个正 Sharpe 机会可以强力分散，组合 Sharpe 随机会数量增加而上升。若资产相关性为正且所有资产 Sharpe 相同，组合 Sharpe 会受到相关性限制；在很多资产时，上限近似为单资产 Sharpe 除以相关性的平方根。若资产 Sharpe 存在横截面分散，组合仍能通过多买高 Sharpe、少买或做空低 Sharpe 来提高表现。

精度矩阵还可以用偏相关理解。`Omega^{-1}` 的非对角元素与两个资产在控制其他资产后的偏相关有关。若资产 i 和 j 在控制其他资产后仍正相关，那么它们共同持有会带来重复风险，最优组合会降低其中一方或双方的持仓。MVO 的“神秘”之处，实际上就是在做条件风险去重。

MVO 还有等价的惩罚形式：

```text
max alpha' w - lambda w' Omega w
```

这里的 `lambda` 可以理解为风险的价格。风险预算越大，风险惩罚越低；风险预算越小，风险惩罚越高。也可以反过来写成：在达到目标收益的条件下最小化方差。这些形式在数学上等价，但在投资沟通中用途不同。风险限额团队可能喜欢风险约束形式，组合经理可能喜欢收益减风险惩罚形式，产品说明可能喜欢目标收益下的最小风险形式。

这个基础解必须谨慎使用。它会把 alpha 和协方差中的微小差异放大，尤其当协方差矩阵接近奇异时。后面第 45 章讨论约束优化，就是因为裸 MVO 经常产生极端杠杆、过度集中和高换手。这里的公式应先被理解为“无摩擦世界的基准解”，再逐步加入约束和成本。

## 小结

均值-方差最优组合在固定风险预算下满足 `w* ∝ Omega^{-1} alpha`。它把 alpha 转化为持仓时会考虑资产协方差和偏相关，而不只是单独看预期收益。alpha 的整体尺度误差在固定波动率约束下不致命，相关结构和相对 alpha 更关键。MVO 可写成风险约束、风险惩罚或目标收益下最小方差等价形式，但裸解对输入误差敏感，需要后续约束和成本修正。
